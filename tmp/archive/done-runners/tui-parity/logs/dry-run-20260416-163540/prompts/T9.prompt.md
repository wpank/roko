## Task

Replace the echo-back stub in `features/messaging.rs` with real dispatch to the agent's configured LLM backend. Both the HTTP `POST /message` route and the `/stream` WebSocket must invoke a real backend via `roko_agent::dispatcher::ToolDispatcher`, returning actual model output.

Today:
- `message()` returns `{"response": "<agent_id>: <prompt>", "engram_id": uuid, "context": <context>}`.
- `handle_stream()` echoes each text frame as `<agent_id>: <text>`.

This is a stub. Issue #45 spec #5 requires real agent behavior.

## Implementation

### 1. Extend `AgentState` (`state.rs`)

Add an optional dispatcher handle:

```rust
pub struct AgentState {
    agent_id: String,
    metrics: Arc<Metrics>,
    dispatcher: Option<Arc<roko_agent::dispatcher::ToolDispatcher>>,
    // ...existing fields
}

impl AgentState {
    pub fn with_dispatcher(mut self, d: Arc<roko_agent::dispatcher::ToolDispatcher>) -> Self {
        self.dispatcher = Some(d);
        self
    }
    pub fn dispatcher(&self) -> Option<&Arc<roko_agent::dispatcher::ToolDispatcher>> {
        self.dispatcher.as_ref()
    }
}
```

### 2. Rewrite `message()` (`features/messaging.rs`)

```rust
async fn message(
    State(state): State<Arc<AgentState>>,
    Json(request): Json<MessageRequest>,
) -> Result<Json<serde_json::Value>, (StatusCode, Json<serde_json::Value>)> {
    state.metrics().record_message();
    let Some(dispatcher) = state.dispatcher() else {
        return Err((StatusCode::SERVICE_UNAVAILABLE, Json(json!({
            "error": "agent has no configured dispatcher"
        }))));
    };
    let req = roko_agent::chat_types::ChatRequest {
        messages: vec![ChatMessage::user(request.prompt.clone())],
        model_slug: dispatcher.default_model().to_string(),
        stream: false,
        ..Default::default()
    };
    match dispatcher.dispatch(req).await {
        Ok(resp) => Ok(Json(json!({
            "response": resp.content,
            "reasoning": resp.reasoning,
            "usage": resp.usage,
            "session": resp.session,
            "finish_reason": resp.finish_reason,
            "context": request.context,
        }))),
        Err(e) => Err((StatusCode::BAD_GATEWAY, Json(json!({
            "error": format!("dispatch failed: {e}")
        })))),
    }
}
```
