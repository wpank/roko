## Implementation

### 1. NervViz effect

Create or extend the postfx module with NervViz-style visualizations:

```rust
pub struct VizContext {
    pub task_progress: f64,      // 0-1
    pub plan_progress: f64,      // 0-1
    pub context_pressure: f64,   // 0-1
    pub token_rate: f64,         // 0-1 normalized
    pub agent_active: bool,
    pub iteration: u32,
    pub error_state: bool,
}
```

Three core visualization primitives:
- `progress_field(area, buf, elapsed, progress)` — pulsing fill proportional to progress
- `activity_ripples(area, buf, elapsed, activity)` — concentric ripple waves from active agents
- `data_rain(area, buf, elapsed, throughput)` — Matrix-style rain proportional to token rate

**Key rules**:
- Only write to empty cells (spaces). NEVER overwrite text content.
- Use braille characters (U+2800 range) for sub-pixel detail.
- All animations driven by `elapsed` seconds from `atmosphere.elapsed()`.
- Colors stay in ROSEDUST palette (rose 325°, violet hues).

### 2. Particle system

Floating dot particles when agents are active:
- Density proportional to number of active agents
- Particles drift upward slowly
- Spawn at random positions within the content area
- Fade out after ~2 seconds
- Use dim braille dots

### 3. EffectsConfig gating

### 4. Persist to roko.toml

Under `[tui.effects]`:
```toml

```

Load on startup, save when user changes preset.

### 5. `v` key cycles VFX presets

In input.rs, when user presses `v` in Normal mode:
```

```

Show a brief notification: "Effects: Minimal" etc.

### 6. Build VizContext from TuiState each frame

```rust
let viz_ctx = VizContext {
```
