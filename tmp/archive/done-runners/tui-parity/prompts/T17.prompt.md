## Task

Fix all navigation and scroll issues in one pass:

1. **PageUp/PageDown** — currently behave like Up/Down (scroll 1 line). Make them scroll by page height.
2. **ScrollAccel** — `tui/scroll.rs::ScrollAccel` is fully implemented but never instantiated. Wire it into `App` or delete it.
3. **DrillIn/DrillOut tab-awareness** — Left/Right arrows always expand/collapse plans regardless of active tab. Make tab-aware.
4. **Logs End/G key** — on Logs tab maps to `ScrollAgentEnd`; should scroll to end of log list.
5. **Scroll upper-bound clamping** — no clamping, user can scroll past content into empty space.
6. **Visible focus indicator** — panel borders don't change style when focus zone changes. Add styled border on focused panel.
7. **Ctrl-C force-quit during modals** — Ctrl-C should always force-quit; currently some modal intercepts swallow it.

## Implementation

### 1. PageUp/PageDown

Add two actions:

```rust
ScrollPageUp,
ScrollPageDown,
```

Map KeyCode::PageUp/PageDown to these in `input.rs`. In the handler, use page size = `terminal_size.1 - 4` (reserve 4 rows for header/footer).

### 2. Wire ScrollAccel

In `App::new()`:

```rust
pub scroll_accel: crate::tui::scroll::ScrollAccel,
```

In each scroll handler:

```rust
let step = state.scroll_accel.tick();
state.plan_scroll = state.plan_scroll.saturating_add_signed(step * direction);
```

Reset on direction change or Escape.

### 3. DrillIn/DrillOut tab-aware

Wrap the existing expand/collapse logic in a match on `state.active_tab`:

```rust
match state.active_tab {
    Tab::Dashboard | Tab::Plans => { /* existing plan expand/collapse */ }
    Tab::Git => { /* expand git branch / worktree */ }
    Tab::Context => { /* drill into context node */ }
    _ => {}
}
```

### 4. Logs End/G

In `input.rs`, find the End/G key handler. When `active_tab == Tab::Logs`, set `state.log_scroll = state.log_count.saturating_sub(1)` instead of `ScrollAgentEnd`.

### 5. Scroll clamping

Add helpers:

```rust
impl TuiState {
    pub fn clamp_plan_scroll(&mut self, max: u16) {
        self.plan_scroll = self.plan_scroll.min(max);
    }
    // same for agent_scroll, log_scroll, etc.
}
```

Call clamp after every scroll operation with the actual content length.

### 6. Focus indicator

In renderers, use `BorderStyle::Bold | Color::Yellow` when `state.focus_zone == CURRENT_ZONE`. Otherwise default border.

### 7. Ctrl-C always

In `handle_key()` at the top:

```rust
if key.modifiers.contains(KeyModifiers::CONTROL) && key.code == KeyCode::Char('c') {
    state.should_quit = true;
    return; // always, even when modal open
}
```

## Rules

1. Scroll clamping must use actual content length at render time, not a static guess.
2. ScrollAccel must decay to 1 when no key pressed for >500ms.
3. Focus indicator must be visually obvious (border color change is minimum).
4. Ctrl-C takes precedence over all modal intercepts.
