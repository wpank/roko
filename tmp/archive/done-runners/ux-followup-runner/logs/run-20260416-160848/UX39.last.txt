Added shared JSON extraction and validation in [extract.rs](/Users/will/dev/nunchi/roko/roko/.roko/worktrees/ux-followup-run-20260416-160848/crates/roko-serve/src/extract.rs), and reshaped [ApiError](/Users/will/dev/nunchi/roko/roko/.roko/worktrees/ux-followup-run-20260416-160848/crates/roko-serve/src/error.rs) to return top-level `{ code, message, details? }`. `ValidJson<T>` now handles typed bodies with validation, while `ApiJson<T>` gives the same parse-error shape for permissive `Value` endpoints. I kept the raw proxy/callback bodies permissive and wired manual validation where `validator` wasn’t a good fit, such as `SubscriptionConfig` and `AgentTemplate`.
