## Implementation

### 1. Shared primitive `JsonlCursor`

`crates/roko-cli/src/tui/jsonl_cursor.rs` (new):

```rust
use std::fs::File;
use std::io::{BufReader, BufRead, Seek, SeekFrom};
use std::path::{Path, PathBuf};

pub struct JsonlCursor {
    path: PathBuf,
    offset: u64,
    last_line: usize,
}

impl JsonlCursor {
    pub fn new(path: impl Into<PathBuf>) -> Self {
        Self { path: path.into(), offset: 0, last_line: 0 }
    }

    /// Read new lines appended since the last call. Handles truncation by
    /// resetting to offset 0.
    pub fn read_new_lines(&mut self) -> std::io::Result<Vec<String>> {
        if !self.path.exists() {
            return Ok(Vec::new());
        }
        let mut file = File::open(&self.path)?;
        let len = file.metadata()?.len();
        if len < self.offset {
            // truncation detected; reset
            self.offset = 0;
            self.last_line = 0;
        }
        file.seek(SeekFrom::Start(self.offset))?;
        let mut buf = String::new();
        let mut reader = BufReader::new(file);
        let mut out = Vec::new();
        loop {
            buf.clear();
            let n = reader.read_line(&mut buf)?;
            if n == 0 { break; }
            if !buf.ends_with('\n') {
                // incomplete trailing line — do not consume
                break;
            }
            self.offset += n as u64;
            self.last_line += 1;
            out.push(buf.trim_end_matches('\n').to_string());
        }
        Ok(out)
    }

    pub fn offset(&self) -> u64 { self.offset }
    pub fn last_line(&self) -> usize { self.last_line }
}
```

### 2. Per-file cursor structs

```rust
// crates/roko-cli/src/tui/cursors.rs
pub struct SignalCursor {
    cursor: JsonlCursor,
    state: SignalsState,  // existing type from dashboard.rs
}

impl SignalCursor {
    pub fn tick(&mut self) -> std::io::Result<()> {
        for line in self.cursor.read_new_lines()? {
            if let Ok(signal) = serde_json::from_str::<Signal>(&line) {
                self.state.ingest(signal);
            }
        }
        Ok(())
    }
    pub fn snapshot(&self) -> &SignalsState { &self.state }
}
```

## Rules

1. **Truncation-safe.** If the file gets rewritten (shorter length), reset
   the offset to 0.
2. **Partial-line safe.** Never commit a cursor advance across an incomplete
   trailing line. Wait for the next notify tick.
3. **No double-ingest.** After `tick()`, the cursor's offset equals the
   file length at read time.
4. **Reset on file missing.** If a file gets deleted and recreated, the
   cursor detects via offset > file length and resets.
5. **Same read rate as UX06.** Do not re-poll outside notify events.
