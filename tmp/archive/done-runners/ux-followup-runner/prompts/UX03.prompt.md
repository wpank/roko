## Task

Write a single integration test (or shell script) that drives the full
self-hosting workflow end-to-end with a `MockDispatcher` returning canned
responses. Assert the expected `.roko/*.jsonl` + `plans/<slug>/tasks.toml`
surface exists at each step.

## CLAUDE.md workflow to reproduce

```bash
# Step 1: capture an idea
roko prd idea "Test: wire XYZ"

# Step 2: draft PRD (agent refines)
roko prd draft new "test-wire-xyz"

# Step 3: research (may no-op in test; mock returns empty citations)
roko research enhance-prd test-wire-xyz

# Step 4: plan
roko prd plan test-wire-xyz

# Step 5: run
roko plan run plans/

# Step 6: resume (no-op if step 5 finished)
# Step 7: dashboard (skipped — TUI in tests)
# Step 8: status
```

### 1. Create `tests/e2e_self_host.rs` at workspace root

Preferred location: `crates/roko-cli/tests/e2e_self_host.rs` (package-scoped
integration test). If the project convention is workspace-root `tests/`,

```rust
//! End-to-end self-hosting smoke test.
//!
//! Drives CLAUDE.md's 8-step workflow with a MockDispatcher. No network,
//! no real LLM, completes in < 60 s on a developer laptop.

use assert_cmd::Command;
use predicates::prelude::*;
use tempfile::TempDir;

#[test]
fn self_hosting_workflow_with_mock_dispatcher() {
    let tmp = TempDir::new().expect("tempdir");
    let workdir = tmp.path();

    // Step 0: init
    run_cli(workdir, &["init"]).success();

    // Step 1: idea
    run_cli(workdir, &["prd", "idea", "Test: wire XYZ"]).success();
    assert!(workdir.join(".roko/prd/ideas").read_dir().unwrap().next().is_some());

    // Step 2: draft (mock returns canned PRD body)
    run_cli(workdir, &["prd", "draft", "new", "test-wire-xyz"])
        .env("ROKO_DISPATCHER", "mock-self-host-fixture")
        .success();
    let draft = workdir.join(".roko/prd/drafts/test-wire-xyz/PRD.md");
    assert!(draft.exists(), "draft at {}", draft.display());

    // Step 3: promote + implicit plan (UX02 fires PrdPublished event in-process)
    run_cli(workdir, &["prd", "draft", "promote", "test-wire-xyz"]).success();
    let plan_toml = workdir.join("plans/test-wire-xyz/tasks.toml");
    assert!(plan_toml.exists(), "plans/test-wire-xyz/tasks.toml missing");

    // Step 4: validate (UX04)
    run_cli(workdir, &["plan", "validate", "plans/"]).success();

    // Step 5: run (mock agent returns canned success)
    run_cli(workdir, &["plan", "run", "plans/"]).success();

    // Step 6: assert episode log has at least one agent_turn entry
    let episodes = workdir.join(".roko/episodes.jsonl");
    assert!(episodes.exists());
    let body = std::fs::read_to_string(&episodes).unwrap();
    assert!(body.lines().any(|l| l.contains("\"kind\":\"agent_turn\"")));

    // Step 7: status
    run_cli(workdir, &["status"]).success();
}

fn run_cli(workdir: &std::path::Path, args: &[&str]) -> assert_cmd::assert::Assert {
    Command::cargo_bin("roko")
        .unwrap()
        .current_dir(workdir)
        .args(args)
        .env("ROKO_TEST_MODE", "1")
        .env("ROKO_DISPATCHER", "mock-self-host-fixture")
        .assert()
}
```

### 2. Extend `MockDispatcher` with scripted replies (if needed)

`crates/roko-std/src/mock.rs` or `crates/roko-agent/src/mock.rs`:

```rust
/// Script a mock dispatcher with a named fixture. Canned replies live in
/// `testdata/mock-dispatcher/<fixture-name>/turn-<n>.json`.
pub fn new_scripted(fixture: &str) -> Result<Self> {
    let root = PathBuf::from(env!("CARGO_MANIFEST_DIR")).join("testdata/mock-dispatcher").join(fixture);
    Ok(Self { replies: load_dir(&root)? })
}
```

## Rules

1. **No real network.** The test must pass with no env vars beyond what it
   sets itself. No Claude CLI, no OpenAI keys.
2. **< 60 s on a laptop.** If slower, reduce the canned plan to a single task.
3. **Idempotent.** `TempDir::new()` per test invocation; never touch
   `/Users/…/roko/.roko/`.
4. **Honest assertions.** Do not mock what the test is trying to prove.
   Verify real file surfaces, not in-memory structs.
5. **Clean teardown.** `TempDir` handles this automatically; do not leave
   behind any files outside the tmp dir.
