## Rules

1. **Incremental-aware.** Allow callers to pass a `JsonlCursor` so repeated
   calls do not re-scan the full file.
2. **Bounded output.** `n_buckets` capped at 168 (7 days of hours).
3. **Thread-safe.** Aggregator is a pure function; no shared mutable state.
4. **Fast.** ≤ 50 ms for 10 k events.
