## Rules

1. **Do not break existing rungs 0-2.** They continue to dispatch the same
   gates as before.
2. **Partial failure surfaces.** One gate failing within a rung does not
   abort the others; the orchestrator already handles a `Vec<Verdict>`.
3. **No new thread panics.** If a gate impl has no real backing
   implementation, its `verify` should return a `Verdict { passed: true,
   note: "stub gate; not yet implemented" }` — do not pretend it passed
   silently.
