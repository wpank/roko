## Task

Compute an `HdcVector` from `(prompt, outcome)` at episode-write time and
stash it on the `Episode` record. Enables future nearest-neighbour retrieval.

### 2. Fingerprint helper

`crates/roko-learn/src/hdc_fingerprint.rs` (new):

```rust
use roko_primitives::HdcVector;

pub fn fingerprint_episode(prompt: &str, outcome: &str) -> HdcVector {
    let mut vec = HdcVector::zero();
    vec.add_symbol(prompt);
    vec.add_symbol(outcome);
    vec.normalize();
    vec
}

pub fn encode(v: &HdcVector) -> String {
    base64::encode(v.to_bytes())
}
```

## Rules

1. **Additive serde field.** Never break existing jsonl readers.
2. **Base64 on disk.** Not raw binary; JSONL requires UTF-8 lines.
3. **Fingerprint is opaque.** Do not include in UI until a retrieval
   call-site (future batch) consumes it.
4. **Small CPU cost.** Fingerprinting must not add > 1 ms per episode.
