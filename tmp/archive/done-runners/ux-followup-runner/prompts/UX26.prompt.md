## Task

`AgentContract` is declared but never constructed outside its own test
module. Wire `SafetyLayer` to load contracts at boot and enforce the
declared invariants per dispatch.

See `context-pack/04-SAFETY-LAYER.md` for the full design context. This
batch implements the "Expected pattern for UX26" section from that pack.

## Implementation

### 1. Load contracts at boot

Add to `ToolDispatcher::new`:

```rust
let contract = AgentContract::load_for_role(&config, &role)
    .unwrap_or_else(|err| {
        tracing::warn!(%role, %err, "no contract for role; using permissive default");
```

### 2. Contract check API

```rust
// safety/mod.rs
impl SafetyLayer {
    pub async fn check_contract(&self, inv: &Invocation) -> Result<(), SafetyError> {
        for inv_rule in &self.contract.invariants {
            match inv_rule {
                Invariant::MaxTokensPerTurn(max) => {
                    if inv.estimated_tokens > *max {
                        return Err(SafetyError::ContractViolation {
                            rule: "MaxTokensPerTurn".into(),
                            detail: format!("{} > {}", inv.estimated_tokens, max),
                        });
                    }
                }
                Invariant::RequireGateBeforeCommit => { /* gate dispatch check */ }
                Invariant::NoNetworkAccess => {
                    if inv.tool.starts_with("net") || inv.tool == "fetch" {
                        return Err(SafetyError::ContractViolation { … });
                    }
                }
            }
        }
        Ok(())
    }
}
```

### 3. Insert into dispatch

```rust
impl ToolDispatcher {
    pub async fn dispatch(&self, req: ChatRequest) -> Result<ChatResponse> {
        self.safety.check_pre_execution(&req, &self.role).await?;
        self.safety.check_contract(&req).await?;   // NEW
        let resp = self.backend.send_turn(req).await?;
        self.safety.check_post_execution(&resp, &self.role).await?;
        if let Some(recovery) = self.safety.contract.recovery.applicable(&resp) {
            recovery.apply(&mut self).await?;   // NEW
        }
        Ok(resp)
    }
}
```

## Rules

1. **Non-breaking.** Absent contract → permissive mode. Existing tests
   continue to pass.
2. **Fail closed on violation.** Never log-and-continue — return
   `Err(SafetyError::ContractViolation)`.
3. **Load at boot, not per call.** Contracts are immutable per dispatcher.
4. **Recovery is optional.** `RecoveryRules::applicable` may return `None`;
   no recovery needed.
5. **YAML loader guard.** If the YAML defaults are missing/malformed, fall
   through to the permissive default and log a warn — do not crash the
   dispatcher.
