## Task

Claude backends have richer test coverage than Cursor / Codex / Exec /
OpenAI. Replicate the 5-scenario suite shape across the non-Claude backends:

1. **Happy path** — plain send_turn returns expected content
2. **Streaming** — chunks arrive in order, final frame closes cleanly
3. **Tool call** — model emits tool call, we return result, model continues
4. **Error path** — HTTP 4xx/5xx maps to typed Rust error
5. **Session continuation** — `session` token plumbs through two turns

## Rules

1. **Reuse, don't copy.** Put shared logic in `testutil.rs`; per-backend
   files should be ≤ 50 LOC.
2. **Record-replay.** All fixtures are byte-exact captures or hand-crafted
   matches of the real protocol.
3. **Offline.** No network.
4. **No flaky timing.** Use `tokio::time::pause()` for anything that sleeps.
