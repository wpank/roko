### 4. Two-session persistence test

```rust
#[test]
fn adaptive_thresholds_persist_across_sessions() {
    let tmp = tempfile::TempDir::new().unwrap();
    let path = tmp.path().join("gate-thresholds.json");

    let mut t1 = AdaptiveThresholds::load(&path).unwrap_or_default();
    t1.observe(rung=2, outcome=true);
    t1.observe(rung=2, outcome=true);
    t1.save(&path).unwrap();

    let t2 = AdaptiveThresholds::load(&path).unwrap();
    assert!((t2.threshold_for(2) - t1.threshold_for(2)).abs() < 1e-6);
}
```

## Rules

1. **Non-breaking.** `threshold_for(rung)` for an unknown rung returns
   0.5 (the neutral default).
2. **Load once per run.** Cache the `AdaptiveThresholds` struct on the
   executor; don't re-read the file per gate-rung call.
3. **Atomic save.** If the threshold is updated mid-run, save via tmp +
   rename.
