docs list additional `[agent.<role>]` keys that never make it into the
runtime config struct: `role`, `budget`, `thresholds`, `routing_overrides`.

## Implementation

Plan: parse them into the struct. Plumb each to the first use-site below.

### 1. Struct extension

```rust
// crates/roko-cli/src/agent_config.rs
pub struct AgentRoleConfig {
    pub model: String,
    pub command: Option<String>,
    pub mcp_config: Option<PathBuf>,
    pub env: HashMap<String, String>,

    // NEW
    pub role: Option<String>,           // explicit role override (default: section name)
    pub budget: Option<AgentBudget>,    // per-turn token/cost caps
    pub thresholds: Option<AgentThresholds>,  // gate pass-rate overrides
    pub routing_overrides: Option<RoutingOverrides>,
    pub tools: Option<Vec<String>>,     // from UX27

pub struct AgentBudget {

    pub max_cost_usd_cents_per_turn: Option<u32>,

}

    pub force_backend: Option<String>,
```
