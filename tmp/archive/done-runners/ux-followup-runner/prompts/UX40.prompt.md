## Task

`Episode` records `model: String` + `agent_template: String` but not which
backend served the turn. Two episodes with `model = "claude-opus-4-6"` could
come from the Claude CLI vs Claude API vs a cascade-routed alternate — no
way to tell them apart in audit or cascade analysis.

Add `backend: String` + populate from the dispatcher.

## Implementation

### 1. Schema additive

```rust
// crates/roko-learn/src/episode_logger.rs
pub struct Episode {
    // existing fields…
    #[serde(default)]
    pub backend: String,   // e.g. "claude_cli", "claude_api", "codex", "cursor", "openai_compat", "ollama", "exec"
}
```

### 2. Backend identifier on the dispatcher

```rust
// crates/roko-agent/src/provider/mod.rs
pub trait LlmBackend {
    // existing…
    fn backend_id(&self) -> &'static str;
}
```

Implement for every backend with a stable string:
- `ClaudeAgent` → `"claude_api"`
- `ClaudeCliAgent` → `"claude_cli"`
- `CodexAgent` → `"codex"`
- `CursorAgent` → `"cursor"`
- `OpenAiAgent` → `"openai"`
- `OpenAiCompatBackend` → `"openai_compat"`
- `OllamaAgent` → `"ollama"`
- `ExecAgent` → `"exec"`
- `GeminiAgent` → `"gemini"`
- `PerplexityAgent` → `"perplexity"`

### 3. Populate on episode write

```rust
// at the episode write site in orchestrate.rs (or dispatcher)
episode.backend = dispatcher.backend_id().to_string();
```

### 4. Schema version bump coordination

If UX12 has landed an `ExecutorSnapshot::schema_version`, consider whether
this change warrants bumping (it does **not** — `Episode` has its own
lifecycle on the JSONL; new field is `#[serde(default)]`-compat).

### 5. Tests

```rust
#[test]
fn episode_backend_populated_on_write() {
    let tmp = tempfile::TempDir::new().unwrap();
    let logger = EpisodeLogger::new(tmp.path().join("episodes.jsonl"));
    let episode = Episode { backend: "claude_cli".into(), ..Default::default() };
    logger.append(&episode).unwrap();
    let lines = logger.read_all().unwrap();
    assert_eq!(lines[0].backend, "claude_cli");
}

#[test]
fn existing_episode_without_backend_deserializes_as_empty_string() {
    let raw = r#"{"kind":"agent_turn","id":"e1"}"#;
    let ep: Episode = serde_json::from_str(raw).unwrap();
    assert_eq!(ep.backend, "");
}
```

## Rules

1. **Additive only.** Old episodes missing the field deserialize with
   `backend = ""`.
2. **Stable identifiers.** The strings are part of the JSONL schema; don't
   rename after the batch lands.
3. **No feature flag.** Always populate.
