   - `grimoire` → `neuro`

2. **Stale-snapshot sidecar** for every `bardo-backup/tmp/roko-progress/*.md`

## Implementation

### 1. Terminology sweep

```bash
grep -rn 'grimoire\|styx\|clade\|mortal\|death\|reincarnation' \
    CLAUDE.md README.md tmp/ \
    --include='*.md'
```

Live docs only: `CLAUDE.md`, `README.md`, `tmp/**` (except
`tmp/ux-followup/` and `tmp/ux-followup-runner/` — those are already clean).
Do **not** modify `bardo-backup/`.

Per hit:
- `grimoire` → `neuro`
- `styx` → `Korai`
- `clade` → `fleet`
- `mortal` → `tier-progressed` / context-specific
- `death` → `retirement`
- `reincarnation` → remove or replace with "tier reset"

### 2. Stale-snapshot banner

For every `bardo-backup/tmp/roko-progress/*.md`, generate a sidecar note
in `tmp/ux-followup-runner/` that carries the warning banner and a pointer
back to the historical archive:

```markdown
> ⚠ Historical snapshot from 2026-04-16 pre-PR-13. **Not** kept in sync
> with current roko. For current state, see `CLAUDE.md` or
> `tmp/ux-followup/`.
```

Keep the operation idempotent. A rerun should overwrite the sidecar, not
append duplicates. Use a simple script:

```bash
#!/usr/bin/env bash
BANNER='> ⚠ Historical snapshot from 2026-04-16 pre-PR-13. **Not** kept in sync
> with current roko. For current state, see `CLAUDE.md` or `tmp/ux-followup/`.

'
out=tmp/ux-followup-runner/bardo-backup-stale-snapshot.md
{
    printf '%s' "$BANNER"
    printf '%s\n' 'Source tree: bardo-backup/tmp/roko-progress/'
    printf '%s\n' 'This directory is read-only; do not edit the archived markdown in place.'
} > "$out"
```

**Important**: `bardo-backup/` is read-only per project rules. Do not
claim an in-place banner mutation is part of the solution. The sidecar note
is the intended warning surface.

### 3. Verify

Grep after the sweep:
```bash
grep -rn 'grimoire\|styx\|clade\|mortal\|death' \
    CLAUDE.md README.md tmp/ --include='*.md' \
    | grep -v 'bardo-backup\|ux-followup/'
```
Expected: empty output.

1. **No touching `bardo-backup/`.** Content stays frozen.
2. **Idempotent.** Re-running the banner script does not duplicate.
3. **Preserve markdown formatting.** Don't reflow prose — only sed on the
   matching tokens.
4. **No silent typo fix.** Only the terminology renames listed above.
