# Domain: Neuroscience

Innovations drawing from neuroscience research — somatic marker hypothesis, active inference, predictive coding, and cognitive load in software development.

| Slug | Description |
|---|---|
| [hdc-active-inference](../hdc-active-inference.md) | Encode the generative world model as a 10,240-bit HDC vector; prediction error becomes Hamming distance; belief updates become vector bundle operations. Draws on Friston's free-energy principle and Bybee & Bhatt (2024) VSA probability computation. |
| [code-somatic-markers](../code-somatic-markers.md) | Auto-generate somatic markers from code intelligence metrics (complexity, novelty, coverage). Operationalises Damasio's somatic marker hypothesis for programming; validated by Fakhoury et al. (2024) EEG studies of developer cognitive load. |

- Renner et al. (2024). Brain-Inspired Computational Intelligence via Predictive Coding. arXiv:2308.07870v3.
- Fakhoury et al. (2024). EEG as a Potential Ground Truth for Cognitive State in Software Development Activities. *PLOS ONE*.
- Kaur et al. (2025). Towards Decoding Developer Cognition in the Age of AI Assistants. arXiv:2501.02684.
