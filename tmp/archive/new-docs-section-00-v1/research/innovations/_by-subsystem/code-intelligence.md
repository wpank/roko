# Subsystem: Code Intelligence (roko-index)

Innovations that interact with the code intelligence system — symbol graph, PageRank, HDC fingerprints, and test coverage metrics.

| Slug | Interaction |
|---|---|
| [code-somatic-markers](../code-somatic-markers.md) | Reads cyclomatic complexity, reverse dependency count, transitive dependency count, PageRank scores, and HDC fingerprints from `roko-index` to compute 8D strategy space coordinates for somatic landscape queries. |
| [knowledge-morphogenesis](../knowledge-morphogenesis.md) | HDC fingerprints from `roko-index` tag code-related knowledge entries that feed the knowledge concentration gradient computation. |
