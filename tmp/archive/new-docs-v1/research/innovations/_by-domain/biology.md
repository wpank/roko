# Domain: Biology

Innovations drawing from biological pattern formation, morphogenesis, and swarm behaviour.

| Slug | Description |
|---|---|
| [knowledge-morphogenesis](../knowledge-morphogenesis.md) | Replace task-return activation in Gierer-Meinhardt reaction-diffusion with knowledge concentration gradients from each agent's Neuro store; agents specialise based on what they know rather than what recently succeeded. |
| [stigmergic-bandits](../stigmergic-bandits.md) | Treat pheromone trails as bandit arms; ant colony stigmergy provides the collective exploration memory that eliminates cold starts for new agents. |
