# Domain: Causal Inference

Innovations that apply causal discovery, do-calculus, and structural causal models (Pearl's framework) to the agent's internal state.

| Slug | Description |
|---|---|
| [affect-causal-discovery](../affect-causal-discovery.md) | Treat PAD affect vector as a node in a Structural Causal Model; use do-calculus interventions to distinguish affect-caused failures from failure-caused affect. |
| [witness-world-model](../witness-world-model.md) | Extract a causal world model (DAG) from Witness DAG Prediction→Resolution chains; use Expected Free Energy over the DAG for active inference action selection. |
