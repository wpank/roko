## Intent

Memory should not accept every reflection or agent claim. Add an admission-control path for useful knowledge and anti-knowledge.

## Implementation Requirements

- Locate current memory/knowledge/admission surfaces.
- Add or refine a knowledge candidate record with source, evidence, confidence, scope, and expiry/decay if appropriate.
- Add anti-knowledge or negative evidence support for repeated bad patterns.
- Implement conservative admission rules tied to gate outcomes and repeated evidence.
- Keep admitted knowledge separate from raw observations.
- Add tests for admission, rejection, anti-knowledge, and evidence threshold behavior.

## Acceptance

- Raw observations do not automatically become trusted knowledge.
- Knowledge admission is evidence-based and auditable.
- Negative evidence can suppress known-bad prompt/context/policy choices.
- Later context bidders can query admitted knowledge safely.
