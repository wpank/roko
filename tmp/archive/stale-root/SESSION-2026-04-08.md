with inline 1-sentence prompts, not knowing Batch 0's Wave G had built 2,500+ lines of templates.

```

  ├─ Wire SafetyLayer into ToolDispatcher call path  ← already partially done
  ├─ Wire SystemPromptBuilder into orchestrate.rs    ← replaces inline 1-sentence prompts

Phase 1: SELF-HOSTING MVP
  ├─ roko plan run ./plans/  works end-to-end
  ├─ roko reads plan files → dispatches to Claude → runs gates → persists
  ├─ Basic TUI status view (watch it work)

  ├─ roko generates plans against its own codebase
  ├─ roko runs plans against itself
  ├─ Gate pipeline catches regressions
  └─ Human reviews PRs, roko does the work
```
