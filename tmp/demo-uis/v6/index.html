—— 03 · HYPERDIMENSIONAL COMPUTING
10,000-bit vectors that search in microseconds
Imagine encoding everything an agent knows as a single point in 10,000-dimensional space. Two similar experiences land near each other. Two different ones land far apart.
Three operations — Bind (combine concepts), Bundle (merge many), Permute (encode sequence) — let you compose, search, and compare knowledge in nanoseconds.
This isn't a database query. It's holographic recall — the whole memory participates in every search.
