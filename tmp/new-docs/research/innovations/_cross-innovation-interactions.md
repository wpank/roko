## Four key feedback loops

1. **HDC Beliefs [1] ↔ Witness World Model [6]**: HDC vectors provide compact belief state (~8 μs per-tick inference); Witness DAG provides causal structure for deeper (~50 ms) reflection. Use HDC for real-time; Witness for periodic Theta-phase reasoning.

2. **Affect Causal [2] ↔ Code Somatic [7]**: Causal discovery learns *why* PAD states cause failures; code somatic markers encode these discoveries as fast-path heuristics. Causal model provides the *why*; somatic markers provide the *speed*.

3. **Dream Verify [3] ↔ Dream Economy [8]**: Verification conditions determine dream quality; dream quality determines budget allocation. Verified dreams earn more budget for future REM imagination.

4. **Knowledge Morphogenesis [4] ↔ Stigmergic Bandits [5]**: Knowledge concentration drives agent specialisation via reaction-diffusion; pheromone trails encode the collective's exploration history as bandit arms. Specialised agents deposit stronger pheromones in their niche, reinforcing the specialisation gradient.
