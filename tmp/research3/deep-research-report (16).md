## Reference architecture

A high-confidence architecture for agentic UI work starts with structured input, not free-form prompting. The ingest layer should accept a PRD, acceptance criteria, named user journeys, viewport definitions, critical interaction states, design constraints, and a library of visual references. ## Evaluation and measurement

The best grading stack is hybrid. Hard gates should decide whether a build is even eligible for design judgment: application boot success, no uncaught page errors, no critical request failures, task completion on key journeys, no obvious layout breakage at required viewports, and zero critical accessibility failures. Only after those pass should a visual or UX critic contribute scalar scores. This is increasingly supported by benchmark work: *WebArena Verified* replaces ambiguous string matching and LLM judging with structured outputs and deterministic scoring, while *AgentRewardBench* shows that no single LLM judge performs best across all web-agent benchmarks and that rule-only evaluation can underreport true success. ţłÇciteţłéturn20search5ţłéturn20search8ţłü
