## Exact changes

### Case A: Cerebras already uses shared parser

If the Cerebras adapter is just config (base URL, auth header) and dispatch flows through `translate::openai`, **no parser changes needed**. Verify with a test:

```rust
#[test]
fn cerebras_response_uses_openai_compatible_parser() {
    let body = r#"{
        "id": "x",
        "object": "chat.completion",
        "choices": [{"message": {"role": "assistant", "content": "hi"}, "finish_reason": "stop"}],
        "usage": {"prompt_tokens": 0, "completion_tokens": 0, "total_tokens": 0}
    }"#;
    let parsed = roko_agent::translate::openai::parse_response(body).unwrap();
    let u = parsed.usage.expect("usage block present");
    assert_eq!(u.input_tokens, Some(0));
    assert_eq!(u.output_tokens, Some(0));
    assert_eq!(u.total_tokens, Some(0));
}

#[test]
fn cerebras_response_no_usage_block_yields_none() {
    let body = r#"{
        "id": "x",
        "choices": [{"message": {"role": "assistant", "content": "hi"}}]
    }"#;
    let parsed = roko_agent::translate::openai::parse_response(body).unwrap();
    assert!(parsed.usage.is_none());
}
```

These tests live near the Cerebras adapter to document the contract.

### Case B: Cerebras has its own parser

If `providers/cerebras.rs` has `fn parse_response` or similar, **delete the duplicate** and route through `translate::openai::parse_response` instead.

Before:

```rust
fn parse_cerebras_response(body: &str) -> Result<...> {
    // Custom parsing, possibly with `Usage { input_tokens: 0, ... }` defaults
}
```

After:

```rust
// Use shared OpenAI-compatible parser; Cerebras returns OpenAI-compatible bodies.
use crate::translate::openai;

let parsed = openai::parse_response(body)?;
```

Update any call site to use the shared parser. Same for streaming (`translate::openai::parse_event` or whatever the streaming parser is named).

If the Cerebras adapter has Cerebras-specific extensions (custom error fields, latency hints), keep them but **do not** duplicate the usage parsing. Extract usage from the shared parsed value.
