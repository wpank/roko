## Exact changes

### 1. Replace dispatch with shared service

Find the handler. Typical shape:

```rust
async fn complete(
    State(state): State<Arc<AppState>>,
    Json(req): Json<CompleteRequest>,
) -> Result<Json<CompleteResponse>, ApiError> {
    let api_key = std::env::var("ANTHROPIC_API_KEY")?;
    let client = reqwest::Client::new();
    let body = json!({"model": req.model, "messages": req.messages, ...});
    let resp = client.post("https://api.anthropic.com/v1/messages")
        .header("x-api-key", &api_key)
        .json(&body)
        .send().await?;
    let parsed: AnthropicResponse = resp.json().await?;
    // ...
}
```

Replace with:

```rust
async fn complete(
    State(state): State<Arc<AppState>>,
    Json(req): Json<CompleteRequest>,
) -> Result<Json<CompleteResponse>, ApiError> {
    let messages = req.messages.into_iter().map(roko_agent::Message::from).collect();
    let model_req = roko_agent::ModelCallRequest {
        model: req.model.clone(),
        provider: None,    // resolved from model
        messages,
        max_tokens: req.max_tokens.unwrap_or(2048),
        temperature: req.temperature,
        top_p: None,
        stop: vec![],
        timeout: None,
        tools: vec![],
        streaming: false,
    };
    let resp = state
        .model_call_service
        .call(model_req)
        .await
        .map_err(|e| ApiError::internal(format!("dispatch: {e}")))?;
    Ok(Json(CompleteResponse {
        text: resp.text,
        model: resp.model,
        usage: resp.usage,   // UsageObservation, optional
        // ...
    }))
}
```

The exact field names and types must match what `ModelCallRequest` / `ModelCallResponse` actually expose. **Read the canonical types** in `crates/roko-agent/src/model_call_service.rs` before guessing.

### 3. If the route streamed before, use `model_call_service.stream(...)`

For SSE / streaming responses:

```rust
use futures::StreamExt;
use axum::response::sse::{Event, Sse};

let stream = state.model_call_service.stream(model_req).await?;
let sse_stream = stream.map(|ev| match ev {
    ModelStreamEvent::TextDelta(t) => Ok::<_, std::convert::Infallible>(
        Event::default().event("delta").data(t),
    ),
    ModelStreamEvent::Completed { text, usage } => Ok(Event::default()
        .event("done")
        .json_data(serde_json::json!({"text": text, "usage": usage}))
        .unwrap_or_else(|_| Event::default().event("done").data(""))),
    ModelStreamEvent::Failed { reason, code } => Ok(Event::default()
        .event("error")
        .json_data(serde_json::json!({"reason": reason, "code": code}))
        .unwrap_or_else(|_| Event::default().event("error").data(reason))),
});
Sse::new(sse_stream)
```

## Do NOT

- Do NOT bundle multiple route migrations into one batch.
- Do NOT refactor neighboring routes "while you're there."
- Do NOT change the route's path or method.
- Do NOT change the request schema (`CompleteRequest`).
- Do NOT change the response schema unless absolutely required by the new dispatch shape; if so, log it.
- Do NOT remove `reqwest` from `roko-serve` `Cargo.toml` — other routes still use it for non-LLM HTTP.
- Do NOT change provider auth resolution (lives in `DispatchResolver`).
- Do NOT add a fallback path that calls `reqwest::Client` if `model_call_service` errors. Errors propagate.
