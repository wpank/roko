# T5-36c: Migrate per-agent message dispatch to ModelCallService

## Task
The `POST /api/agents/{id}/messages` route in `routes/agents.rs` may still build its own HTTP client to dispatch to the agent's configured model. Migrate it to `ModelCallService`.

## Runner Context
Runner audit-2026-05-01, group T5. No dependencies. Wave 1.

## Source plan
`tmp/subsystem-audits/implementation-plans/22-dispatch-streaming-completion.md` § Phase 1.

## Read first

```bash
rg 'fn send_message|fn post_message|/messages|reqwest::Client' crates/roko-serve/src/routes/agents.rs -n
```

The route handles two cases:

- **Self-hosted agents**: roko dispatches to a model on behalf of the agent.
- **Externally-registered agents**: the request is *proxied* to the agent's external endpoint.

Only the **self-hosted** path migrates. The proxy path uses `reqwest::Client` legitimately to forward to the agent's endpoint (a real external API call, not LLM dispatch).

## Exact changes

### 1. Distinguish the two paths

```rust
async fn post_message(
    Path(id): Path<String>,
    State(state): State<Arc<AppState>>,
    Json(req): Json<MessageRequest>,
) -> Result<...> {
    let agent = state.find_agent(&id)?;
    if agent.is_self_hosted() {
        dispatch_self_hosted(&state, agent, req).await
    } else {
        proxy_to_agent_endpoint(&state, agent, req).await   // unchanged; uses reqwest
    }
}
```

### 2. Migrate the self-hosted path

`dispatch_self_hosted` uses `state.model_call_service`:

```rust
async fn dispatch_self_hosted(
    state: &AppState,
    agent: &AgentRecord,
    req: MessageRequest,
) -> Result<...> {
    let model_req = ModelCallRequest {
        model: agent.model.clone().unwrap_or_else(|| state.config.agent.default_model.clone()),
        provider: None,
        messages: req.messages.into_iter().map(roko_agent::Message::from).collect(),
        max_tokens: req.max_tokens.unwrap_or(2048),
        temperature: req.temperature,
        top_p: None,
        stop: vec![],
        timeout: None,
        tools: vec![],
        streaming: false,
    };
    let resp = state.model_call_service.call(model_req).await?;
    Ok(Json(MessageResponse {
        text: resp.text,
        model: resp.model,
        usage: resp.usage,
    }))
}
```

### 3. Leave the proxy path alone

`proxy_to_agent_endpoint` still uses `reqwest::Client` to forward to `agent.endpoints.rest`. **Do not** migrate it. Add a doc-comment explaining why:

```rust
/// Proxy a message to an externally-registered agent's REST endpoint.
/// Uses `reqwest::Client` directly because this is an arbitrary external
/// HTTP call (not LLM dispatch). `ModelCallService` is for LLM provider
/// dispatch only.
```
