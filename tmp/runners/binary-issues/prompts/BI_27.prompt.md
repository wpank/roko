# BI_27: Capture MCP server stderr to log file (no Stdio::inherit)

## Task
Stop letting MCP server child processes spam stderr into the user's TUI. Switch `StdioTransport::spawn_with_env` to `Stdio::piped()` for stderr, then spawn a background task that reads child stderr line-by-line and forwards each line to `tracing::debug!` (with a server-scoped target). Tee the same lines to `.roko/mcp/<server-name>.log` so post-mortem debugging is easy.

## Runner Context
Group **BI_SUB**, batch 27 of 56. Closes MASTER-INDEX § **S9.2**. Deps: none.

## Problem
`crates/roko-agent/src/mcp/client.rs:187` currently configures the spawned MCP child with `.stderr(std::process::Stdio::inherit())`. When the user is in inline chat (raw-mode TTY), any noise the MCP server writes to stderr lands directly in the terminal scroll buffer and corrupts the cursor / status bar.

Per `00-RULES.md` U2 (Subprocess rules): *"Stderr is captured, not inherited. Use `Stdio::piped()` and forward to `tracing` or a log file."*

The fix has three pieces:
1. **Pipe** stderr instead of inheriting it.
2. **Drain** the pipe in a background tokio task — otherwise the kernel pipe buffer fills and the child blocks on its next stderr write.
3. **Tag** each forwarded line with the server name so multi-MCP setups remain debuggable.

Replace the existing `impl StdioTransport { ... }` block with:
```rust
impl StdioTransport {
    /// Spawn a child MCP server process with default environment.
    pub fn spawn(command: &str, args: &[String]) -> Result<Self, McpError> {
        Self::spawn_with_env(command, args, &HashMap::new())
    }

    /// Spawn with extra env vars; uses the binary name as the log scope.
    pub fn spawn_with_env(
        command: &str,
        args: &[String],
        env: &HashMap<String, String>,
    ) -> Result<Self, McpError> {
        Self::spawn_named(command, command, args, env, None)
    }

    /// Spawn a child MCP server with an explicit server name and optional
    /// log directory. Stderr is piped and tee'd to
    /// `<log_dir>/<server_name>.log` (when `log_dir` is provided) AND
    /// forwarded line-by-line to `tracing::debug!` with target
    /// `mcp::<server_name>`.
    pub fn spawn_named(
        server_name: &str,
        command: &str,
        args: &[String],
        env: &HashMap<String, String>,
        log_dir: Option<&std::path::Path>,
    ) -> Result<Self, McpError> {
        let resolved_env = resolve_env(env);
        let mut child = tokio::process::Command::new(command)
            .args(args)
            .envs(resolved_env)
            .stdin(std::process::Stdio::piped())
            .stdout(std::process::Stdio::piped())
            .stderr(std::process::Stdio::piped())
            .kill_on_drop(true)
            .spawn()
            .map_err(|e| McpError::Transport(format!("failed to spawn {command}: {e}")))?;

        let stdin = child
            .stdin
            .take()
            .ok_or_else(|| McpError::Transport("child stdin not available".into()))?;
        let stdout = child
            .stdout
            .take()
            .ok_or_else(|| McpError::Transport("child stdout not available".into()))?;
        let stderr = child
            .stderr
            .take()
            .ok_or_else(|| McpError::Transport("child stderr not available".into()))?;

        let log_path = log_dir.map(|d| d.join(format!("{server_name}.log")));
        spawn_stderr_forwarder(server_name.to_string(), stderr, log_path);

        Ok(Self {
            stdin: Mutex::new(BufWriter::new(stdin)),
            stdout: Mutex::new(BufReader::new(stdout)),
            _child: Mutex::new(child),
        })
    }
}
```

### Step 3: Add the stderr forwarder task
Add this free function below `impl StdioTransport { ... }` (and above `#[async_trait] impl Transport for StdioTransport`):
```rust
/// Drain a child process's stderr and forward each line to `tracing::debug!`.
///
/// Optionally tees lines to `log_path` (best-effort; failure to open the log
/// file is logged but does not stop forwarding). The task exits when the
/// child closes its stderr.
fn spawn_stderr_forwarder(
    server_name: String,
    stderr: ChildStderr,
    log_path: Option<PathBuf>,
) {
    tokio::spawn(async move {
        let target = format!("mcp::{server_name}");
        let mut reader = BufReader::new(stderr).lines();

        // Best-effort log file. Failure is just logged once.
        let mut log_file: Option<tokio::fs::File> = if let Some(ref path) = log_path {
            if let Some(parent) = path.parent() {
                if let Err(err) = tokio::fs::create_dir_all(parent).await {
                    tracing::warn!(
                        target: "mcp",
                        server = %server_name,
                        path = %parent.display(),
                        error = %err,
                        "failed to create mcp log dir; lines will not be tee'd",
                    );
                }
            }
            match tokio::fs::OpenOptions::new()
                .create(true)
                .append(true)
                .open(path)
                .await
            {
                Ok(f) => Some(f),
                Err(err) => {
                    tracing::warn!(
                        target: "mcp",
                        server = %server_name,
                        path = %path.display(),
                        error = %err,
                        "failed to open mcp log file; lines will not be tee'd",
                    );
                    None
                }
            }
        } else {
            None
        };

        while let Ok(Some(line)) = reader.next_line().await {
            tracing::debug!(target: &target[..], "{line}");
            if let Some(ref mut f) = log_file {
                let mut buf = line;
                buf.push('\n');
                if let Err(err) = f.write_all(buf.as_bytes()).await {
                    tracing::warn!(
                        target: "mcp",
                        server = %server_name,
                        error = %err,
                        "mcp log file write failed; dropping",
                    );
                    log_file = None;
                }
            }
        }
    });
}
```
