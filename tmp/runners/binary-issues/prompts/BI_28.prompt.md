# BI_28: Thread CancellationToken into chat dispatch (Ctrl+C cancels)

## Task
Wire a `tokio_util::sync::CancellationToken` from the inline chat session into the spawned dispatch tasks so Ctrl+C actually aborts in-flight turns. Today the chat loop's "interrupted" message is cosmetic — the background `tokio::spawn(...)` keeps running until the provider answers. Store the spawned dispatch as a `JoinHandle`, propagate the token into `ChatAgentSession::send_turn_streaming`, and observe the token between turns / between HTTP poll iterations.

## Runner Context
Group **BI_SUB**, batch 28 of 56. Closes MASTER-INDEX § **S9.4 + S9.5**. Deps: none.

## Problem
Two related bugs at the same call site:

1. **Orphaned dispatch tasks.** `crates/roko-cli/src/chat_inline.rs:1892, 1922` both call `tokio::spawn(async move { ... })` and discard the returned `JoinHandle`. There is no abort handle; Ctrl+C only sets `phase = Phase::Input` (line 1317) but the background task keeps running.
2. **No CancellationToken in the chat loop.** `ChatSession` (lines 775-811) has no token field. `send_turn_streaming` (`crates/roko-cli/src/chat_session.rs:1018-1024` and definition at 1375) takes only `prompt` + event tx — there is no way to ask the provider stream to stop.

`tokio-util` is already a workspace dep of `roko-cli` (`crates/roko-cli/Cargo.toml:54`), so no new crate is needed. The pattern in this repo is `roko_runtime::cancel::*` and `tokio_util::sync::CancellationToken` (used in `serve_runtime.rs`, `daemon.rs`, etc.).

### Step 3: Plumb the token into `dispatch_prompt`
In `crates/roko-cli/src/chat_inline.rs:1875-1934`, replace the function body so it (a) creates a fresh per-turn token, (b) stores the spawned task's `JoinHandle`, and (c) forwards the token into the spawned future:

```rust
fn dispatch_prompt(session: &mut ChatSession, prompt: &str) {
    session.streaming = StreamingState::new("resolving...");
    let (tx, rx) = tokio::sync::mpsc::channel(1);
    session.response_rx = Some(rx);
    let text = prompt.to_string();

    // New per-turn token. Replaces the previous token if any.
    let token = CancellationToken::new();
    session.cancel_token = token.clone();

    let handle = match &session.dispatch {
        DispatchMode::Http { client, backend_url, is_sidecar, .. } => {
            let client_clone = client.clone();
            let url_owned = backend_url.clone();
            let agent_id_owned = session.agent_id.clone();
            let sidecar = *is_sidecar;
            let token = token.clone();
            tokio::spawn(async move {
                let result = tokio::select! {
                    biased;
                    _ = token.cancelled() => {
                        let _ = tx.send(Err("__cancelled__".to_string())).await;
                        return;
                    }
                    res = send_and_receive(&client_clone, &url_owned, &agent_id_owned, &text, sidecar) => res,
                };
                let _ = tx
                    .send(result.map(|r| r.into()).map_err(|e| e.to_string()))
                    .await;
            })
        }
        DispatchMode::Direct { .. } => {
            let _ = tx.try_send(Err(
                ChatInlineDispatchError::DirectDispatchDisabled.to_string(),
            ));
            tokio::spawn(async {})
        }
        DispatchMode::Session => {
            let Some(agent_session) = session.agent_session.as_ref() else {
                let _ = tx.try_send(Err("agent session unavailable".to_string()));
                return;
            };
            let mut agent_session = clone_chat_agent_session(agent_session);
            let (event_tx, event_rx) = tokio::sync::mpsc::channel::<roko_agent::AgentRuntimeEvent>(256);
            session.streaming_event_rx = Some(event_rx);
            let token = token.clone();
            tokio::spawn(async move {
                let result = agent_session
                    .send_turn_streaming(&text, event_tx, token)
                    .await;
                let mapped = match result {
                    Ok(turn) if turn.cancelled => Err("__cancelled__".to_string()),
                    Ok(turn) => Ok(turn_result_to_dispatch_result(turn)),
                    Err(error) => Err(error.to_string()),
                };
                let _ = tx.send(mapped).await;
            })
        }
    };

    session.dispatch_handle = Some(handle);
}
```
