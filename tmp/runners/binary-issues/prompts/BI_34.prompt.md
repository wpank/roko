# BI_34: Surface background-serve failure to user

## Task
A failed `roko serve` startup currently produces a single `tracing::warn!` and is otherwise invisible — the chat session boots without an HTTP plane and any user that types `/share` or opens the dashboard sees a confusing 404. Surface the failure to the chat TUI status bar, attempt one retry with backoff, and disable any chat command that depends on serve until the next session.

## Runner Context
Group **BI_ERR**, batch 34 of 56. Closes MASTER-INDEX § **S4.4**. Deps: BI_30 (non-blocking startup) is recommended but not strictly required — this batch can land first by surfacing the failure before chat boot.

## Problem
`crates/roko-cli/src/unified.rs:167-179`:

## Required Changes

### Step 1: Add a `ServeStartupOutcome` enum

At the top of `crates/roko-cli/src/unified.rs`, after the `use` block:
```rust
/// Outcome of a background-serve startup attempt.
///
/// Consumed by the chat TUI to render an accurate status line and to
/// disable serve-dependent chat commands when startup failed.
pub enum ServeStartupOutcome {
    /// Caller asked us to skip serve (or auto_start is off).
    Disabled,
    /// Serve booted; HTTP plane is live.
    Ready {
        state: Arc<roko_serve::state::AppState>,
        handle: JoinHandle<Result<()>>,
    },
    /// Serve attempted to boot but failed; chat should run in degraded mode.
    Failed { error: String },
}

impl ServeStartupOutcome {
    pub fn is_ready(&self) -> bool {
        matches!(self, Self::Ready { .. })
    }
}
```

### Step 2: Make `spawn_background_serve` return the new enum, with one retry

Replace `spawn_background_serve` with:
```rust
async fn spawn_background_serve(
    config: &crate::config::Config,
    workdir: &std::path::Path,
) -> ServeStartupOutcome {
    let attempts: [tokio::time::Duration; 2] = [
        tokio::time::Duration::from_millis(0),
        tokio::time::Duration::from_millis(750),
    ];

    let mut last_err: Option<String> = None;
    for (i, backoff) in attempts.iter().enumerate() {
        if !backoff.is_zero() {
            tokio::time::sleep(*backoff).await;
        }
        let runtime = RokoCliRuntime::new(config.clone(), RepoRegistry::default()).into_arc();
        match roko_serve::start_server_background(workdir.to_path_buf(), runtime, None, None).await {
            Ok((state, handle)) => return ServeStartupOutcome::Ready { state, handle },
            Err(e) => {
                let msg = format!("{e:#}");
                tracing::warn!(
                    target: "serve",
                    attempt = i + 1,
                    error = %msg,
                    "background serve failed to start",
                );
                last_err = Some(msg);
            }
        }
    }

    ServeStartupOutcome::Failed {
        error: last_err.unwrap_or_else(|| "unknown error".to_string()),
    }
}
```
