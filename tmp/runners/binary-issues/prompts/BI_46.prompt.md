# BI_46: Extract shared chat event-loop body

## Task

## Problem
`crates/roko-cli/src/chat_inline.rs` contains two `loop { ... }` blocks that are byte-identical apart from the surrounding setup:

1. `run_chat_inline` event loop at **lines 1270-1471** (preceded by `// Main event loop`).
2. `run_unified_inline` event loop at **lines 1649-1832** (preceded by `// Main event loop (identical structure to run_chat_inline)`).

Both loops perform the same six steps per frame:

1. `term.draw(|frame| { render_viewport(frame, &session, &theme); })?;`
2. `event::poll(Duration::from_millis(33))` → `event::read()` and a `match session.phase` arm that handles `Input`, `Thinking | Streaming` (Ctrl+C double-tap, partial-output flush, retry/quit), `Error { ref prompt, .. }` (retry / cancel), and `Done`.
3. Drain `session.streaming_event_rx` (`MessageDelta`, `ToolCall`, `ToolOutput`).
4. Drain `session.response_rx` (success → `push_tool_outputs` + `push_agent_response` + ts/cost/usage + auto-save; cancel; error; channel-closed).
5. `session.tick += 1; session.streaming.tick();`
6. (Implicit `loop` continuation.)

Per `00-RULES.md` rule **A4** ("If two functions render the same UI or call the same provider, extract the shared helper — never copy-paste a fix into both"), this is required. The risk is the diff size: extracting too aggressively would either move local state across function boundaries or pull non-shared setup (auth label / agent_session id update) into a hot path that does not need it. Mitigation: the helper takes `&mut ChatSession` (which already holds `dispatch`, `agent_session`, `cost_table` etc.) and extra borrows for `term`, `theme`, and `cost_table`. The session-mode-only `if matches!(&session.dispatch, DispatchMode::Session) { ... agent_session.session_id = ... }` block stays inside the helper because both branches already gate on the dispatch enum.

The cost table is currently constructed *before* each loop (`run_chat_inline:1233-1236` builds `CostTable::default().with_defaults()`; `run_unified_inline:1530` builds `CostTable::from_config(&model_config.models).with_defaults()`). The helper takes a `&CostTable` borrow so each caller passes its own table without reallocating.

### Step 2: Add the `chat_event_loop_step` helper

Define the helper near the other render helpers, e.g. directly above `fn dispatch_prompt(...)` at `chat_inline.rs:1875`. Move the **entire per-frame body** of either existing loop into this helper, parameterised on `&mut ChatSession`, `&mut InlineTerminal`, `&Theme`, and `&CostTable`. Replace `break;` statements with `return Ok(EventLoopOutcome::Break);` and add a final `Ok(EventLoopOutcome::Continue)`.

```rust
/// Run one frame of the chat REPL event loop.
///
/// Shared between [`run_chat_inline`] (HTTP/sidecar bootstrap) and
/// [`run_unified_inline`] (live `ChatAgentSession` path). Both call this
/// helper inside their own `loop { ... }`. Per BI_46, all per-frame logic
/// lives here so behaviour cannot drift between the two surfaces.
async fn chat_event_loop_step(
    session: &mut ChatSession,
    term: &mut InlineTerminal,
    theme: &Theme,
    cost_table: &CostTable,
) -> Result<EventLoopOutcome> {
    // ── 1. Draw viewport ─────────────────────────────────────────────
    term.draw(|frame| {
        render_viewport(frame, session, theme);
    })?;

    // ── 2. Poll input / key handling ─────────────────────────────────
    if event::poll(Duration::from_millis(33)).context("poll events")? {
        if let Event::Key(key) = event::read().context("read event")? {
            match session.phase {
                Phase::Input => {
                    if handle_input_key(key, session, term, theme).await? {
                        return Ok(EventLoopOutcome::Break);
                    }
                }
                Phase::Thinking | Phase::Streaming => {
                    // Ctrl+C interrupts generation; double-tap exits.
                    if key.code == KeyCode::Char('c')
                        && key.modifiers.contains(KeyModifiers::CONTROL)
                    {
                        let now = Instant::now();
                        let double_tap = session.last_ctrl_c.map_or(false, |prev| {
                            now.duration_since(prev) < Duration::from_millis(500)
                        });
                        session.last_ctrl_c = Some(now);

                        if !session.streaming.is_empty() {
                            let text = session.streaming.take_buffer();
                            push_agent_response(term, theme, &text, &session.agent_id)?;
                        }
                        if double_tap {
                            term.push_lines(&[styled::continuation(
                                theme, "", "[exiting]", None,
                            )])?;
                            session.phase = Phase::Done;
                            return Ok(EventLoopOutcome::Break);
                        }
                        term.push_lines(&[styled::continuation(
                            theme,
                            "",
                            "[interrupted — Ctrl-C again to exit]",
                            None,
                        )])?;
                        session.phase = Phase::Input;
                    }
                }
                Phase::Error { ref prompt, .. } => match key.code {
                    KeyCode::Char('r') => {
                        let retry_prompt = prompt.clone();
                        session.phase = Phase::Thinking;
                        session.thinking_started = Some(Instant::now());
                        dispatch_prompt(session, &retry_prompt);
                    }
                    KeyCode::Char('q') | KeyCode::Esc => {
                        term.push_blank()?;
                        session.phase = Phase::Input;
                    }
                    _ => {}
                },
                Phase::Done => return Ok(EventLoopOutcome::Break),
            }
        }
    }

    // ── 3. Drain streaming-event channel ─────────────────────────────
    if let Some(ref mut event_rx) = session.streaming_event_rx {
        loop {
            match event_rx.try_recv() {
                Ok(event) => match &event {
                    roko_agent::AgentRuntimeEvent::MessageDelta { text } => {
                        if session.phase == Phase::Thinking {
                            session.phase = Phase::Streaming;
                        }
                        session.streaming.append(text);
                    }
                    roko_agent::AgentRuntimeEvent::ToolCall { name, .. } => {
                        session.streaming.set_phase(format!("tool:{name}"));
                    }
                    roko_agent::AgentRuntimeEvent::ToolOutput { .. } => {
                        session.streaming.set_phase("Streaming");
                    }
                    _ => {}
                },
                Err(tokio::sync::mpsc::error::TryRecvError::Empty) => break,
                Err(tokio::sync::mpsc::error::TryRecvError::Disconnected) => break,
            }
        }
    }

    // ── 4. Drain response channel ────────────────────────────────────
    if let Some(ref mut rx) = session.response_rx {
        match rx.try_recv() {
            Ok(Ok(result)) => {
                let latency = session
                    .thinking_started
                    .map(|t| t.elapsed().as_secs_f64())
                    .unwrap_or(0.0);
                push_tool_outputs(term, theme, &result.tool_outputs)?;
                push_agent_response(term, theme, &result.text, &session.agent_id)?;
                let ts = format_time(Instant::now());
                term.push_lines(&[Line::from(vec![Span::styled(
                    format!("  {ts} ({latency:.1}s)"),
                    Style::default().fg(Theme::TEXT_GHOST),
                )])])?;
                session.conversation.push(ConversationMessage {
                    role: "assistant".into(),
                    text: result.text.clone(),
                    timestamp: format!("{ts} ({latency:.1}s)"),
                });
                let cost = cost_from_result(cost_table, &result);
                let naive = naive_opus_cost(result.input_tokens, result.output_tokens);
                session.cost.record_run(
                    cost,
                    result.input_tokens,
                    result.output_tokens,
                    &result.model,
                    naive,
                );
                if matches!(&session.dispatch, DispatchMode::Session) {
                    if let Some(agent_session) = session.agent_session.as_mut() {
                        if let Some(session_id) = result.session_id.clone() {
                            agent_session.session_id = Some(session_id);
                        }
                    }
                }
                session.turn_count += 1;
                session.thinking_started = None;
                if !session.quiet {
                    push_usage_line(
                        term,
                        theme,
                        &result.model,
                        result.input_tokens,
                        result.output_tokens,
                        cost,
                        latency,
                    )?;
                }
                if latency > 10.0 {
                    print!("\x07");
                }
                if session.turn_count % 5 == 0 {
                    save_session(session);
                }
                session.phase = Phase::Input;
                session.response_rx = None;
                session.streaming_event_rx = None;
                term.push_blank()?;
            }
            Ok(Err(err)) => {
                if err == "__cancelled__" {
                    session.thinking_started = None;
                    session.phase = Phase::Input;
                    session.response_rx = None;
                    session.streaming_event_rx = None;
                } else {
                    push_error_with_suggestions(term, theme, &err)?;
                    let prompt = session.last_prompt.clone().unwrap_or_default();
                    session.thinking_started = None;
                    session.phase = Phase::Error { prompt, error: err };
                    session.response_rx = None;
                    session.streaming_event_rx = None;
                }
            }
            Err(tokio::sync::mpsc::error::TryRecvError::Empty) => {}
            Err(tokio::sync::mpsc::error::TryRecvError::Disconnected) => {
                term.push_lines(&[styled::continuation(
                    theme,
                    "error",
                    "response channel closed",
                    None,
                )])?;
                session.thinking_started = None;
                session.phase = Phase::Input;
                session.response_rx = None;
                session.streaming_event_rx = None;
            }
        }
    }

    // ── 5. Tick ──────────────────────────────────────────────────────
    session.tick += 1;
    session.streaming.tick();

    Ok(EventLoopOutcome::Continue)
}
```
