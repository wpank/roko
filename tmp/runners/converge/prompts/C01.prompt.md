### Task

#### 1. Create `crates/roko-cli/src/output_format.rs`

The file provides Clack-style CLI output primitives for the roko demo. All output goes to
stdout. The module re-uses symbols from `crate::inline::symbols` and adds ANSI color helpers
implemented with raw escape codes (no external crate).

##### 1a. ANSI color constants (private)

```rust
const RESET: &str = "\x1b[0m";
const BOLD: &str = "\x1b[1m";
const DIM: &str = "\x1b[2m";

// Foreground colors
const FG_GREEN: &str = "\x1b[32m";
const FG_RED: &str = "\x1b[31m";
const FG_YELLOW: &str = "\x1b[33m";
const FG_CYAN: &str = "\x1b[36m";
const FG_MAGENTA: &str = "\x1b[35m";
const FG_WHITE: &str = "\x1b[97m";
const FG_GRAY: &str = "\x1b[90m";
```

##### 1b. Color helper functions (pub)

```rust
/// Wrap text in bold.
pub fn bold(s: &str) -> String { format!("{BOLD}{s}{RESET}") }
/// Wrap text in dim/gray.
pub fn dim(s: &str) -> String { format!("{DIM}{FG_GRAY}{s}{RESET}") }
/// Wrap text in green.
pub fn green(s: &str) -> String { format!("{FG_GREEN}{s}{RESET}") }
/// Wrap text in red.
pub fn red(s: &str) -> String { format!("{FG_RED}{s}{RESET}") }
/// Wrap text in yellow.
pub fn yellow(s: &str) -> String { format!("{FG_YELLOW}{s}{RESET}") }
/// Wrap text in cyan.
pub fn cyan(s: &str) -> String { format!("{FG_CYAN}{s}{RESET}") }
/// Wrap text in magenta/rose.

```

##### 1c. Layout primitives (pub)

Use `crate::inline::symbols` for the Clack glyphs.

```rust
use crate::inline::symbols;

/// Print the intro line: `◆  <title>` in bold.
///
/// Example output:
/// ```text
/// ◆  roko run
/// ```
pub fn intro(title: &str) {
    println!("{} {}", symbols::START, bold(title));
}

/// Print a step line: `◇  <label>` with an optional value.
///
/// Example output:
/// ```text
/// ◇  model  claude-sonnet-4-6
/// ```
pub fn step(label: &str, value: &str) {
    if value.is_empty() {
        println!("{} {}", symbols::START_EMPTY, bold(label));
    } else {
        println!("{} {}  {}", symbols::START_EMPTY, bold(label), dim(value));
    }
}

/// Print a continuation line: `│  <text>`.
///
/// Example output:
/// ```text
/// │  routing: cascade-router → claude-sonnet-4-6
/// ```
pub fn bar(text: &str) {
    println!("{}  {}", symbols::BAR, text);
}

/// Print a note line: `│  <text>` in dim style.
pub fn note(text: &str) {
    println!("{}  {}", symbols::BAR, dim(text));
}

/// Print a success line: `✔  <msg>` in green.
///
/// Example output:
/// ```text
/// ✔  compile passed
/// ```
pub fn success(msg: &str) {
    println!("{}  {}", green(symbols::PASS), green(msg));
}

/// Print an error line: `✖  <msg>` in red.
pub fn error(msg: &str) {
    println!("{}  {}", red(symbols::FAIL), red(msg));
}

/// Print a warning line: `⚠  <msg>` in yellow.
pub fn warning(msg: &str) {
    println!("{}  {}", yellow(symbols::WARN), yellow(msg));
}

/// Print an empty `│` line (visual spacer between sections).
pub fn divider() {
    println!("{}", symbols::BAR);
}

/// Print a branch line: `├  <text>`.
pub fn branch(text: &str) {
    println!("{}  {}", symbols::BRANCH, text);
}

/// Print an end line: `└  <text>`.
pub fn end(text: &str) {
    println!("{}  {}", symbols::END, text);
}
```
