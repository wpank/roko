#### Implementation

```rust
pub fn print_knowledge_loaded(fact_count: usize, avg_confidence: f64) {
    step("Knowledge", "");
    if fact_count == 0 {
        note("no facts loaded");
    } else {
        bar(&format!(
            "{} facts loaded {}",
            cyan(&fact_count.to_string()),
            dim(&format!("(avg confidence: {:.2})", avg_confidence)),
        ));
    }
}
```

The confidence value is formatted to 2 decimal places (e.g., `0.87`) because
knowledge confidence scores range 0.0–1.0 and more precision is noise.
