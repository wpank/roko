### Task

#### 1. Create `KnowledgeEntries.tsx`

Fetch `/api/knowledge/entries`. Interface:
```typescript
interface KnowledgeEntry {
  id: string; domain?: string; citations?: number;
  label?: string; confidence?: number;
}
```

Display: StatCard grid (Total, Domains, Avg Citations, Avg Confidence), then
a table with columns Label, Domain, Citations, Confidence. Poll every 30s.
Follow `AgentFleet.tsx` style: rosedust theme inline styles, `var(--mono)`,
loading/empty states.

#### 2. Create `CascadeRouter.tsx`

Fetch `/api/learning/cascade-router`. Interface:
```typescript
interface ModelStat { confidence: number; uses: number; }
interface CascadeState { model_stats?: Record<string, ModelStat>; last_updated?: string; }
```

Display: StatCard grid (Models, Total Uses, Avg Confidence), table of model
stats (Model, Confidence, Uses), footer with `last_updated`. Poll every 10s.

#### 3. Add nav entries to `Layout.tsx`

After `Knowledge`, before `Chain`:
```typescript
{ to: '/dashboard/entries', label: 'Entries', end: false },
{ to: '/dashboard/routing', label: 'Routing', end: false },
```

#### 4. Add routes to `main.tsx`

```tsx
import KnowledgeEntries from './pages/dashboard/KnowledgeEntries';
import CascadeRouter from './pages/dashboard/CascadeRouter';
// Inside dashboard Route:
<Route path="entries" element={<KnowledgeEntries />} />
<Route path="routing" element={<CascadeRouter />} />
```
