### Task

#### 1. Create `ShareView.tsx`

Fetch `/api/shared/${token}` using `useParams` and `useApi`. Display:
- Header with agent, role, timestamp
- Prompt in a `.pane` box
- StatCards: Result (PASS/FAIL), Model, Cost, Duration
- GateBar mapped from `gates: [string, boolean][]` to `{name, status}[]`
- Output in a scrollable `<pre>` block
- Episode ID footer

Follow the pattern of existing `Share.tsx` but use dashboard-relative imports
(`../../hooks/useApi`, `../../components/StatCard`, `../../components/GateBar`).
Style with rosedust theme inline styles (`var(--mono)`, `var(--text-dim)`, etc.).

Handle three states: loading, error/not-found, success.

##### Gate mapping

The `RunTranscript.gates` field is an array of `[string, boolean]` tuples.
Map these to the `GateBar` format:
```typescript
const gateItems = report.gates.map(([name, passed]) => ({
  name,
  status: passed ? 'pass' as const : 'fail' as const,
}));
```

##### StatCard layout

Use a 4-column grid matching the existing dashboard pages:
```tsx
<div style={{ display: 'grid', gridTemplateColumns: 'repeat(4, 1fr)', gap: 16 }}>
  <StatCard label="Result" value={report.success ? 'PASS' : 'FAIL'}
    color={report.success ? 'sage' : 'warn'} />
  <StatCard label="Model" value={report.model ?? '\u{2014}'} color="rose" />
  <StatCard label="Cost" value={report.cost_usd != null
    ? `$${report.cost_usd.toFixed(4)}` : '\u{2014}'} color="bone" />
  <StatCard label="Duration" value={report.duration_s != null
    ? `${report.duration_s.toFixed(1)}s` : '\u{2014}'} color="sage" />
</div>
```

##### Output section

Render agent output in a scrollable `<pre>` with `maxHeight: 400`:
```tsx
{report.output && (
  <pre className="pane" style={{
    padding: 16, fontSize: 12, color: 'var(--text-primary)',
    whiteSpace: 'pre-wrap', maxHeight: 400, overflow: 'auto',
  }}>
    {report.output}
  </pre>
)}
```
