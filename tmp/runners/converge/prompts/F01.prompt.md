## Batch F01: Remove roko-runtime from roko-core/Cargo.toml

### Write Scope
- **MODIFY**: `crates/roko-core/Cargo.toml` (remove `roko-runtime` dependency)
- **MODIFY**: `crates/roko-core/src/lib.rs` (remove any re-exports that require roko-runtime)

### DO NOT
- Modify any files outside roko-core/
- Add new dependencies to any Cargo.toml
- Remove the `pub mod runtime_event;` declaration — that module is already IN roko-core
- Remove the `pub mod foundation;` declaration
- Touch roko-runtime at all

### Existing Code Context

Current `crates/roko-core/Cargo.toml` (lines 13-28):

The `roko-runtime` line is the only thing creating the cycle. The `RuntimeEvent` and
`WorkflowOutcome` types are already defined in `crates/roko-core/src/runtime_event.rs`.
The `EventConsumer`, `ModelCaller`, `PromptAssembler`, `FeedbackSink`, `GateRunner` traits
are already defined in `crates/roko-core/src/foundation.rs`. roko-core does not need
roko-runtime for any of these.

Current `crates/roko-core/src/lib.rs` re-exports involving runtime_event (lines 219, from
the existing file):
```rust
pub use runtime_event::{RuntimeEvent, WorkflowOutcome};
```

These are re-exporting from `crates/roko-core/src/runtime_event.rs` — an internal module —
so they do NOT require roko-runtime and must be kept.

### Task

1. In `crates/roko-core/Cargo.toml`, delete the line:
   ```toml
   roko-runtime = { path = "../roko-runtime" }
   ```
   Leave all other dependencies exactly as they are.

2. In `crates/roko-core/src/lib.rs`, search for any re-exports that reference types coming
   from roko-runtime (i.e., `use roko_runtime::` or re-exports that only exist in that crate).
   If any exist, remove only those re-export lines. The internal modules `runtime_event` and
   `foundation` are part of roko-core itself — their re-exports must stay.

   Based on reading the actual lib.rs, no re-exports reference roko_runtime directly, so
   step 2 is expected to require no changes. Verify this by checking for any `roko_runtime`
   imports in `crates/roko-core/src/` before concluding.
