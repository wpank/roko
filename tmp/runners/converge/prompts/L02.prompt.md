# Batch L02 — Create layer-check binary

## Write Scope
- **CREATE**: `crates/roko-cli/src/layer_check.rs`
- **MODIFY**: `crates/roko-cli/src/lib.rs` (add `pub mod layer_check;`)
- **MODIFY**: `crates/roko-cli/Cargo.toml` (add `cargo_metadata` dependency if not already present)

## Task

Create a `layer_check` module that reads the workspace dependency graph via `cargo_metadata`, extracts the `[package.metadata.roko].layer` value from each crate, and verifies that no crate depends on a crate from a higher layer.

### 1. Add `cargo_metadata` dependency

In `crates/roko-cli/Cargo.toml`, add:
```toml
cargo_metadata = "0.19"
```

Check the latest compatible version. If `cargo_metadata` is already present, skip this step.

### 2. Create `crates/roko-cli/src/layer_check.rs`

```rust
//! Layer dependency checker.
//!
//! Reads `[package.metadata.roko].layer` from each workspace crate and
//! verifies that dependency edges only point from higher layers to equal
//! or lower layers (layer(dependent) >= layer(dependency)).

use anyhow::{Context, Result};
use std::collections::HashMap;
```

#### 2a. Core data structures

```rust
/// A single layer violation: crate at layer N depends on crate at layer M where M > N.
#[derive(Debug)]
pub struct LayerViolation {
    pub from_crate: String,
    pub from_layer: u32,
    pub to_crate: String,
    pub to_layer: u32,
}

impl std::fmt::Display for LayerViolation {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        write!(
            f,
            "L{} {} -> L{} {} (higher layer dependency)",
            self.from_layer, self.from_crate, self.to_layer, self.to_crate
        )
    }
}
```

#### 2b. Layer extraction

```rust
/// Extract layer assignments from workspace metadata.
fn extract_layers(metadata: &cargo_metadata::Metadata) -> HashMap<String, u32> {
    let mut layers = HashMap::new();
    for package in &metadata.packages {
        if let Some(roko_meta) = package.metadata.get("roko") {
            if let Some(layer) = roko_meta.get("layer").and_then(|v| v.as_u64()) {
                layers.insert(package.name.clone(), layer as u32);
            }
        }
    }
    layers
}
```

#### 2c. Violation detection

```rust
/// Check all workspace dependency edges for layer violations.
fn check_layers(metadata: &cargo_metadata::Metadata) -> Vec<LayerViolation> {
    let layers = extract_layers(metadata);
    let workspace_members: std::collections::HashSet<_> = metadata
        .workspace_members
        .iter()
        .map(|id| id.to_string())
        .collect();

    let mut violations = Vec::new();

    for package in &metadata.packages {
        // Only check workspace members
        if !workspace_members.contains(&package.id.to_string()) {
            continue;
        }
        let Some(&from_layer) = layers.get(&package.name) else {
            continue; // skip crates without layer metadata
        };

        for dep in &package.dependencies {
            let Some(&to_layer) = layers.get(&dep.name) else {
                continue; // external or unlabeled dependency
            };
            if from_layer < to_layer {
                violations.push(LayerViolation {
                    from_crate: package.name.clone(),
                    from_layer,
                    to_crate: dep.name.clone(),
                    to_layer,
                });
            }
        }
    }

    violations.sort_by(|a, b| {
        a.from_layer.cmp(&b.from_layer)
            .then(a.from_crate.cmp(&b.from_crate))
            .then(a.to_crate.cmp(&b.to_crate))
    });

    violations
}
```

#### 2d. Public entry point

```rust
/// Run the layer check and return the process exit code (0 = pass, 1 = violations found).
pub fn run_layer_check() -> Result<i32> {
    let metadata = cargo_metadata::MetadataCommand::new()
        .exec()
        .context("failed to run `cargo metadata`")?;

    let layers = extract_layers(&metadata);
    let workspace_count = metadata.workspace_members.len();
    let labeled_count = metadata
        .packages
        .iter()
        .filter(|p| metadata.workspace_members.contains(&p.id))
        .filter(|p| layers.contains_key(&p.name))
        .count();

    println!("Layer check: {labeled_count}/{workspace_count} crates have layer metadata");

    if labeled_count == 0 {
        println!("WARNING: No crates have [package.metadata.roko].layer set. Run L01 first.");
        return Ok(1);
    }

    // Print layer summary
    let mut by_layer: HashMap<u32, Vec<&str>> = HashMap::new();
    for (name, &layer) in &layers {
        by_layer.entry(layer).or_default().push(name.as_str());
    }
    for layer in 0..=4 {
        if let Some(crates) = by_layer.get(&layer) {
            let mut names: Vec<_> = crates.to_vec();
            names.sort();
            println!("  L{layer}: {}", names.join(", "));
        }
    }

    let violations = check_layers(&metadata);

    if violations.is_empty() {
        println!("\nNo layer violations found.");
        Ok(0)
    } else {
        println!("\nFound {} layer violation(s):\n", violations.len());
        for v in &violations {
            println!("  ERROR: {v}");
        }
        Ok(1)
    }
}
```

### 3. Register module in lib.rs

Add `pub mod layer_check;` in alphabetical order in `crates/roko-cli/src/lib.rs`.

### 4. Wire as CLI subcommand

Find the CLI dispatch file (likely `crates/roko-cli/src/main.rs` or `crates/roko-cli/src/run.rs` where clap subcommands are defined). Add a `LayerCheck` variant and dispatch to `layer_check::run_layer_check()`.

The subcommand should be `roko layer-check` (with hyphen). If the clap setup uses an enum, add:
```rust
/// Check workspace layer dependency rules
LayerCheck,
```

In the dispatch match arm:
```rust
Command::LayerCheck => {
    let code = roko_cli::layer_check::run_layer_check()?;
    std::process::exit(code);
}
```
