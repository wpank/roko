#### 3. Add `format_duration` helper

Add a private helper function in `run.rs` (or inside `run_with_workflow_engine`)
that formats a `std::time::Duration` for human display:

```rust
/// Format a duration for human display: "3.2s", "1m 42s", "0.8s".
fn format_duration(d: std::time::Duration) -> String {
    let secs = d.as_secs_f64();
    if secs < 60.0 {
        format!("{secs:.1}s")
    } else {
        let mins = secs as u64 / 60;
        let remaining = secs as u64 % 60;
        format!("{mins}m {remaining}s")
    }
}
```

closes cleanly:
