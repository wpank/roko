
`CascadeSelection.primary` is the model slug to use. `CascadeStage` can be `Static`,
`Confidence`, or `Ucb1`.

`Option<String>` — convert it to `AgentRole` by parsing with `serde_json` or by matching the
label strings (use `AgentRole::ALL_AGENTS` iterator).

3. In `resolve_model`, when the request model is empty AND a router is configured, call
   the router to select a model. Pass the `role` field from the request as the role hint:
   ```rust
   fn resolve_model(&self, req: &ModelCallRequest) -> String {
       if req.model.is_empty() {
           if let Some(router) = &self.cascade_router {
               return router.select_for_role(req.role.as_deref());
           }
           self.default_model.clone()
       } else {
           req.model.clone()
       }
   }
   ```
