### Existing Code Context

`AgentOptions` already has the `mcp_config` field:

The `ClaudeCliAdapter` reads `options.mcp_config` and passes it as `--mcp-config <path>` to
the `claude` CLI subprocess. The path must point to a valid MCP JSON config file.

Current `ModelCallService` struct (all fields):

Current `call` method builds `AgentOptions` like this (abbreviated):
```rust
let mut options = AgentOptions {
    system_prompt,
    name: req.role.clone().unwrap_or_else(|| "model_call".to_string()),
```

The `RokoConfig` agent section also has an `mcp_config` field:
```rust
// In roko_core::config::schema::AgentConfig:
// pub mcp_config: Option<PathBuf>,
```

Precedence order to implement (highest priority first):
1. MCP config explicitly set via `with_mcp_config()` on the service
2. MCP config from `self.config.agent.mcp_config` (loaded from `roko.toml`)
3. None (no `--mcp-config` passed)

### Task

1. Add an `mcp_config: Option<PathBuf>` field to `ModelCallService`.

2. Add a `with_mcp_config(path: impl Into<PathBuf>) -> Self` builder method:
   ```rust
   #[must_use]
   pub fn with_mcp_config(mut self, path: impl Into<PathBuf>) -> Self {
       self.mcp_config = Some(path.into());
       self
   }
   ```

3. In the `call` method, after building the initial `AgentOptions`, set `options.mcp_config`
   following the precedence above:
   ```rust
   options.mcp_config = self.mcp_config.clone()
       .or_else(|| self.config.agent.mcp_config.clone());
   ```

4. Update `new()` to initialize `mcp_config: None`.

5. Add a unit test `mcp_config_is_threaded_to_agent_options`:
   - Construct a `ModelCallService::new("claude".into()).with_mcp_config("/tmp/mcp.json")`
   - Call `resolve_model` to confirm model resolution still works
   - Directly inspect the logic path: build `AgentOptions` and confirm `mcp_config` is
     `Some(PathBuf::from("/tmp/mcp.json"))` using a helper method or by extracting the
     option-building logic into a testable `build_agent_options(&self, req)` method

6. Add a unit test `mcp_config_falls_back_to_roko_config`:
   - Create a `RokoConfig` with `agent.mcp_config = Some(PathBuf::from("/roko/mcp.json"))`
   - Build `ModelCallService::new("m".into()).with_config(config)` (no `with_mcp_config`)
   - Confirm `AgentOptions.mcp_config` resolves to the config value
