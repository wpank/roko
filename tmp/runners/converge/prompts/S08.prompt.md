Current `assemble` method (uses `SystemPromptBuilder`):
```rust
#[async_trait]
impl PromptAssembler for PromptAssemblyService {
    async fn assemble(&self, spec: PromptSpec) -> Result<String> {
        let role = resolve_role(spec.role.as_deref());
        let identity = role_identity_for(role);

        let mut builder = SystemPromptBuilder::new(identity);

        if let Some(conventions) = conventions_for_spec(&spec, self.default_conventions.as_deref()) {
            builder = builder.with_conventions(conventions);
        }

        if let Some(task) = spec.task {
            builder = builder.with_task(task);
        }

        for feedback in spec.gate_feedback {
            builder = builder.with_gate_feedback_text(feedback);
        }

        if !spec.anti_patterns.is_empty() {
            builder = builder.with_anti_patterns(spec.anti_patterns);
        }

        Ok(builder.build())
    }
}
```

### Task

   ```rust

       default_conventions: Option<String>,
   ```

3. Add corresponding builder methods:
   ```rust

   pub fn with_playbook_context(mut self, store: Arc<PlaybookStore>) -> Self
   ```

4. In `assemble`, after the conventions block and before the task block, inject episode
   context. Read the last 5 episodes from the JSONL file and format them as a context
   summary:
   ```rust
   if let Some(ref episodes_path) = self.episodes_path {
       if let Ok(episodes) = EpisodeLogger::read_all(episodes_path).await {
           let recent: Vec<_> = episodes.into_iter().rev().take(5).collect();
           if !recent.is_empty() {
               let summary = format_episode_context(&recent);
               builder = builder.with_context(summary);
           }
       }
   }
   ```

5. Write a helper `format_episode_context(episodes: &[Episode]) -> String` that produces
   a compact context block like:
   ```
   ## Recent Execution History (last 5 runs)
   - [SUCCESS] task=T-042 model=sonnet duration=12.3s tokens=1500
   - [FAILED]  task=T-041 model=opus  duration=45.1s gate=compile
   ...
   ```
   Include: success/fail, task_id, model, duration_secs, and first failed gate name if any.

6. In `assemble`, after episode context, inject playbook context. Query the store using the
   task from the spec and inject matching playbooks via `builder.with_relevant_playbooks()`:
   ```rust
   if let Some(ref store) = self.playbook_store {
       let task_text = spec.task.as_deref().unwrap_or("");
       let ctx = QueryContext {
           task_id: String::new(),
           task_title: task_text.to_string(),
           task_body: String::new(),
           role: spec.role.clone().unwrap_or_default(),
           recent_episodes: 0,
           max_results: 3,
       };
       if let Ok(playbooks) = store.query(&ctx).await {
           if !playbooks.is_empty() {
               builder = builder.with_relevant_playbooks(playbooks);
           }
       }
   }
   ```
