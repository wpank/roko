```rust

impl CascadeRouter {
    /// Record a reward observation for a model from a raw context vector.
    pub fn observe(&self, context_vec: Vec<f64>, model_idx: usize, reward: f64);

```

The `observe` method takes:
- `context_vec`: a feature vector describing the task context
- `model_idx`: index into `model_slugs` identifying which model was used
- `reward`: scalar reward signal (1.0 for success, 0.0 for failure is simplest)

To find the `model_idx`, you need to look up the model slug in `model_slugs`. The
`CascadeRouter` has a public method or you can access `model_slugs` — check the struct.
If `model_slugs` is private, use the `model_index` helper if one exists, or build a
simple context vector and reward signal without the index lookup (log a warning if the
model is not found).

4. Write the observation helper as a private method on `FeedbackService`:
   ```rust
   fn observe_model_call(&self, model: &str, success: bool, role: &str, latency_ms: u64) {
       let Some(ref router) = self.cascade_router else { return };

       // Build a minimal context vector for the bandit.
       // Features: [role_hash_normalized, latency_normalized, 1.0 (bias)]
       let role_feature = simple_role_hash(role);
       let latency_feature = (latency_ms as f64 / 60_000.0).min(1.0); // normalize to [0, 1]
       let context_vec = vec![role_feature, latency_feature, 1.0];

       // Find model index in the router's slug list
       let model_idx = match router.model_index(model) {
           Some(idx) => idx,
           None => {
               tracing::debug!("model {model} not in cascade router slug list, skipping observe");
               return;
           }
       };

       let reward = if success { 1.0 } else { 0.0 };
       router.observe(context_vec, model_idx, reward);
   }
   ```

       let hash: u32 = role.bytes().fold(0u32, |acc, b| acc.wrapping_mul(31).wrapping_add(b as u32));
       (hash % 1000) as f64 / 1000.0
