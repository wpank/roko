### Task

#### 1. Add an `EngineVariant` enum

Add this enum immediately before or after the `Command` enum definition. Do not
put it inside `Command`:

```rust
/// Which execution engine to use for `roko run`.
#[derive(Debug, Clone, Copy, PartialEq, Eq, clap::ValueEnum, Default)]
pub enum EngineVariant {
    /// WorkflowEngine v2 (event-driven, default).
    #[default]
    V2,
    /// Legacy run_once() path (PlanRunner + orchestrate.rs).
    Legacy,
}
```

The enum must derive `clap::ValueEnum` so clap can parse `--engine v2` and
`--engine legacy` from the command line. `Default` is needed so the field can
omit an explicit `default_value` in the arg definition.

#### 2. Add `--engine` field to `Command::Run`

Add the new field to the `Run` variant after `share`:

```rust
Run {
    prompt: String,
    #[arg(long)]
    workdir: Option<PathBuf>,
    #[arg(long)]
    serve: bool,
    #[arg(long)]
    share: bool,
    /// Execution engine: v2 (WorkflowEngine, default) or legacy (run_once).
    #[arg(long, value_enum, default_value_t = EngineVariant::V2)]
    engine: EngineVariant,
},
```

#### 3. Thread `engine` through the dispatch arm

Update the `Command::Run` arm to destructure and pass the new field:

```rust
Command::Run {
    prompt,
    workdir,
    serve,
    share,
    engine,
} => commands::util::cmd_run(cli, workdir, prompt, serve || share, engine).await,
```

#### 4. Update `cmd_run` signature in `commands/util.rs`

The `commands::util::cmd_run` function currently has this signature:
```rust
pub(crate) async fn cmd_run(
    cli: &Cli,
    workdir: Option<PathBuf>,
    prompt: String,
    serve: bool,
) -> Result<i32>
```

Update it to accept the engine variant (but do NOT implement the dispatch logic
yet — that is W02's job):

```rust
pub(crate) async fn cmd_run(
    cli: &Cli,
    workdir: Option<PathBuf>,
    prompt: String,
    serve: bool,
    _engine: crate::EngineVariant,
) -> Result<i32>
```

The `_engine` parameter is intentionally unused at this stage. W02 removes the
leading underscore and adds the branch.
