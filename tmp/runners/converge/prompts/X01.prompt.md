# Batch X01 — Fix contract fail-open to fail-closed

## Write Scope
- **MODIFY**: `crates/roko-agent/src/safety/mod.rs`

## Task

Change the `contract_for_role()` method in the safety module to use the restricted (deny-all) fallback instead of the permissive (allow-all) fallback when no contract YAML exists for a role. This is a security hardening change -- agents without explicit contracts should have zero capabilities, not unlimited ones.

### 1. Replace the fallback in `contract_for_role()`

Current code at line ~864:
```rust

    let mut contract = AgentContract::load_for_role(role).unwrap_or_else(|err| {
        tracing::warn!(
            %role,
            %err,
            "no contract for role; using permissive default"
        );
        AgentContract::permissive(role.to_string())
    });
    // ... role override budget injection follows ...
    contract
}
```

Replace with:
```rust
fn contract_for_role(&self, role: &str) -> AgentContract {
    let mut contract = AgentContract::load_for_role_with_mode(
        role,
        ContractLoadMode::RestrictedFallback,
    )
    .unwrap_or_else(|err| {
        // load_for_role_with_mode with RestrictedFallback should never
        // error (it returns Ok(restricted) on missing asset), but handle
        // the impossible case defensively.
        tracing::error!(
            %role,
            %err,
            "contract load failed even with restricted fallback; using deny-all"
        );
        AgentContract::restricted(role.to_string())
    });
    // ... rest of the method (role override budget injection) unchanged ...
    contract
}
```

pub fn load_for_role_with_mode(
    role: impl AsRef<str>,
    mode: ContractLoadMode,
) -> Result<Self, ContractLoadError> {
    match Self::load_for_role(role_ref) {
        Ok(contract) => Ok(contract),
        Err(err) => match mode {
            ContractLoadMode::Strict => Err(err),
            ContractLoadMode::RestrictedFallback => {
                tracing::warn!(role = %role_ref, %err, "no contract for role; using restricted (deny-all) fallback");
                Ok(Self::restricted(role_ref))
            }
        },
    }
}
