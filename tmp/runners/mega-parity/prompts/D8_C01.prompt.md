## Problem
`AppState` has a `cascade_router: RwLock<Option<CascadeRouter>>` field that is always initialized to `None` (see `state.rs:541`). The `CascadeRouter` persists its state to `.roko/learn/cascade-router.json` and the CLI orchestrator loads it via `CascadeRouter::load_or_new()`, but the HTTP server never loads this persisted state. This means every gateway inference request that hits `select_model_via_router()` constructs a temporary router from scratch, discarding all accumulated bandit observations. Adaptive gate thresholds from `.roko/learn/gate-thresholds.json` are similarly never loaded into the server.

## Architecture Contract
- The CascadeRouter is thread-safe: wrap in `Arc` for shared access, or use the existing `RwLock<Option<CascadeRouter>>` on AppState.
- Loading must be non-blocking and failure-tolerant: if the JSON file is missing or corrupted, log a warning and fall back to a fresh default router.
- Do NOT change the AppState struct definition — the `cascade_router: RwLock<Option<CascadeRouter>>` field already exists.
- The `build_app_state()` function in `lib.rs` is the single initialization point for server state; wire it there.

### Step 1 — Load CascadeRouter in build_app_state (lib.rs)

After the `AppState::new(...)` call and before the StateHub seeding, load the persisted CascadeRouter and store it into the state.

Add after `AppState::new(...)` but before `bootstrap_from_workdir`:
```rust
    let deploy_backend = create_deploy_backend(&roko_config);
    let state = AppState::new(workdir, runtime, roko_config, deploy_backend);

    // Load persisted CascadeRouter so the gateway has accumulated bandit state.
    {
        let config = state.load_roko_config();
        let effective_models = config.effective_models();
        let mut model_slugs: Vec<String> = effective_models
            .values()
            .filter(|p| !p.is_embedding_model)
            .map(|p| p.slug.clone())
            .collect();
        model_slugs.sort();

        if !model_slugs.is_empty() {
            let router_path = state.workdir.join(".roko").join("learn").join("cascade-router.json");
            let router = roko_learn::cascade_router::CascadeRouter::load_or_new(&router_path, model_slugs);
            let obs = router.total_observations();
            if obs > 0 {
                info!(observations = obs, path = %router_path.display(), "loaded persisted CascadeRouter");
            } else {
                debug!("initialized fresh CascadeRouter (no persisted state)");
            }
            // Use block_in_place because we hold no async locks here.
            tokio::task::block_in_place(|| {
                *state.cascade_router.blocking_write() = Some(router);
            });
        }
    }

    // Load adaptive gate thresholds so the serve layer can surface them.
    {
        let thresholds_path = state.workdir.join(".roko").join("learn").join("gate-thresholds.json");
        if thresholds_path.exists() {
            info!(path = %thresholds_path.display(), "gate thresholds file found (available for serve routes)");
        }
    }

    let _ = state.state_hub.bootstrap_from_workdir(&state.workdir);
```

Replace with:

        tracing::info!("server shutdown initiated");
        // Persist CascadeRouter observations so they survive restarts.
        if let Some(ref router) = *self.cascade_router.read().await {
            let path = self.workdir.join(".roko").join("learn").join("cascade-router.json");
            if let Err(err) = router.save(&path) {
                tracing::warn!(error = %err, "failed to save CascadeRouter on shutdown");

            }

    }
