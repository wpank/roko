## Problem
The gateway `inference_complete` handler currently publishes a single `ServerEvent::AgentOutput` event after inference completes. This conflates inference telemetry with agent output streaming and provides no structured event for inference start, model selection, or cost/latency tracking. Dashboards (TUI and web) need distinct, typed events to show real-time inference activity: which model was selected, how long the call took, and what it cost. The `ServerEvent` enum needs new variants for inference lifecycle, and the gateway handler needs to emit them.

## Architecture Contract
- `ServerEvent` variants are `#[serde(tag = "type", rename_all = "snake_case")]` — new variant names must be snake_case.
- `EventBus::publish()` is non-blocking: `pub fn publish(&self, event: E)` (event_bus.rs:82).
- New variants must be serializable and include enough context for dashboard display.
- The existing `AgentOutput` event should remain for backward compatibility — add new events alongside it.
- Events are streamed to SSE/WS consumers, so keep payloads compact (no full message bodies).

## Changes Required

### Step 1 — Add new ServerEvent variants for inference lifecycle

**File:** `crates/roko-serve/src/events.rs`

Add these variants to the `ServerEvent` enum, before the `ServerShutdown` variant:

```rust
    /// A gateway inference request was started.
    InferenceStarted {
        /// Unique request identifier.
        request_id: String,
        /// Model slug selected for this request.
        model: String,
        /// Agent that initiated the request.
        #[serde(default)]
        agent_id: String,
        /// Whether the model was explicitly requested or auto-selected.
        #[serde(default)]
        auto_routed: bool,
    },

    /// A gateway inference request completed successfully.
    InferenceCompleted {
        /// Unique request identifier.
        request_id: String,
        /// Model that actually served the request.
        model: String,
        /// Agent that initiated the request.
        #[serde(default)]
        agent_id: String,
        /// Input tokens consumed.
        input_tokens: u64,
        /// Output tokens generated.
        output_tokens: u64,
        /// Estimated cost in USD.
        cost_usd: f64,
        /// Wall-clock duration in milliseconds.
        duration_ms: u64,
    },

    /// A gateway inference request failed.
    InferenceFailed {
        /// Unique request identifier.
        request_id: String,
        /// Model that was targeted.
        model: String,
        /// Agent that initiated the request.
        #[serde(default)]
        agent_id: String,
        /// Error description.
        error: String,
    },
```
