## Problem
Roko's neuro knowledge store is a key differentiator — it grows across runs, so repeated execution makes the system smarter. No existing scenario demonstrates this accumulation. Users need to see the knowledge store start empty, grow after a task run, grow again after a second run, and see how queries return richer results over time.

## Architecture Contract
- Scenarios MUST implement the `Scenario` interface from `scenarios.ts`
- Use the existing terminal handle API (`execCmd`, `typeCmd`, `waitForPrompt`) via `showCmd`
- Use `playback.waitForStep()` for timing between steps
- All scenarios must work with both live server and simulated output
- Add the scenario to the `SCENARIOS` array export
- Use `setupWorkspace` / `joinWorkspace` from `useTerminalSession`
- 2-pane layout: left runs tasks, right queries knowledge

## Current State
The existing `explore` scenario (scenarios.ts ~line 562) runs `knowledge stats` and `knowledge query "routing"` as one-off commands in a 4-pane explorer. It does not show accumulation across multiple runs. The `selfhost` scenario (~line 360) runs a single PRD cycle but never queries the knowledge store.

The `showCmd` helper returns `CommandResult` with `{ ok, elapsed, gates, cost, tokens }`. The `setMetric` context method updates sidebar mosaic cells. The `logCommand` method adds entries to the command log panel.

### Step 1 — Define the scenario object
Add this scenario constant before the `SCENARIOS` array export:

```typescript
const knowledgeAccumulation: Scenario = {
  id: 'knowledge-accumulation',
  title: 'Knowledge Growth',
  subtitle: 'Watch the knowledge store grow across successive runs.',
  panes: 2,
  labels: ['task runner', 'knowledge store'],
  panel: true,
  promptBar: false,
  steps: [
    { label: 'Initial query', sublabel: 'empty store' },
    { label: 'Run 1', sublabel: 'build a CLI tool' },
    { label: 'Knowledge check', sublabel: 'query after run 1' },
    { label: 'Run 2', sublabel: 'add error handling' },
    { label: 'Knowledge growth', sublabel: 'query after run 2' },
    { label: 'Final state', sublabel: 'knowledge stats' },
  ],
  async run({ entries, playback, timeline, setMetric, logCommand }) {
    const [runner, knowledge] = entries;

    const dir = await setupWorkspace(runner, 'roko-knowledge');
    await joinWorkspace(knowledge, dir);
    const ROKO = getRoko();

    timeline.init(this.steps);
    setMetric('model', 'cascade');

    // Step 0: Initial query — show empty store
    await playback.waitForStep();
    timeline.setActive(0);
    playback.setProgress(1, 6, `${ROKO} knowledge stats`);
    logCommand('knowledge stats', 'Checking the initial state of the neuro knowledge store — should be empty.');
    await showCmd(knowledge, `${ROKO} knowledge stats`, { timeout: 30000, onLog: logCommand });
    await showCmd(knowledge, `${ROKO} knowledge query "error handling patterns"`, {
      timeout: 30000,
      onLog: logCommand,
      customDesc: 'Querying for "error handling patterns" — the store is empty, so no results expected.',
    });
    setMetric('tokens', '0 entries');

    // Step 1: Run 1 — build something to generate knowledge
    await playback.waitForStep();
    timeline.setActive(1);
    playback.setProgress(2, 6, `${ROKO} run "Build a Rust CLI..."`);
    logCommand('run 1', 'First task run — the agent builds code. Episodes and efficiency data flow into the knowledge store.');
    await showCmd(runner, `${ROKO} run "Build a Rust CLI that parses JSON from stdin"`, {
      timeout: 180000,
      onLog: logCommand,
      customDesc: 'First run: builds a JSON parser CLI. Agent episodes, gate results, and efficiency events all persist to the knowledge store.',
    });
    setMetric('cost', '$ run 1');

    // Step 2: Knowledge check after run 1
    await playback.waitForStep();
    timeline.setActive(2);
    playback.setProgress(3, 6, `${ROKO} knowledge query ...`);
    knowledge.clearTerminal();
    logCommand('knowledge check 1', 'Querying the store after the first run — should now have entries about Rust, JSON parsing, and CLI patterns.');
    await showCmd(knowledge, `${ROKO} knowledge stats`, { timeout: 30000, onLog: logCommand });
    await showCmd(knowledge, `${ROKO} knowledge query "JSON parsing"`, {
      timeout: 30000,
      onLog: logCommand,
      customDesc: 'After run 1 the store should contain entries about JSON parsing learned from the first task.',
    });
    setMetric('tokens', 'growing');

    // Step 3: Run 2 — build more, growing the store
    await playback.waitForStep();
    timeline.setActive(3);
    playback.setProgress(4, 6, `${ROKO} run "Add error handling..."`);
    runner.clearTerminal();
    logCommand('run 2', 'Second task — adds error handling to the existing code. Knowledge compounds with the first run.');
    await showCmd(runner, `${ROKO} run "Add comprehensive error handling with anyhow and thiserror"`, {
      timeout: 180000,
      onLog: logCommand,
      customDesc: 'Second run: adds error handling. The agent now has prior knowledge from run 1 to inform its approach.',
    });
    setMetric('cost', '$ run 1+2');

    // Step 4: Knowledge growth — query shows richer results
    await playback.waitForStep();
    timeline.setActive(4);
    playback.setProgress(5, 6, `${ROKO} knowledge query ...`);
    knowledge.clearTerminal();
    logCommand('knowledge check 2', 'After two runs the store should have accumulated entries across both tasks.');
    await showCmd(knowledge, `${ROKO} knowledge stats`, { timeout: 30000, onLog: logCommand });
    await showCmd(knowledge, `${ROKO} knowledge query "error handling patterns"`, {
      timeout: 30000,
      onLog: logCommand,
      customDesc: 'The same query that returned nothing initially now returns entries learned across both runs.',
    });
    setMetric('tokens', 'accumulated');

    // Step 5: Final state — full stats
    await playback.waitForStep();
    timeline.setActive(5);
    playback.setProgress(6, 6, `${ROKO} knowledge stats`);
    knowledge.clearTerminal();
    logCommand('final state', 'Full knowledge store statistics after two accumulation cycles.');
    await showCmd(knowledge, `${ROKO} knowledge stats`, {
      timeout: 30000,
      onLog: logCommand,
      customDesc: 'Final stats: shows entry count, tier distribution, memory usage, and last distillation time.',
    });
    await showCmd(knowledge, `${ROKO} learn all`, {
      timeout: 30000,
      onLog: logCommand,
      customDesc: 'Shows all learning state: cascade router weights now reflect experience from both runs.',
    });
    setMetric('model', 'accumulated');

    timeline.setActive(6); // all completed
  },
};
```
