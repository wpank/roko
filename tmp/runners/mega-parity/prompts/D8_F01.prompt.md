## Important Design Notes

- Task prompts should include **starter code** that the agent finds in the workdir (D8_F03 scaffolds this)
- Each prompt should tell the agent to run `cargo test` or `cargo check` as the final step
- `expected_output` validation is **substring match** on the agent's final output text
- Keep prompts under 500 words each â€” small models struggle with long instructions
- Tag all tasks with `"learnable"` for filtering
