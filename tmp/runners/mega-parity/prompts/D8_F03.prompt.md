## Task

Wire bench workdir scaffold: initialize a temporary Cargo project before each suite run so that agent tasks can compile and test code.

## Runner Context

You are working in runner `mega-parity`, batch D8_F03.
Part of D8F — Bench Self-Learning Pipeline.

## Problem

Bench tasks like "implement a function and run `cargo test`" need a valid Cargo project in the working directory. Currently `execute_bench_run` uses `state.workdir` directly, which is the roko workspace root — not a clean project for benchmark tasks. Each bench run needs an isolated temp directory with a pre-initialized `Cargo.toml` and starter code.

## Changes Required

### Step 1 — Create bench workdir scaffold function

Add to `routes/bench.rs` (or a helper module):

```rust
/// Create a temporary Cargo project for benchmark execution.
/// Returns the path to the temp directory.
async fn scaffold_bench_workdir(suite_id: &str, run_id: &str) -> anyhow::Result<PathBuf> {
    let dir = std::env::temp_dir().join(format!("roko-bench-{run_id}"));
    tokio::fs::create_dir_all(&dir).await?;

    // Write Cargo.toml
    tokio::fs::write(dir.join("Cargo.toml"), /* minimal Cargo.toml */);

    // Write starter src/lib.rs with TODO stubs, broken imports, etc.
    // that the learnable-rust tasks reference
    tokio::fs::create_dir_all(dir.join("src")).await?;
    tokio::fs::write(dir.join("src/lib.rs"), /* starter code */);
    tokio::fs::write(dir.join("src/main.rs"), /* minimal main */);

    Ok(dir)
}
```

### Step 2 — Wire into execute_bench_run

Before the task loop, call `scaffold_bench_workdir`. Pass the temp dir to `run_once_with_config` instead of `state.workdir`.

### Step 3 — Write starter code for learnable-rust tasks

The `src/lib.rs` should contain:
- A function with a `// TODO: implement` comment (for task 1)
- Duplicated loop logic (for task 2 — extract helper)
- A wrong `use` import (for task 3 — fix import)
- An empty module `helpers.rs` that defines the missing type
- A `CountUp` struct stub (for task 5)

### Step 4 — Cleanup

After bench run completes (success or failure), delete the temp directory.
