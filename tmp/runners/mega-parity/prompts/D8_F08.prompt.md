## Problem

The Context-Enriched strategy is the first enrichment tier above Minimal. It enables:
- **Playbook injection**: Query `PlaybookStore` for relevant playbooks, render them into the system prompt Layer 6 ("Relevant Techniques")

It does NOT enable:
- Knowledge store queries (strategy fragments, anti-patterns)
- Gate failure replan

This strategy demonstrates how learned playbooks from previous runs improve performance.
