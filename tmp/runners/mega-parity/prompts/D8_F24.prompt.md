Build `KnowledgeDeltaPanel`: show what playbooks and anti-patterns were created or injected per run.

## Problem

The demo needs a panel that shows the "knowledge delta" — what the system learned between runs. This makes the self-improvement tangible: "Run 2 created 2 playbooks and 1 anti-pattern from Run 1's results."

## Changes Required

### Step 1 — Create component files

Create `demo/demo-app/src/components/KnowledgeDeltaPanel.tsx` and `.css`.

### Step 2 — Define interface

```typescript
interface KnowledgeDelta {
  runId: string;
  runIndex: number;
  strategy: string;
  playbooks_created: number;
  playbooks_used: number;
  anti_patterns_created: number;
  anti_patterns_used: number;
  knowledge_entries_queried: number;
}

interface KnowledgeDeltaPanelProps {
  deltas: KnowledgeDelta[];
  currentRunIndex?: number;
}
```

### Step 3 — Panel layout

A vertical timeline showing each run's learning delta:

```
Run 1 (Minimal)
  No learning enabled

Run 2 (Context-Enriched)
  📖 Used 2 playbooks from Run 1
  ✨ Created 1 new playbook

Run 3 (Neuro-Augmented)
  📖 Used 3 playbooks
  🧠 Queried 5 knowledge entries
  ⚠️ Used 1 anti-pattern from Run 2's failure
  ✨ Created 2 new playbooks

Run 4 (Full Cascade)
  📖 Used 4 playbooks
  🧠 Queried 8 knowledge entries
  ⚠️ Used 2 anti-patterns
  🔄 1 gate-failure replan
  ✨ Created 1 new playbook
```

### Step 4 — Visual design

- Timeline on the left (vertical line with dots for each run)
- Strategy badge next to run number (colored to match LearningCurveChart colors)
- Metrics as compact lines with icons
- Current run highlighted with a glow effect
- Empty state: "No learning data yet. Run a benchmark to start learning."

### Step 5 — CSS

Match the demo app's dark theme:
- Background: transparent or `rgba(170, 112, 136, 0.05)`
- Text: `#d4c4cf` (light) / `#6a5a68` (dim)
- Timeline line: `#3a2a38`
- Active run dot: `#AA7088` with glow
- Strategy colors matching the chart
