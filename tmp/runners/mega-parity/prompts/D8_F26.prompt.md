## Task

Show strategy badge and learning indicators in the Live tab activity feed.

## Runner Context

You are working in runner `mega-parity`, batch D8_F26.
Part of D8F — Bench Self-Learning Pipeline.

**Depends on:** D8_F21 (BenchLearningEvent handler in useBench)

## Problem

The Live tab shows a real-time activity feed of bench tasks as they run. Currently it shows task name, status (pass/fail), and duration. It should also show:
1. The strategy being used (as a colored badge)
2. Learning events as they happen (e.g., "+1 playbook created" after a successful task)

## Changes Required

### Step 1 — Add strategy badge to task cards

In the Live tab's task grid or activity feed, add a strategy badge:

```tsx
<span className={`strategy-badge strategy-${config.strategy}`}>
  {config.strategy.replace('_', ' ')}
</span>
```

### Step 2 — Show learning events in activity feed

When a `BenchLearningEvent` is received (from `learningState` in useBench), add an entry to the activity feed:

```tsx
{learningState && learningState.playbooks_created > 0 && (
  <div className="feed-item feed-learning">
    <span className="feed-icon">+</span>
    <span>{learningState.playbooks_created} playbook{learningState.playbooks_created > 1 ? 's' : ''} created</span>
  </div>
)}
{learningState && learningState.anti_patterns_created > 0 && (
  <div className="feed-item feed-learning">
    <span className="feed-icon">!</span>
    <span>{learningState.anti_patterns_created} anti-pattern{learningState.anti_patterns_created > 1 ? 's' : ''} recorded</span>
  </div>
)}
```
