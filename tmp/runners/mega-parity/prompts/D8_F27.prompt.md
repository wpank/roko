### Step 3 — Make the progression realistic

Each run's results should tell a story:
- **Run 1**: Only simple tasks pass. Complex tasks fail with compile errors.
- **Run 2**: The playbooks from Run 1's successes (implement_function, fix_import) help extract_helper pass. error_handling still fails.

### Step 4 — Add realistic task results

- Cost based on Cerebras pricing ($0.0001/1K tokens)
- Error messages for failed tasks that look like real Rust compiler errors
