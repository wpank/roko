## Problem

`roko init` currently emits a config template that has no `schema_version`, no `[providers.*]` block, and no `[models.*]` block. It also emits a placeholder `[[gate]]` entry with `program = "true"` that always passes. This means every workspace initialized with `roko init` is technically a v1 config from the moment it is created.

The template is produced by `Config::default_toml_template(cloud: bool)` in `crates/roko-cli/src/config.rs`. `cmd_init` in `crates/roko-cli/src/commands/util.rs` calls it at line 168: `let default = Config::default_toml_template(cloud)?;`

```rust
    /// Render the default `roko.toml` template used by `roko init`.
    pub fn default_toml_template(cloud: bool) -> Result<String> {
        let mut config = Self::default();
        // Use "claude" as the default agent command for init — not the struct
        // default ("cat") which is a safe no-op for tests.  Users running
        // `roko init` expect a working config out of the box.
        config.agent.command = "claude".into();
        // Clear the placeholder "true" gate so the generated config does not
        // contain a gate that always passes.  Profile-specific real gates are
        // added by `roko init --profile` (R2_A05).
        config.gates = Vec::new();
        if cloud {
            config.log_format = Some("json".to_string());
            config.bind = Some("0.0.0.0".to_string());
            config.data_dir = Some(PathBuf::from("/data/.roko"));
        }
        let rendered = config.to_toml()?;
        let cloud_deploy = if cloud {
            "\n# Auto-register webhooks after deploy\n\
             [[serve.deploy.webhooks]]\n\
             provider = \"github\"\n\
             owner = \"nunchi\"\n\
             repo = \"roko\"\n\
             \n\
             [[serve.deploy.webhooks]]\n\
             provider = \"github\"\n\
             owner = \"nunchi\"\n\
             repo = \"collaboration\"\n"
        } else {
            ""
        };

        // V2: provider and model registry blocks.
        let provider_block = if claude_on_path() {
            "\n[providers.claude_cli]\nkind = \"claude_cli\"\ncommand = \"claude\"\n"
                .to_string()
        } else {
            concat!(
                "\n# [providers.claude_cli]",
                "  # uncomment after installing the claude CLI\n",
                "# kind = \"claude_cli\"\n",
                "# command = \"claude\"\n",
            )
            .to_string()
        };
        let model_block =
            "\n[models.claude-sonnet-4-6]\nprovider = \"claude_cli\"\n".to_string();

        Ok(format!(
            "schema_version = 2\n\
             \n\
             # REQUIRED_ENV\n\
             # Required environment variables (set in .env or shell):\n\
             # GITHUB_TOKEN       — GitHub personal access token (for MCP GitHub server)\n\
             # GITHUB_WEBHOOK_SECRET — GitHub webhook secret for deploy registration\n\
             # SLACK_BOT_TOKEN    — Slack bot token (for MCP Slack server)\n\
             # SLACK_SIGNING_SECRET — Slack webhook signing secret\n\
             # ANTHROPIC_API_KEY  — Claude API key (for direct API agents, not needed for CLI agents)\n\
             \n\
             {rendered}{cloud_deploy}{provider_block}{model_block}\n\
             # PRD settings (parsed by `RokoConfig`)\n\
             [prd]\n\
             auto_plan = false\n"
        ))
    }
```
