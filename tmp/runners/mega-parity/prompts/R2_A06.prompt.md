## Problem

## Architecture Contract

- State views agree — if warnings are printed, the count must match
- Honest reporting is a prerequisite for automated validation

## Changes Required

- Final output line must match actual warnings: "N warnings" where N equals the number of warning lines printed
- Categorize warnings: schema warnings, field warnings, migration warnings
- Each category is counted and reported separately in the summary
