## Problem

`GateVerdict` in `foundation.rs` only has `passed: bool`. Stub gates (not yet implemented) and
gates skipped by the adaptive threshold system both return `passed: true`, inflating pass rates
and giving false confidence. There is no way to distinguish "gate ran and passed" from "gate was
skipped." Downstream consumers (dashboard, learning) count skipped gates as real passes.

```rust
fn to_gate_verdict(gate_name: String, verdict: Verdict) -> GateVerdict {
    let output = verdict
        .error_digest
        .filter(|output| !output.is_empty())
        .or_else(|| verdict.detail.filter(|output| !output.is_empty()))
        .unwrap_or(verdict.reason);

    GateVerdict {
        gate_name,
        passed: verdict.passed,
        output,
        duration_ms: verdict.duration_ms,
    }
}
```
