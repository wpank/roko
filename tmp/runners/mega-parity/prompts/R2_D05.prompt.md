## Problem

## Architecture Contract

- Canonical paths are preferred, fallbacks are documented

- Error message names both checked paths when neither exists
- `resume <plan-id>` looks in `.roko/plans/<id>/` first, then `./plans/<id>/`
- `--resume <state-file>` continues to work with explicit path
