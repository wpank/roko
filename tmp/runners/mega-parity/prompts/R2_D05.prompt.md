### Step 1: Add a plan directory resolution helper function

Add a helper function to `main.rs` that implements the two-step lookup with fallback
note:

```rust
/// Resolve the plans directory: prefer `.roko/plans/`, fall back to `./plans/`.
///
/// Returns the resolved path and a flag indicating whether the fallback was used.
/// Prints a note to stderr when falling back.
fn resolve_plans_dir(workdir: &Path, explicit: Option<&Path>) -> PathBuf {
    if let Some(p) = explicit {
        // Explicit path always wins.
        return p.to_path_buf();
    }
    let canonical = workdir.join(".roko").join("plans");
    if canonical.exists() {
        return canonical;
    }
    let fallback = workdir.join("plans");
    if fallback.exists() {
        eprintln!(
            "note: using {} (not found in {})",
            fallback.display(),
            canonical.display(),
        );
        return fallback;
    }
    // Neither exists: return canonical and let load_plans produce the error message.
    // The caller will then produce a helpful error naming both paths checked.
    canonical
}
```

Place this function near the other helpers in main.rs (e.g., after `resolve_workdir`).
