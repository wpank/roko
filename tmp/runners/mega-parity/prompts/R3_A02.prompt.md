## Problem

The `ChatAgentSession` constructor (R3_A01) has a placeholder empty string for `system_prompt`. Interactive chat and one-shot mode currently send no system prompt at all — the raw `dispatch_direct::dispatch_claude_cli` just pipes the user prompt to stdin. Meanwhile, `orchestrate.rs` uses a full 9-layer `SystemPromptBuilder` with workspace context, role identity, conventions, domain knowledge, and tool hints. The chat session should use the same builder.

## Architecture Contract

- System prompt is resolved once at session construction time
- Uses `SystemPromptBuilder` from `roko-compose` (not a new builder)
- Includes workspace context (project name, language, directory)
- Includes role identity for "interactive-chat" mode
- Bounded to ~4000 tokens (reasonable for chat without eating context)
- Falls back to a minimal prompt if builder fails
- One-shot uses the exact same prompt (no separate path)

Add a private helper:

```rust
/// Build a system prompt for interactive/one-shot chat using SystemPromptBuilder.
///
/// Uses workspace context from the working directory. Falls back to a minimal
/// prompt if the builder encounters errors.
fn build_chat_system_prompt(workdir: &Path, config: &crate::config::Config) -> String {
    let role = "You are an expert software engineer working in an interactive chat session. \
                You have access to tools for reading, editing, and searching code.";

    let mut builder = SystemPromptBuilder::new(role);

    // Layer 2: Project conventions from config or workspace
    if let Some(ref conventions) = config.agent.conventions {
        builder = builder.with_conventions(conventions);
    }

    // Layer 3: Domain context from working directory
    let domain = format!(
        "Working directory: {}\nProject: {}",
        workdir.display(),
        workdir.file_name()
            .and_then(|n| n.to_str())
            .unwrap_or("unknown"),
    );
    builder = builder.with_domain(&domain);

    // Layer 4: Workspace context (git branch, recent files, etc.)
    if let Ok(context) = gather_workspace_context(workdir) {
        builder = builder.with_context(&context);
    }

    let prompt = builder.build();

    // Fallback if builder produces empty
    if prompt.trim().is_empty() {
        role.to_string()
    } else {
        prompt
    }
}

/// Gather lightweight workspace context: git branch, language hints.
fn gather_workspace_context(workdir: &Path) -> Result<String> {
    let mut parts = Vec::new();

    // Git branch
    if let Ok(output) = std::process::Command::new("git")
        .args(["branch", "--show-current"])
        .current_dir(workdir)
        .output()
    {
        if output.status.success() {
            let branch = String::from_utf8_lossy(&output.stdout).trim().to_string();
            if !branch.is_empty() {
                parts.push(format!("Git branch: {branch}"));
            }
        }
    }

    // Language hint from Cargo.toml or package.json
    if workdir.join("Cargo.toml").exists() {
        parts.push("Language: Rust".to_string());
    } else if workdir.join("package.json").exists() {
        parts.push("Language: TypeScript/JavaScript".to_string());
    }

    Ok(parts.join("\n"))
}
```
