## Task

Resolve MCP config from existing discovery paths

## Runner Context

You are working in runner `mega-parity`, batch R3_A04.
This batch is part of Runner 3: agent-session-parity — Make interactive roko and one-shot use a real agent session via existing adapters.

## Problem

The `ChatAgentSession` constructor (R3_A01) has `None` for `mcp_config`. Interactive chat and one-shot mode never pass `--mcp-config` to Claude CLI, so MCP tools are unavailable. Meanwhile, `orchestrate.rs` and `ClaudeCliAgent` already support MCP config discovery from `roko.toml`, `.roko/mcp.json`, and `~/.claude/mcp-config.json`. The chat session should use the same discovery.

## Architecture Contract

- MCP config resolved once at session construction time
- Discovery order: `config.agent.mcp_config` (explicit) -> `.roko/mcp.json` (workspace) -> `~/.claude/mcp-config.json` (global)
- Stores `Option<PathBuf>` — None means no MCP
- Passes `--mcp-config <path>` and `--strict-mcp-config` to Claude CLI
- Never fails on missing MCP config (it is optional)
- Never parses MCP config contents (just passes the path)

## Changes Required

In `crates/roko-cli/src/chat_session.rs`, replace the mcp_config placeholder:

```rust
// Inside ChatAgentSession::new():
```

Add a private helper:

```rust
/// Discover MCP config file using the same resolution order as orchestrate.rs.
///
/// Priority:
/// 1. Explicit path in `config.agent.mcp_config`
/// 2. Workspace `.roko/mcp.json`
/// 3. Global `~/.claude/mcp-config.json`
/// 4. roko-agent's `find_mcp_config` fallback
///
/// Returns None if no MCP config is found (not an error).
fn resolve_mcp_config(workdir: &Path, config: &crate::config::Config) -> Option<PathBuf> {
    // 1. Explicit config path
    if let Some(ref path) = config.agent.mcp_config {
        let resolved = if path.is_absolute() {
            path.clone()
        } else {
            workdir.join(path)
        };
        if resolved.exists() {
            tracing::debug!("MCP config from roko.toml: {}", resolved.display());
            return Some(resolved);
        }
        tracing::debug!("MCP config in roko.toml does not exist: {}", resolved.display());
    }

    // 2. Workspace .roko/mcp.json
    let workspace_mcp = workdir.join(".roko/mcp.json");
    if workspace_mcp.exists() {
        tracing::debug!("MCP config from workspace: {}", workspace_mcp.display());
        return Some(workspace_mcp);
    }

    // 3. Global ~/.claude/mcp-config.json
    if let Some(home) = dirs::home_dir() {
        let global_mcp = home.join(".claude/mcp-config.json");
        if global_mcp.exists() {
            tracing::debug!("MCP config from global: {}", global_mcp.display());
            return Some(global_mcp);
        }
    }

    // 4. roko-agent's find_mcp_config fallback
    if let Some(found) = roko_agent::mcp::find_mcp_config(workdir) {
        tracing::debug!("MCP config from roko-agent discovery: {}", found.display());
        return Some(found);
    }

    tracing::debug!("no MCP config found");
    None
}
```
