## Problem

Interactive chat supports slash commands (`/system`, `/model`, `/effort`, `/reset`, `/tools`, `/mcp`) that mutate session state. These mutations need to live on `ChatAgentSession` so they affect subsequent `send_turn` calls. This batch implements the mutation logic with unit tests but does NOT wire it into the REPL loop.

## Architecture Contract

- Slash commands mutate `ChatAgentSession` fields directly
- `handle_slash_command` returns a `SlashResult` enum (no side effects beyond self-mutation)
- Regular text (not starting with `/`) returns `SlashResult::NotACommand`
- Unknown `/foo` returns `SlashResult::Unknown`
- Unit tests cover every command variant

```rust
/// Result of processing a potential slash command.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum SlashResult {
    /// Command recognized and session state updated. String is user-facing message.
    Updated(String),
    /// Command recognized but had an error. String is error message.
    Error(String),
    /// Input starts with `/` but command is not recognized.
    Unknown(String),
    /// Input is not a slash command (regular chat text).
    NotACommand,
}

impl ChatAgentSession {
    /// Process input that may be a slash command.
    ///
    /// If the input starts with `/`, attempts to parse and execute the command,
    /// mutating session state as needed. Returns `NotACommand` for regular text.
    pub fn handle_slash_command(&mut self, input: &str) -> SlashResult {
        let trimmed = input.trim();
        if !trimmed.starts_with('/') {
            return SlashResult::NotACommand;
        }

        let parts: Vec<&str> = trimmed.splitn(2, char::is_whitespace).collect();
        let cmd = parts[0];
        let arg = parts.get(1).map(|s| s.trim()).unwrap_or("");

        match cmd {
            "/system" => {
                if arg.is_empty() {
                    SlashResult::Updated(format!(
                        "Current system prompt ({} chars): {}",
                        self.system_prompt.len(),
                        &self.system_prompt[..self.system_prompt.len().min(200)],
                    ))
                } else {
                    self.system_prompt = arg.to_string();
                    SlashResult::Updated(format!("System prompt set ({} chars)", arg.len()))
                }
            }
            "/model" => {
                if arg.is_empty() {
                    // EffectiveModelSelection has `backend_slug: String` (not a .slug() method)
                    SlashResult::Updated(format!("Current model: {}", self.model_selection.backend_slug))
                } else {
                    // Update the backend slug directly — full resolution happens at send_turn time.
                    // EffectiveModelSelection fields: backend_slug, effective_model_key, provider_key, etc.
                    self.model_selection.backend_slug = arg.to_string();
                    SlashResult::Updated(format!("Model set to: {arg}"))
                }
            }
            "/effort" => {
                match arg {
                    "low" | "medium" | "high" | "max" => {
                        self.effort = arg.to_string();
                        SlashResult::Updated(format!("Effort set to: {arg}"))
                    }
                    "" => SlashResult::Updated(format!("Current effort: {}", self.effort)),
                    _ => SlashResult::Error(format!("Invalid effort level: {arg} (use low/medium/high/max)")),
                }
            }
            "/reset" => {
                self.session_id = None;
                self.api_history.clear();
                SlashResult::Updated("Session reset: cleared session_id and history".to_string())
            }
            "/tools" => {
                if arg.is_empty() {
                    SlashResult::Updated(format!("Current tools: {}", self.allowed_tools_csv))
                } else {
                    self.allowed_tools_csv = arg.to_string();
                    SlashResult::Updated(format!("Tools set to: {arg}"))
                }
            }
            "/mcp" => {
                if arg.is_empty() {
                    let status = match &self.mcp_config {
                        Some(p) => format!("MCP config: {}", p.display()),
                        None => "No MCP config".to_string(),
                    };
                    SlashResult::Updated(status)
                } else {
                    let path = PathBuf::from(arg);
                    if path.exists() {
                        self.mcp_config = Some(path.clone());
                        SlashResult::Updated(format!("MCP config set to: {}", path.display()))
                    } else {
                        SlashResult::Error(format!("MCP config not found: {}", path.display()))
                    }
                }
            }
            _ => SlashResult::Unknown(cmd.to_string()),
        }
    }
}
```
