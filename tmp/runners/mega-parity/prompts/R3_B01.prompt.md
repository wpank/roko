## Task

Add send_turn using ClaudeCliAgent

## Runner Context

You are working in runner `mega-parity`, batch R3_B01.
This batch is part of Runner 3: agent-session-parity — Make interactive roko and one-shot use a real agent session via existing adapters.

## Problem

`ChatAgentSession` (from R3_A01-A04) has all session state but no way to send a turn. Currently, `chat_inline.rs` calls `dispatch_direct::dispatch_prompt` which spawns a raw `Command::new("claude")` missing system_prompt, tools, MCP, resume, effort, and safety settings. `ClaudeCliAgent` already handles all of these correctly via its builder API and `Agent::run` trait method. We need `send_turn` to build a properly configured `ClaudeCliAgent` per turn and delegate to it.

## Architecture Contract

- `send_turn` builds a fresh `ClaudeCliAgent` each call (because model/effort/system_prompt can change via slash commands between turns)
- Passes ALL fields: workdir, model, effort, system_prompt, tools, mcp, resume (session_id), timeout, settings_json
- Calls `ClaudeCliAgent`'s `Agent::run` method
- Returns `TurnResult` with text, model, tokens, tool_calls, session_id, duration
- No streaming in this batch (that is R3_C*)
- No API provider support (that is R3_E01)

```rust
use roko_agent::claude_cli_agent::ClaudeCliAgent;
use roko_agent::agent::Agent;
use roko_core::{Body, Engram, Kind};
use std::time::Instant;

/// Summary of a tool call captured during a turn.
#[derive(Debug, Clone)]
pub struct ToolCallSummary {
    /// Tool name (e.g. "Read", "Bash", "Edit").
    pub name: String,
    /// Abbreviated input (first 200 chars).
    pub input_abbrev: String,
    /// Whether the tool call succeeded.
    pub success: bool,
}

/// Result of a single agent turn.
#[derive(Debug, Clone)]
pub struct TurnResult {
    /// The model's text response.
    pub text: String,
    /// Which model actually responded.
    pub model: String,
    /// Input tokens consumed.
    pub input_tokens: u64,
    /// Output tokens produced.
    pub output_tokens: u64,
    /// Tool calls executed during this turn.
    pub tool_calls: Vec<ToolCallSummary>,
    /// Session ID for resume (extracted from Claude CLI result event).
    pub session_id: Option<String>,
    /// Wall-clock duration of this turn.
    pub duration: Duration,
    /// Whether the turn was cancelled by the user.
    pub cancelled: bool,
}

impl ChatAgentSession {
    /// Send a single turn to the agent and return the result.
    ///
    /// Builds a `ClaudeCliAgent` with current session state, delegates
    /// via `Agent::run`, extracts usage from the result, and returns a
    /// unified `TurnResult`.
    ///
    /// **Note on session_id**: `ClaudeCliAgent` does not currently
    /// extract `session_id` from stream-json into `AgentResult`. The
    /// streaming path (R3_C01-C03) will capture it from the raw stream.
    /// For the non-streaming path, session_id remains None.
    pub async fn send_turn(&mut self, prompt: &str) -> Result<TurnResult> {
        let start = Instant::now();

        // Build ClaudeCliAgent with all session state.
        // Check `ClaudeCliAgent::new` for the constructor signature:
        //   new(program, current_dir, model) -> Self
        let program = "claude"; // TODO: resolve from config.agent.command
        let mut agent = ClaudeCliAgent::new(program, &self.workdir, &self.model_selection.backend_slug)
            .with_effort(&self.effort)
            .with_bare_mode(false); // chat mode, not bare

        // System prompt
        if !self.system_prompt.is_empty() {
            agent = agent.with_system_prompt(&self.system_prompt);
        }

        // Tool allowlist
        if !self.allowed_tools_csv.is_empty() {
            agent = agent.with_tools(&self.allowed_tools_csv);
        }

        // MCP config
        if let Some(ref mcp_path) = self.mcp_config {
            agent = agent.with_mcp_config(mcp_path);
        }

        // Resume from previous turn
        if let Some(ref sid) = self.session_id {
            agent = agent.with_resume(sid);
        }

        // Timeout
        if let Some(t) = self.timeout {
            agent = agent.with_timeout_ms(t.as_millis() as u64);
        }

        // Build the input engram using the builder API (not struct literal).
        // Engram has many fields (id, fingerprint, decay, score, etc.) that
        // the builder fills with defaults.
        let engram = Engram::builder(Kind::Prompt)
            .body(Body::text(prompt))
            .build();

        // Delegate to Agent::run (the trait method is `run`, not `act`).
        // Signature: async fn run(&self, input: &Engram, ctx: &Context) -> AgentResult
        let ctx = roko_core::Context::default();
        let result = agent.run(&engram, &ctx).await;

        // Extract response text from result.output (an Engram).
        // The output Engram's body contains the agent's response.
        let text = result.output.body.as_text()
            .map(|s| s.to_string())
            .unwrap_or_default();

        // Session_id: ClaudeCliAgent does NOT currently surface session_id
        // in AgentResult. The streaming path (R3_C01-C03) captures it from
        // the raw stream-json "result" event. For now, leave as None.
        // The output Engram has tags ("agent", "model") but not "session_id".
        let new_session_id: Option<String> = result.output.tag("session_id")
            .map(String::from);

        // Update session_id for next turn's --resume
        if new_session_id.is_some() {
            self.session_id = new_session_id.clone();
        }

        // Extract usage from result.usage (a Usage struct with typed fields).
        // Usage fields: input_tokens (u32), output_tokens (u32), wall_ms (u64), etc.
        let input_tokens = u64::from(result.usage.input_tokens);
        let output_tokens = u64::from(result.usage.output_tokens);

        Ok(TurnResult {
            text,
            model: self.model_selection.backend_slug.clone(),
            input_tokens,
            output_tokens,
            tool_calls: Vec::new(), // R3_B03 fills this in
            session_id: new_session_id,
            duration: start.elapsed(),
            cancelled: false,
        })
    }
}
```
