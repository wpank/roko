- The non-streaming `send_turn` keeps `tool_calls: Vec::new()` (ClaudeCliAgent buffers all output; structured tool data is inaccessible without the raw stream)
- Caller renders tool output (session does not print)

### Step 1: Add a function to accumulate tool calls from streaming events

This function is called by `send_turn_streaming` (R3_C03) as events arrive:

```rust
use roko_agent::runtime_events::AgentRuntimeEvent;

/// Process a single `AgentRuntimeEvent` and update the tool_calls accumulator.
///
/// Called from `send_turn_streaming` for each event in the stream.
/// The non-streaming `send_turn` does NOT call this — tool data is
/// inaccessible without the raw stream.
///
/// Matching strategy:
/// - `ToolCall` event: push a new ToolCallSummary with success=true (optimistic)
/// - `ToolOutput` event: try to match by id and update input_abbrev with the output
///
/// The tool_name in ToolOutput (from ClaudeToolEvent.tool_name) may differ from
/// the name in ToolCall, so we match by tool_use_id.
pub fn accumulate_tool_event(
    tool_calls: &mut Vec<ToolCallSummary>,
    pending_ids: &mut Vec<(String, usize)>, // (tool_use_id, index in tool_calls)
    event: &AgentRuntimeEvent,
) {
    match event {
        AgentRuntimeEvent::ToolCall { id, name } => {
            let idx = tool_calls.len();
            tool_calls.push(ToolCallSummary {
                name: name.clone(),
                input_abbrev: String::new(), // filled by ToolOutput event
                success: true,              // optimistic; corrected on ToolOutput
            });
            pending_ids.push((id.clone(), idx));
        }
        AgentRuntimeEvent::ToolOutput { id, output } => {
            // Find the matching ToolCall by id.
            if let Some(pos) = pending_ids.iter().position(|(tid, _)| tid == id) {
                let (_, idx) = pending_ids.remove(pos);
                if let Some(tc) = tool_calls.get_mut(idx) {
                    // Abbreviate the output to 200 chars.
                    tc.input_abbrev = if output.len() > 200 {
                        format!("{}...", &output[..200])
                    } else {
                        output.clone()
                    };
                    // ToolOutput always means success (errors use is_error flag in result).
                    tc.success = true;
                }
            }
        }
        _ => {} // Other events handled elsewhere
    }
}
```
