## Problem

When the user presses Ctrl-C during a turn, or a timeout expires, the agent subprocess must be killed cleanly. Currently `chat_inline.rs` has ad-hoc Ctrl-C handling, and `dispatch_direct.rs` has no timeout or cancellation. `ClaudeCliAgent` already has `kill_tree` and process group management in `roko-agent/src/process.rs`, but `send_turn` needs to hook into cancellation properly.

## Architecture Contract

- Ctrl-C during a turn sends SIGTERM to agent process, then SIGKILL after 5 seconds
- Configurable per-turn timeout (from session.timeout)
- Cancelled turn returns `TurnResult { cancelled: true, ... }` with partial text
- No zombie processes left behind
- Uses existing `kill_tree` / `register_spawned_pid` from `roko-agent/src/process.rs`
- Does not create a new process supervisor

In `crates/roko-cli/src/chat_session.rs`, wrap the `Agent::run` call with cancellation support:

```rust
use tokio::select;
use tokio::signal;
use tokio_util::sync::CancellationToken;

impl ChatAgentSession {
    /// Send a turn with cancellation support.
    ///
    /// Wraps Agent::run with:
    /// 1. Timeout from self.timeout
    /// 2. Ctrl-C signal handler
    /// Returns TurnResult with cancelled=true if interrupted.
    pub async fn send_turn(&mut self, prompt: &str) -> Result<TurnResult> {
        let start = Instant::now();
        let cancel = CancellationToken::new();

        // Build agent (same as before)
        let agent = self.build_agent()?;
        let engram = self.build_engram(prompt);

        // Race: agent execution vs timeout vs Ctrl-C
        let result = {
            let cancel_clone = cancel.clone();
            let timeout_duration = self.timeout.unwrap_or(Duration::from_secs(300));

            select! {
                res = agent.run(&engram, &ctx) => res,
                _ = tokio::time::sleep(timeout_duration) => {
                    cancel_clone.cancel();
                    tracing::warn!("turn timed out after {:?}", timeout_duration);
                    return Ok(TurnResult::cancelled(start.elapsed()));
                }
                _ = signal::ctrl_c() => {
                    cancel_clone.cancel();
                    tracing::info!("turn cancelled by Ctrl-C");
                    return Ok(TurnResult::cancelled(start.elapsed()));
                }
            }
        };

        // Process result (same as R3_B01)
        self.process_result(result?, start)
    }
}

impl TurnResult {
    /// Create a cancelled result with partial information.
    pub fn cancelled(duration: Duration) -> Self {
        Self {
            text: String::new(),
            model: String::new(),
            input_tokens: 0,
            output_tokens: 0,
            tool_calls: Vec::new(),
            session_id: None,
            duration,
            cancelled: true,
        }
    }
}
```
