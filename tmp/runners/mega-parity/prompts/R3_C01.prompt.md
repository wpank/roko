## Problem

`dispatch_direct.rs` buffers ALL stream-json output then processes it, meaning the user sees nothing until the full response completes. `ClaudeCliAgent` processes stream events internally (in its `run` method) but does not expose them to callers. For interactive chat, we need a streaming turn method that exposes events as they arrive so the caller can display text deltas immediately and show tool activity in real-time.

**Critical**: The stream parser already exists in `crates/roko-agent/src/provider/claude_cli/stream.rs`. It emits `AgentRuntimeEvent` variants. Do NOT create a new `StreamEvent` enum — reuse `AgentRuntimeEvent`.

## Architecture Contract

- Uses the existing `parse_stream_line` from `roko_agent::provider::claude_cli` (returns `Vec<AgentRuntimeEvent>`)
- Uses `AgentRuntimeEvent` variants (NOT a new `StreamEvent` enum)
- The streaming turn method spawns the same subprocess as `ClaudeCliAgent::build_command` would, reads stdout line-by-line, and calls `parse_stream_line` on each line
- Events are forwarded to a `tokio::sync::mpsc::Sender<AgentRuntimeEvent>` channel
- The non-streaming `send_turn` (R3_B01) is NOT replaced — both paths coexist

```rust
impl ChatAgentSession {
    /// Send a turn with streaming output via a channel.
    ///
    /// Spawns the Claude CLI subprocess directly (same flags as
    /// ClaudeCliAgent::build_command), reads stdout line-by-line,
    /// calls `parse_stream_line` on each line, and forwards events
    /// to the provided channel.
    ///
    /// Returns a `TurnResult` after the stream completes.
    /// Session_id and token counts are populated from the stream
    /// (unlike the non-streaming `send_turn` which returns None/0).
    ///
    /// The caller is responsible for consuming events from `tx`.
    /// Dropping the sender before the stream finishes is safe — sends
    /// are fire-and-forget with `let _ = tx.send(event).await`.
    pub async fn send_turn_streaming(
        &mut self,
        prompt: &str,
        tx: tokio::sync::mpsc::Sender<AgentRuntimeEvent>,
    ) -> Result<TurnResult> {
        let start = Instant::now();
        let mut accumulated_text = String::new();
        let mut tool_calls: Vec<ToolCallSummary> = Vec::new();
        let mut pending_tool_ids: Vec<(String, usize)> = Vec::new();
        let mut final_session_id: Option<String> = None;
        let mut final_input_tokens: u64 = 0;
        let mut final_output_tokens: u64 = 0;

        // Build the subprocess command directly from session fields.
        // This replicates ClaudeCliAgent::build_command() for the streaming path.
        let mut cmd = build_streaming_command(self)?;

        let mut child = cmd.spawn()
            .context("spawn claude subprocess for streaming")?;

        // Write prompt to stdin, then close stdin so claude knows input is done.
        if let Some(mut stdin) = child.stdin.take() {
            stdin.write_all(prompt.as_bytes()).await
                .context("write prompt to claude stdin")?;
        }
        // stdin is dropped here, closing the pipe.

        let stdout = child.stdout.take()
            .context("capture claude stdout for streaming")?;
        let reader = BufReader::new(stdout);
        let mut lines = reader.lines();

        // Read stdout line-by-line and parse each line with the existing parser.
        while let Some(line) = lines.next_line().await
            .context("read claude stdout line")?
        {
            for event in parse_stream_line(&line) {
                // Accumulate state from event.
                match &event {
                    AgentRuntimeEvent::MessageDelta { text } => {
                        accumulated_text.push_str(text);
                    }
                    AgentRuntimeEvent::SystemInit { session_id, .. } => {
                        // Capture session_id from the init handshake.
                        if !session_id.is_empty() {
                            final_session_id = Some(session_id.clone());
                        }
                    }
                    AgentRuntimeEvent::TurnCompleted { session_id, .. } => {
                        // Capture session_id from the terminal event.
                        if let Some(sid) = session_id {
                            if !sid.is_empty() {
                                final_session_id = Some(sid.clone());
                            }
                        }
                    }
                    AgentRuntimeEvent::TokenUsage { input_tokens, output_tokens, .. } => {
                        // Accumulate token counts (last value wins).
                        final_input_tokens = *input_tokens;
                        final_output_tokens = *output_tokens;
                    }
                    _ => {}
                }

                // Capture tool events.
                accumulate_tool_event(&mut tool_calls, &mut pending_tool_ids, &event);

                // Forward to caller (fire-and-forget — caller may have dropped rx).
                let _ = tx.send(event).await;
            }
        }

        // Wait for subprocess to exit.
        let status = child.wait().await.context("wait for claude subprocess")?;
        if !status.success() {
            let code = status.code().map_or_else(|| "signal".to_string(), |c| c.to_string());
            tracing::warn!(exit_code = %code, "claude subprocess exited with error");
        }

        // Update session_id for next turn's --resume.
        if let Some(ref sid) = final_session_id {
            tracing::debug!(session_id = %sid, "captured session_id from stream");
            self.session_id = Some(sid.clone());
        }

        Ok(TurnResult {
            text: accumulated_text,
            model: self.model.clone(),
            input_tokens: final_input_tokens,
            output_tokens: final_output_tokens,
            tool_calls,
            session_id: final_session_id,
            duration: start.elapsed(),
            cancelled: false,
        })
    }
}

/// Build the subprocess command for streaming, replicating ClaudeCliAgent::build_command.
///
/// See crates/roko-agent/src/claude_cli_agent.rs lines 302-359 for the
/// authoritative flag list. This function must stay in sync with that method.
fn build_streaming_command(session: &ChatAgentSession) -> Result<Command> {
    let mut cmd = Command::new("claude");
    cmd.arg("--print")
       .arg("--verbose")
       .arg("--output-format").arg("stream-json")
       .arg("--model").arg(&session.model)
       .arg("--effort").arg(&session.effort)
       .arg("--settings").arg(build_settings_json())
       .arg("--dangerously-skip-permissions");

    if !session.system_prompt.is_empty() {
        cmd.arg("--append-system-prompt").arg(&session.system_prompt);
    }
    if !session.allowed_tools_csv.is_empty() {
        cmd.arg("--tools").arg(&session.allowed_tools_csv);
    }
    if let Some(ref mcp_path) = session.mcp_config {
        cmd.arg("--mcp-config").arg(mcp_path)
           .arg("--strict-mcp-config");
    }
    if let Some(ref sid) = session.session_id {
        cmd.arg("--resume").arg(sid);
    }

    cmd.current_dir(&session.workdir)
       .stdin(std::process::Stdio::piped())
       .stdout(std::process::Stdio::piped())
       .stderr(std::process::Stdio::piped())
       .kill_on_drop(true);
    set_process_group(&mut cmd);
    cmd.env_remove("CLAUDECODE");

    Ok(cmd)
}
```
