## Problem

The `AgentRuntimeEvent::TurnCompleted` and `AgentRuntimeEvent::TokenUsage` events from R3_C01's stream carry `session_id` and token counts, but `send_turn_streaming` from R3_C01 needs to be verified to correctly extract and apply this metadata. This batch audits and hardens the metadata capture path in `send_turn_streaming`.

## Architecture Contract

- `AgentRuntimeEvent::SystemInit { session_id, ... }` → update `self.session_id` (first available)
- `AgentRuntimeEvent::TurnCompleted { session_id, ... }` → update `self.session_id` (authoritative)
- `AgentRuntimeEvent::TokenUsage { input_tokens, output_tokens, ... }` → update `TurnResult` counts
- If stream is interrupted (Ctrl-C / timeout), partial metadata is acceptable (tokens=0, session_id=None)
- No second pass over output (metadata extracted inline during streaming)
- No global/static state (only session struct fields)
- Debug log emitted when session_id is captured

### Step 1: Verify the event match arms in `send_turn_streaming`

The match block inside `send_turn_streaming` MUST include these arms exactly:

```rust
match &event {
    AgentRuntimeEvent::MessageDelta { text } => {
        accumulated_text.push_str(text);
    }

    AgentRuntimeEvent::SystemInit { session_id, .. } => {
        // First session_id in stream — set if not already captured.
        // SystemInit.session_id is String (not Option), check for empty.
        if !session_id.is_empty() && final_session_id.is_none() {
            final_session_id = Some(session_id.clone());
            tracing::debug!(session_id = %session_id, "captured session_id from SystemInit");
        }
    }

    AgentRuntimeEvent::TurnCompleted { session_id, is_error, .. } => {
        // Authoritative session_id from terminal event — overwrites SystemInit.
        // TurnCompleted.session_id IS Option<String> (None if empty in wire event).
        if let Some(sid) = session_id {
            if !sid.is_empty() {
                final_session_id = Some(sid.clone());
                tracing::debug!(session_id = %sid, "captured session_id from TurnCompleted");
            }
        }
        if *is_error {
            tracing::warn!("Claude CLI reported is_error=true in result event");
        }
    }

    AgentRuntimeEvent::TokenUsage { input_tokens, output_tokens, .. } => {
        // Always update to latest value — multiple TokenUsage events may arrive.
        // The last one (from the result event) is the authoritative final count.
        final_input_tokens = *input_tokens;
        final_output_tokens = *output_tokens;
        tracing::debug!(
            input_tokens = %input_tokens,
            output_tokens = %output_tokens,
            "token usage update"
        );
    }

    _ => {}
}
```
