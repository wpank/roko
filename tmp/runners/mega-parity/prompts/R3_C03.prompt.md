Capture final result metadata from stream

## Runner Context

## Problem

The `StreamEvent::Result` event from R3_C01 carries `session_id` and `usage` (input/output tokens), but these need to be captured and used to update session state and populate `TurnResult`. Currently R3_C02 renders events but `Result` events are just ignored. This batch ensures the final metadata is extracted from the stream and applied correctly.

## Architecture Contract

- `Result` event updates `self.session_id` for next turn's `--resume`
- `Result` event populates `TurnResult.usage` (input_tokens, output_tokens)
- If stream is interrupted (Ctrl-C/timeout), partial metadata is acceptable
- No second pass over output (metadata extracted inline during streaming)
- No global state (only session struct fields)

```rust
impl ChatAgentSession {
    pub async fn send_turn_streaming(
        &mut self,
        prompt: &str,
        tx: tokio::sync::mpsc::Sender<StreamEvent>,
    ) -> Result<TurnResult> {
        let start = Instant::now();
        let mut accumulated_text = String::new();
        let mut tool_calls = Vec::new();
        let mut final_session_id: Option<String> = None;
        let mut final_usage: Option<StreamUsage> = None;

        // ... spawn subprocess, read lines ...

        // For each line from stdout:
        for event in parse_stream_line(&line) {
            match &event {
                StreamEvent::TextDelta(text) => {
                    accumulated_text.push_str(text);
                }
                StreamEvent::ToolStart { name } => {
                    tool_calls.push(ToolCallSummary {
                        name: name.clone(),
                        input_abbrev: String::new(),
                        success: true, // updated on ToolEnd
                    });
                }
                StreamEvent::ToolEnd { name, success } => {
                    // Update the matching ToolStart entry
                    if let Some(tc) = tool_calls.iter_mut().rev().find(|t| t.name == *name) {
                        tc.success = *success;
                    }
                }
                StreamEvent::Result { session_id, usage } => {
                    // Capture final metadata
                    final_session_id = session_id.clone();
                    final_usage = usage.clone();

                    // Update session for next turn's --resume
                    if let Some(ref sid) = final_session_id {
                        self.session_id = Some(sid.clone());
                        tracing::debug!(session_id = %sid, "captured session_id from stream");
                    }
                }
                StreamEvent::Error(_) => {}
            }

            // Forward to caller
            let _ = tx.send(event).await;
        }

        let (input_tokens, output_tokens) = match final_usage {
            Some(u) => (u.input_tokens, u.output_tokens),
            None => (0, 0),
        };

        Ok(TurnResult {
            text: accumulated_text,
            model: self.model_selection.backend_slug.clone(),
            input_tokens,
            output_tokens,
            tool_calls,
            session_id: final_session_id,
            duration: start.elapsed(),
            cancelled: false,
        })
    }
}
```
