## Problem

`ChatAgentSession` currently only works with Claude CLI (via `ClaudeCliAgent`). When the model resolves to an API provider (Anthropic API, OpenAI-compatible, Ollama, etc.), the session should return a clear typed error instead of silently failing or falling back to CLI. This is a stub that will be replaced by actual API provider support in a future runner.

## Architecture Contract

- If model resolves to a non-CLI provider, return a typed error
- Error message identifies which provider was requested
- Log which provider at warn level
- No silent fallback to CLI
- No empty success response
- Explicit "not yet implemented" — this is a known gap, not a bug

```rust
use roko_core::agent::{ProviderKind, resolve_model};

/// Error returned when an API provider is requested but not yet supported
/// in the chat session path.
#[derive(Debug, thiserror::Error)]
pub enum SessionError {
    #[error("API provider chat not yet implemented for provider '{provider}' (model: {model}). \
             Use `roko run` for API provider dispatch, or switch to Claude CLI with `/model claude-sonnet-4-20250514`.")]
    ApiProviderNotImplemented {
        provider: String,
        model: String,
    },

    #[error(transparent)]
    Other(#[from] anyhow::Error),
}

impl ChatAgentSession {
    /// Check if the current model resolves to a CLI provider.
    fn is_cli_provider(&self) -> bool {
        // Resolve the model to determine provider kind
        // Claude CLI models: claude-sonnet-4-20250514, claude-haiku-4-5, etc.
        // Check if ProviderKind is ClaudeCli
        // EffectiveModelSelection has `backend_slug: String` and `provider_kind: String`
        // (fields, not methods). Use provider_kind for accurate routing.
        let kind = &self.model_selection.provider_kind;
        kind == "claude_cli"
    }

    pub async fn send_turn(&mut self, prompt: &str) -> Result<TurnResult, SessionError> {
        if !self.is_cli_provider() {
            // EffectiveModelSelection fields: provider_key, backend_slug (not methods)
            let provider = self.model_selection.provider_key.clone();
            let model = self.model_selection.backend_slug.clone();
            tracing::warn!(
                provider = %provider,
                model = %model,
                "API provider chat not yet implemented in ChatAgentSession"
            );
            return Err(SessionError::ApiProviderNotImplemented {
                provider,
                model,
            });
        }

        // ... existing CLI turn logic ...
    }
}
```
