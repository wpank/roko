## Problem

ACP pipeline phases run with zero safety enforcement after agent dispatch completes. There are no post-dispatch checks on agent output, no secret scrubbing, and no path-escape detection. Specifically:

- `run_agent_phase()` in `crates/roko-acp/src/runner.rs` (lines 1102–1150) dispatches an agent and gets back a text string but never checks it for leaks or violations.
- `handle_session_prompt_inner()` in `crates/roko-acp/src/bridge_events.rs` (line ~356) does a single-agent dispatch but has no pre-dispatch safety check.

`crates/roko-agent/src/safety/mod.rs` already implements this:
- `SafetyLayer::pre_dispatch_check(plan_id, task_id, role, exec_dir)` — validates before execution
- `SafetyLayer::post_dispatch_check(plan_id, task_id, role, agent_output, changed_files)` — validates after execution

## Architecture Contract
- **SP-1:** Default is safe — missing contracts fall back to RESTRICTIVE defaults, not permissive
- **ACP-5:** Session state is shared-nothing — safety is created per-dispatch, not stored on AcpSession
- **CP-2:** Delegate to existing adapters — use `SafetyLayer` from `roko-agent`, don't reimplement

```rust
/// Build a SafetyLayer appropriate for an ACP session mode.
/// SP-1: Default is RESTRICTIVE — missing contracts use deny-everything fallback.
fn safety_layer_for_mode(mode: &str) -> SafetyLayer {
    // with_defaults() uses AgentContract::restricted("default") — deny-everything.
    // We use with_role() to load whatever contract exists for the mode string,
    // falling back to restrictive if no contract file exists (ContractLoadMode::RestrictedFallback).
    SafetyLayer::with_defaults().with_role(mode)
}

/// Build a SafetyLayer appropriate for a pipeline phase role.
/// Strategist and Reviewer get a read-oriented contract; Implementer/AutoFixer get broader.
fn safety_layer_for_pipeline_role(role: &str) -> SafetyLayer {
    SafetyLayer::with_defaults().with_role(role)
}
```

```rust
    // Pre-dispatch safety check (SP-1: restrictive default).
    {
        use roko_agent::safety::{SafetyLayer, ViolationSeverity};
        let safety = SafetyLayer::with_defaults().with_role(&session.config_state.agent_mode);
        if let Err(violation) = safety.pre_dispatch_check(
            &session.session_id,
            "session-prompt",
            &session.config_state.agent_mode,
            workdir,
        ) {
            match violation.severity {
                ViolationSeverity::Block => {
                    tracing::error!(
                        session_id = %session.session_id,
                        violation = ?violation.violation_type,
                        message = %violation.message,
                        "ACP pre-dispatch safety check BLOCKED dispatch"
                    );
                    // Emit a failure card and return early.
                    let (event_sender_tmp, mut event_receiver_tmp) = tokio::sync::mpsc::channel(4);
                    let _ = event_sender_tmp
                        .send(CognitiveEvent::TokenChunk(format!(
                            "Safety check blocked this action: {}",
                            violation.message
                        )))
                        .await;
                    let _ = event_sender_tmp
                        .send(CognitiveEvent::Complete {
                            stop_reason: StopReason::EndTurn,
                            usage: None,
                        })
                        .await;
                    drop(event_sender_tmp);
                    return stream_events_to_editor(
                        transport,
                        &session.session_id,
                        event_receiver_tmp,
                        &session.cancel_token,
                    )
                    .await
                    .map(|sr| sr.prompt_result);
                }
                ViolationSeverity::Warn => {
                    tracing::warn!(
                        session_id = %session.session_id,
                        violation = ?violation.violation_type,
                        message = %violation.message,
                        "ACP pre-dispatch safety warning"
                    );
                }
            }
        }
```
