## Problem
Generation prompts need to know which files and symbols are relevant to the feature being generated. Without this, agents invent file paths and struct names. We need a fast, bounded search that finds key files by name and content lines by keyword, populating `key_files` and `matching_symbols` in `RepoContextPack`.

## Architecture Contract
- `RepoContextPack.key_files`: up to 20 files found by name matching against feature keywords
- `RepoContextPack.matching_symbols`: up to 30 `SymbolHit` entries from content grep
- Exclude: `target/`, `node_modules/`, `.git/`, `tmp/`, `vendor/`
- Sort by relevance (exact name match > partial match > content match)
- Must complete within 5 seconds for a 200-crate workspace
- `SymbolHit.text` trimmed to max 200 chars
