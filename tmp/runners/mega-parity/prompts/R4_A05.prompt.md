## Task
Detect context-root mismatch

## Runner Context
You are working in runner `mega-parity`, batch R4_A05.
This batch is part of Runner 4: plan-grounding — Ground PRD/plan generation in the real repository and reject invalid artifacts.

## Problem
The most damaging failure mode is when a generation prompt targets a workspace that doesn't match the feature keywords. For example, generating a PRD about "roko-compose prompt assembly" in a temp directory, or generating a plan referencing "roko-agent" when running in an unrelated Go project. The `context_root_verified` flag must detect these mismatches and emit a visible warning.

## Architecture Contract
- `RepoContextPack.context_root_verified` = true when:
  1. At least one feature keyword matches a workspace member name, AND
  2. Real source files exist for that member (not just a manifest entry)
- `context_root_verified` = false when:
  1. Keywords reference crate/package names that don't exist in workspace, OR
  2. No source files found for any keyword match, OR
  3. Running in a temp/empty directory
- `to_prompt_section()` includes a prominent warning when `context_root_verified == false`

## Changes Required

Add to `crates/roko-cli/src/repo_context.rs`:

```rust
/// Verify that feature keywords are grounded in the actual workspace.
/// Returns true when keywords match workspace members AND real source exists.
pub fn verify_context_root(
    root: &Path,
    keywords: &[&str],
    workspace_members: &[String],
    key_files: &[PathBuf],
) -> bool {
    if keywords.is_empty() || workspace_members.is_empty() {
        return false;
    }

    // Check 1: At least one keyword matches a workspace member
    let keyword_matches_member = keywords.iter().any(|kw| {
        let kw_lower = kw.to_lowercase();
        workspace_members.iter().any(|m| {
            let m_lower = m.to_lowercase();
            m_lower.contains(&kw_lower) || kw_lower.contains(&m_lower)
        })
    });

    if !keyword_matches_member {
        return false;
    }

    // Check 2: At least one key file actually exists on disk
    let has_real_source = key_files.iter().any(|f| {
        let full_path = root.join(f);
        full_path.exists() && full_path.is_file()
    });

    has_real_source
}

/// Generate a context verification warning for unverified roots.
fn context_verification_warning(keywords: &[&str], workspace_members: &[String]) -> String {
    let kw_list = keywords.join(", ");
    let members_preview: Vec<&str> = workspace_members
        .iter()
        .take(10)
        .map(String::as_str)
        .collect();
    let members_list = members_preview.join(", ");
    let suffix = if workspace_members.len() > 10 {
        format!(" (and {} more)", workspace_members.len() - 10)
    } else {
        String::new()
    };
    format!(
        "> **WARNING: Repository context not verified.**\n\
         > The feature keywords [{}] do not match any workspace members [{}{}].\n\
         > Generated artifacts may reference nonexistent code. Review carefully before accepting.\n",
        kw_list, members_list, suffix
    )
}
```

Update `to_prompt_section()` to include the warning when `context_root_verified == false`:
```rust
// In to_prompt_section():
if !self.context_root_verified {
```
