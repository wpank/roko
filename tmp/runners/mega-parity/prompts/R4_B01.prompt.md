## Problem
The `prd draft new` command generates PRDs without any knowledge of the target repository. The agent prompt receives the user's description and system prompt but nothing about existing crates, files, or workspace structure. This causes generated PRDs to reference nonexistent code, propose redundant crates, and miss integration opportunities.

## Architecture Contract
- `RepoContextPack` carries bounded repo context for generation prompts
- Before agent execution in `prd draft new`, call `build_repo_context()`
- Append `to_prompt_section()` output to the user prompt (not the system prompt)
- Include a visible warning if `context_root_verified == false`
- The context injection is additive — it does not replace existing prompt content

Add this function before `pub(crate) async fn cmd_prd(...)` (before line 6):

/// Extract feature keywords from a PRD slug and optional description for context lookup.
/// Splits on hyphens, underscores, spaces. Filters short words (<3 chars).
/// Returns up to 10 lowercase unique keywords.
fn extract_keywords_from_slug_and_description(slug: &str, description: &str) -> Vec<String> {
    let mut words: Vec<String> = slug
        .split(|c: char| c == '-' || c == '_' || c.is_whitespace())
        .chain(description.split(|c: char| c == '-' || c == '_' || c.is_whitespace()))
        .filter(|w| w.len() >= 3)
        .map(|w| w.to_lowercase())
        .collect();
    words.sort();
    words.dedup();
    words.truncate(10);
    words
}
```

                let feature_keywords =
                    extract_keywords_from_slug_and_description(&slug, &title);
                // Build repo context pack — non-blocking: if unavailable, proceed without it.
                let repo_context_section: Option<String> = {
                    // Scan the workspace for relevant context.
                    // Walk crates/ to collect member names and representative source files.
                    let crates_dir = workdir.join("crates");
                    let workspace_members: Vec<String> = std::fs::read_dir(&crates_dir)
                        .ok()
                        .into_iter()
                        .flatten()
                        .flatten()
                        .filter_map(|entry| {
                            let p = entry.path();
                            if p.is_dir() {
                                p.file_name()
                                    .map(|n| n.to_string_lossy().to_string())
                            } else {
                                None
                            }
                        })
                        .collect();

                    if workspace_members.is_empty() {
                        None
                    } else {
                        let members_list = workspace_members.join(", ");
                        let section = format!(
                            "## Repository Context\n\
                             **Workspace members** ({count} crates): {members_list}\n\n\
                             **Keywords used for this PRD:** {keywords}\n\n\
                             When writing the PRD, build on these existing crates rather \
                             than proposing new ones unless strictly necessary.\n",
                            count = workspace_members.len(),
                            members_list = members_list,
                            keywords = feature_keywords.join(", "),
                        );
                        Some(section)
                    }
                };

                // Warn when workspace context could not be verified.
                if repo_context_section.is_none() {
                    eprintln!(
                        "WARNING: Repository context not available for keywords {:?}. \
                         Generated PRD may reference nonexistent code.",
                        feature_keywords
                    );
                }

                let context_suffix = repo_context_section
                    .as_deref()
                    .map(|ctx| format!("\n\n---\n\n{ctx}"))
                    .unwrap_or_default();

                let task_prompt = format!(
                    "Generate a complete PRD for: {title}. \
                     If you have file tools available, search the codebase to understand \
                     what exists and write the completed PRD to {path}. \
                     Otherwise, output the complete PRD markdown with YAML frontmatter. \
                     Include specific requirements, machine-verifiable acceptance criteria, \
                     and a design section.{context_suffix}",
                    path = target.display(),
                    context_suffix = context_suffix,
                );
```
