## Task
Validate PRD references existing surfaces

## Runner Context
You are working in runner `mega-parity`, batch R4_B03.
This batch is part of Runner 4: plan-grounding — Ground PRD/plan generation in the real repository and reject invalid artifacts.

## Problem
After a PRD is generated with a Repository Grounding section, we need to validate that the references in that section actually match the repository. A PRD that claims "no existing crates" when the workspace has 18 crates is dangerous. A PRD that proposes "new crate roko-compose" when it already exists is worse. This batch adds semantic validation of the grounding section.

## Architecture Contract
- `ArtifactValidationReport` separates process success from artifact quality
- Parse the `## Repository Grounding` section from the generated PRD
- Cross-reference against `RepoContextPack.workspace_members`
- "no existing crates" but workspace has members: WARNING
- "new crate roko-X" but roko-X already exists: ERROR
- Store results in `<slug>.validation.json` sidecar
- Print summary to stdout

use crate::repo_context::RepoContextPack;

/// Validation severity levels.
#[derive(Debug, Clone, Serialize, Deserialize, PartialEq)]
pub enum Severity {
    Error,
    Warning,
    Info,
}

/// A single validation issue found in an artifact.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ValidationIssue {
    pub severity: Severity,
    pub category: String,
    pub message: String,
    pub location: Option<String>,
}

/// Validation report for a generated artifact.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct ArtifactValidationReport {
    pub artifact_path: PathBuf,
    pub artifact_kind: String, // "prd" or "plan"
    pub process_success: bool,
    pub artifact_valid: bool,
    pub issues: Vec<ValidationIssue>,
    pub timestamp: String,
}

/// Validate PRD grounding section against the repository context.
fn validate_prd_grounding(
    prd_content: &str,
    slug: &str,
    context: &RepoContextPack,
    prd_path: &Path,
) -> ArtifactValidationReport {
    let mut issues = Vec::new();

    // 1. Extract the Repository Grounding section
    let grounding_text = extract_section(prd_content, "Repository Grounding");

    if grounding_text.is_none() {
        issues.push(ValidationIssue {
            severity: Severity::Warning,
            category: "missing_section".to_string(),
            message: "PRD has no '## Repository Grounding' section".to_string(),
            location: None,
        });
    }

    if let Some(text) = &grounding_text {
        let text_lower = text.to_lowercase();

        // 2. Check "no existing crates" claim against reality
        if (text_lower.contains("no existing crates") || text_lower.contains("no relevant crates"))
            && !context.workspace_members.is_empty()
        {
            issues.push(ValidationIssue {
                severity: Severity::Warning,
                category: "false_negative".to_string(),
                message: format!(
                    "PRD claims no existing crates but workspace has {} members",
                    context.workspace_members.len()
                ),
                location: Some("Repository Grounding".to_string()),
            });
        }

        // 3. Check "new crate" proposals against existing members
        for line in text.lines() {
            if let Some(crate_name) = extract_new_crate_proposal(line) {
                if context.do_not_create.iter().any(|existing| {
                    existing.to_lowercase() == crate_name.to_lowercase()
                }) {
                    issues.push(ValidationIssue {
                        severity: Severity::Error,
                        category: "duplicate_crate".to_string(),
                        message: format!(
                            "PRD proposes new crate '{}' but it already exists",
                            crate_name
                        ),
                        location: Some("Repository Grounding".to_string()),
                    });
                }
            }
        }
    }

    // Determine artifact_valid: no Errors means valid
    let has_errors = issues.iter().any(|i| i.severity == Severity::Error);

    ArtifactValidationReport {
        artifact_path: prd_path.to_path_buf(),
        artifact_kind: "prd".to_string(),
        process_success: true, // agent ran successfully if we got here
        artifact_valid: !has_errors,
        issues,
        timestamp: chrono::Utc::now().to_rfc3339(),
    }
}
