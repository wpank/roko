## Problem
There is no canonical type for usage observations with proper `Option<T>` semantics. The existing `roko_core::chat_types::Usage` uses non-Option `u32`/`f32`, making it impossible to distinguish "not reported" from "reported as zero." A new `UsageObservation` type is needed to carry Optional telemetry fields from the stream-json result event through to the episode and efficiency loggers.

## Architecture Contract
- All numeric fields in `UsageObservation` are `Option<T>` — None means "not reported"
- `wall_ms` is non-Option — we always measure wall time
- `serde(default)` on the struct for backward compatibility with old JSONL files
- Do NOT replace or remove `pub usage: Usage` on `AgentResult` — only ADD alongside it

### Step 1: Extend `crates/roko-agent/src/usage.rs`

Replace the entire file with:

```rust
//! Usage metrics for agent invocations.
//!
//! Two types are provided:
//!
//! - [`Usage`] — re-exported from `roko_core::chat_types`. Non-Option fields.
//!   Used by the existing `AgentResult.usage` field and compatibility code.
//!
//! - [`UsageObservation`] — new type with `Option<T>` fields distinguishing
//!   "not reported" (None) from "reported as zero" (Some(0)).
//!   Attached to `AgentResult.usage_obs`.

pub use roko_core::chat_types::Usage;

use serde::{Deserialize, Serialize};

/// Observed token and cost metrics from a single agent invocation.
///
/// All numeric fields are `Option<T>`:
/// - `None` means the provider did not report this value.
/// - `Some(0)` / `Some(0.0)` means the provider reported zero.
///
/// # Serde rules
///
/// Missing fields in existing JSONL deserialize to `None` (via `#[serde(default)]`).
/// `None` fields are omitted from JSON output (via `skip_serializing_if`).
#[derive(Debug, Clone, Default, Serialize, Deserialize)]
pub struct UsageObservation {
    /// Prompt / input tokens billed by the provider.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub input_tokens: Option<u64>,
    /// Completion / output tokens billed by the provider.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub output_tokens: Option<u64>,
    /// Tokens written to the provider-side prompt cache.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub cache_creation_tokens: Option<u64>,
    /// Tokens served from the provider-side prompt cache.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub cache_read_tokens: Option<u64>,
    /// Actual USD cost (after cache discounts), as reported by the provider.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub cost_usd: Option<f64>,
    /// Provenance of this observation.
    #[serde(default)]
    pub source: UsageSource,
    /// Model that produced these tokens, if known from the stream-json `result` event.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub model: Option<String>,
    /// Wall-clock duration in milliseconds. Always present (we control the timer).
    pub wall_ms: u64,
}

/// How the usage data was obtained.
#[derive(Debug, Clone, Default, Serialize, Deserialize, PartialEq, Eq)]
#[serde(rename_all = "snake_case")]
pub enum UsageSource {
    /// Provider reported usage directly in the stream-json `result` event.
    ProviderReported,
    /// Usage was estimated from model price tables.
    Estimated,
    /// Source is unknown.
    #[default]
    Unknown,
}
```
