## Problem
Cost events may be emitted multiple times for the same dispatch attempt — once on dispatch completion, again on gate failure retry, and possibly from multiple code paths. This inflates cost tracking and makes dashboards inaccurate.

## Architecture Contract
- One cost event per dispatch attempt
- Gate results are separate events, not cost events
- No duplication for gate failure + retry
