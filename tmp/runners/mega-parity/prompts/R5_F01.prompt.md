## Problem
ACP sessions produce zero episode data. Every dispatch in `orchestrate.rs` logs episodes to `.roko/episodes.jsonl` via `EpisodeLogger`. ACP dispatches are completely invisible to the learning system:
- CascadeRouter can't learn from editor usage (highest volume path)
- Efficiency tracking is blind to ACP
- Playbook generation has no ACP data to mine from
- Dream consolidation never sees ACP sessions

This is the worst form of the "built but never connected" anti-pattern: the EpisodeLogger exists, episodes.jsonl exists, orchestrate.rs writes to it — but ACP doesn't.

## Architecture Contract
- **TL-1:** Unknown != zero — store None/null for unavailable data, NEVER store 0
- **TL-3:** Model is known before logging — never log model = "unknown"
- **TL-5:** Learning reads what execution writes — write to the canonical path
- **ACP-3:** No silent learning gaps — every ACP dispatch MUST emit an episode
