## Problem
`WorkflowRun` has `total_cost_usd: f64` and `total_tokens: u64` fields that are ALWAYS 0. Provider responses include token counts and sometimes cost, but this data is never extracted or accumulated. The editor shows $0.00 for every workflow run.

Additionally, ACP never emits `UsageUpdate` notifications — the `SessionUpdate::UsageUpdate` variant is defined in `types.rs` but never constructed or sent. The editor's token/cost counter stays at zero.

This is a classic TL-1 violation: 0 is shown not because zero tokens were used, but because nobody extracts the data.

## Architecture Contract
- **TL-1:** Unknown != zero — use `Option<u64>` / `Option<f64>`. 0 means "actually zero." None means "unknown."
- **TL-2:** One cost event per attempt — not per stream chunk, not per turn
- **ACP-4:** No dead metrics — UsageUpdate must reflect actual usage after every dispatch

Add this free function in `bridge_events.rs`:

```rust
/// Calculate cost for a model given token usage.
/// Returns `None` if pricing is unknown (TL-1: unknown != zero).
pub fn calculate_cost_for_model_slug(
    model_slug: &str,
    input_tokens: u64,
    output_tokens: u64,
    cache_read_tokens: u64,
) -> Option<f64> {
    let (input_per_m, output_per_m) = match model_slug {
        s if s.contains("claude-opus") => (15.0_f64, 75.0_f64),
        s if s.contains("claude-sonnet") => (3.0_f64, 15.0_f64),
        s if s.contains("claude-haiku") => (0.25_f64, 1.25_f64),
        s if s.contains("gpt-4o-mini") => (0.15_f64, 0.6_f64),
        s if s.contains("gpt-4o") => (2.5_f64, 10.0_f64),
        s if s.contains("gemini-2.5-pro") => (1.25_f64, 10.0_f64),
        s if s.contains("gemini-2.0-flash") => (0.1_f64, 0.4_f64),
        _ => return None, // Unknown model pricing — return None, not 0.0 (TL-1)
    };

    let input_cost = (input_tokens as f64 / 1_000_000.0) * input_per_m;
    let output_cost = (output_tokens as f64 / 1_000_000.0) * output_per_m;
    // Cache reads at 10% of full input price
    let cache_discount = (cache_read_tokens as f64 / 1_000_000.0) * input_per_m * 0.9;

    Some((input_cost + output_cost - cache_discount).max(0.0))
}
```
