## Problem
When roko applies a pattern or makes a decision, the user has no visibility into WHY. Other agents treat context as a flat bag of tokens. Roko has the infrastructure to trace every decision back through:
- Which playbook suggested the pattern (mined from N episodes)
- Which episodes taught it (with success/failure rates)
- Which knowledge entries informed it (from neuro store)
- Which dream insight reinforced it (if any)

This provenance chain is the single most differentiating UX roko can offer — no other agent can do this because no other agent has persistent episodic memory + playbook mining + knowledge tiers.

## Architecture Contract
- **TL-5:** Learning reads what execution writes — provenance shows the full read-back chain
- **ACP-3:** No silent learning gaps — if roko learned something and uses it, the user sees the chain
- **ACP-4:** No dead metrics — confidence score must reflect real data (playbook success rates, knowledge scores), not fabricated constants
- **MP-4:** Improve truth before appearance — provenance must be factual analysis, not decoration

## Current State

### `bridge_events.rs` — No provenance emission
After R5_F05, knowledge hits and playbooks are queried. But there's no chain connecting:
- Playbook → source episodes → knowledge entries → dream insights
The user sees "3 knowledge hits" but not "this playbook was mined from 12 episodes where 10 succeeded, and dream cycle confirmed the pattern."

### Reference: Episode → Playbook link
Episodes have `source_episodes` and playbooks have `success_count`/`failure_count`. The link is:
- Episode.task_id → Playbook.id (playbook named after task type)
- PlaybookStore::record_outcome() increments success/failure counts
- Dreams mine episodes into knowledge entries (KnowledgeEntry.source_episodes)

### 2. Build provenance from existing query results

**Critical:** R5_F05 already queries KnowledgeStore and PlaybookStore. DO NOT re-query. Accept their results as input parameters.

```rust
/// Build provenance chain from already-queried results + additional episode/dream lookups.
/// The knowledge hits and playbooks come from R5_F05's query — we don't re-query them.
async fn build_provenance(
    knowledge_hits: &[KnowledgeQueryHit],
    playbooks: &[Playbook],
    prompt: &str,
    workdir: &Path,
) -> Option<ProvenanceChain> {
    let mut sources = Vec::new();

    // 1. Playbooks (from R5_F05 query — already available)
    for pb in playbooks {
        if let Some(rate) = pb.success_rate() {
            sources.push(ProvenanceSource::Playbook {
                id: pb.id.clone(),
                goal: pb.goal.chars().take(80).collect(),
                total_outcomes: pb.total_outcomes(),
                success_rate: rate,
            });
        }
    }

    // 2. Similar episodes (new query — scan recent episodes for task_id overlap)
    let episodes_path = workdir.join(".roko/episodes.jsonl");
    if episodes_path.exists() {
        if let Ok(episodes) = EpisodeLogger::read_all(&episodes_path).await {
            // Match episodes by playbook ID or by prompt similarity
            let matched_ids: std::collections::HashSet<&str> = playbooks.iter()
                .map(|pb| pb.id.as_str())
                .collect();

            let similar: Vec<_> = episodes.iter()
                .rev()
                .take(100) // scan last 100
                .filter(|ep| matched_ids.contains(ep.task_id.as_str()))
                .take(3)   // at most 3 episodes
                .collect();

            for ep in similar {
                let gate_summary = ep.gate_verdicts.iter()
                    .map(|gv| format!("{}:{}", gv.gate, if gv.passed { "pass" } else { "FAIL" }))
                    .collect::<Vec<_>>()
                    .join(" ");
                sources.push(ProvenanceSource::Episode {
                    task_id: ep.task_id.clone(),
                    success: ep.success,
                    gate_summary,
                });
            }
        }
    }

    // 3. Knowledge entries (from R5_F05 query — already available)
    for hit in knowledge_hits.iter().take(3) {
        sources.push(ProvenanceSource::Knowledge {
            kind: hit.entry.kind.clone(),
            tier: hit.entry.tier.clone(),
            score: hit.total_score,
            summary: hit.entry.content.chars().take(80).collect(),
        });
    }

    // 4. Dream routing advice (load from file if available)
    let advice_path = workdir.join(roko_dreams::DREAM_ROUTING_ADVICE_PATH);
    if advice_path.exists() {
        if let Ok(advice) = roko_dreams::load_dream_routing_advice(&advice_path) {
            // Find patterns relevant to prompt keywords
            let keywords: Vec<&str> = prompt.split_whitespace()
                .filter(|w| w.len() > 4)
                .take(5)
                .collect();
            let patterns = roko_dreams::relevant_pattern_summaries(&advice, &keywords);
            for pat in patterns.into_iter().take(2) {
                sources.push(ProvenanceSource::DreamPattern {
                    description: pat.description.clone(),
                    guidance: pat.guidance.chars().take(80).collect(),
                    confidence: pat.confidence,
                });
            }
        }
    }

    // Only emit if there's meaningful provenance (at least 1 playbook or 2+ sources)
    if sources.is_empty() || (playbooks.is_empty() && sources.len() < 2) {
        return None;
    }

    // Confidence from REAL data — average of source-specific scores
    let confidence = {
        let scores: Vec<f64> = sources.iter().filter_map(|s| match s {
            ProvenanceSource::Playbook { success_rate, .. } => Some(*success_rate),
            ProvenanceSource::Knowledge { score, .. } => Some(*score),
            ProvenanceSource::DreamPattern { confidence, .. } => Some(*confidence),
            ProvenanceSource::Episode { success, .. } => Some(if *success { 0.8 } else { 0.3 }),
        }).collect();

        if scores.is_empty() {
            0.0
        } else {
            scores.iter().sum::<f64>() / scores.len() as f64
        }
    };

    Some(ProvenanceChain { sources, confidence })
}
```

### 3. Emit provenance card

```rust
async fn emit_provenance_card(
    chain: &ProvenanceChain,
    event_sender: &mpsc::Sender<CognitiveEvent>,
) {
    let mut lines = Vec::new();

    for source in &chain.sources {
        match source {
            ProvenanceSource::Playbook { goal, total_outcomes, success_rate, .. } => {
                lines.push(format!(
                    "  Playbook ({} runs, {:.0}% success)\n    {}",
                    total_outcomes, success_rate * 100.0, goal
                ));
            }
            ProvenanceSource::Episode { task_id, success, gate_summary } => {
                let icon = if *success { "pass" } else { "FAIL" };
                lines.push(format!(
                    "  Episode {} [{}]\n    Gates: {}",
                    task_id, icon, gate_summary
                ));
            }
            ProvenanceSource::Knowledge { kind, tier, score, summary } => {
                lines.push(format!(
                    "  Knowledge [{:?}/{:?}] ({:.2})\n    {}",
                    tier, kind, score, summary
                ));
            }
            ProvenanceSource::DreamPattern { description, guidance, confidence } => {
                lines.push(format!(
                    "  Dream pattern ({:.0}%)\n    {} — {}",
                    confidence * 100.0, description, guidance
                ));
            }
        }
    }

    let card_text = format!(
        "{}sources, {:.0}% confidence\n\n{}",
        chain.sources.len(),
        chain.confidence * 100.0,
        lines.join("\n\n")
    );

    let tool_call_id = "provenance".to_string();
    let _ = event_sender.send(CognitiveEvent::ToolCallStart {
        tool_call_id: tool_call_id.clone(),
        title: "Decision provenance".to_string(),
        kind: ToolCallKind::Other,
    }).await;

    let _ = event_sender.send(CognitiveEvent::ToolCallComplete {
        tool_call_id,
        status: ToolCallStatus::Completed,
        content: vec![ContentBlock::Text { text: card_text }],
    }).await;
}
```
