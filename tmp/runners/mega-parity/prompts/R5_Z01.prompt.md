## Problem
Telemetry data flows through multiple writers and readers with no single map of what gets written where, in what format, or what reads it back. Unknown models appear as strings, costs show $0.00 when they should show unknown, and dashboards may read paths that execution never writes to.

## Architecture Contract
- UsageObservation type: unknown != zero
- One cost event per attempt
- Model known before logging
- Skipped gates are not passes
- Learning reads what execution writes
