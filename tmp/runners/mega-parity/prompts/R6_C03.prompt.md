### Step 5: Add expiry check helper

Add a private helper function `is_expired` after `load_transcript` (after line 146):

```rust
/// Returns `true` when the transcript has expired.
///
/// Transcripts without `expires_at` are treated as permanent (backward compatible).
fn is_expired(transcript: &RunTranscript) -> bool {
    let Some(ref expires_str) = transcript.expires_at else {
        return false; // permanent
    };
    let Ok(expires_at) = chrono::DateTime::parse_from_rfc3339(expires_str) else {
        return false; // malformed — treat as permanent
    };
    chrono::Utc::now() > expires_at
}
```
