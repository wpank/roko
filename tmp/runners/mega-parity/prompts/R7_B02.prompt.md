## Problem
After each assistant turn in `chat_inline.rs`, the cost and token information is silently recorded into `session.cost` but nothing is displayed to the user. Users have no visibility into per-turn cost and token usage. They must type `/stats` or `/cost` to see cumulative totals.

This task adds a brief usage line to the scrollback after each assistant turn completes.

## Architecture Contract
- Truth before appearance: unknown = "unknown", never "$0.00"
- Suppressible with a `quiet` field (added to ChatSession)
- Zero cost: show "free" not "$0.00"
- Print to the inline terminal (scrollback), NOT to stderr — the inline terminal IS the display

```rust
/// Print a brief cost/token summary line after each assistant turn.
///
/// Format: `  [model via provider | N tokens | $X.XX | Xs]`
///
/// If tokens are unknown (Claude CLI), shows "usage: unknown".
/// If cost is zero and tokens are zero, shows "free".
fn push_usage_line(
    term: &mut InlineTerminal,
    theme: &Theme,
    model: &str,
    input_tokens: u64,
    output_tokens: u64,
    cost: f64,
    latency_s: f64,
) -> std::io::Result<()> {
    let total = input_tokens + output_tokens;
    let time_str = format!("{latency_s:.1}s");

    let usage_text = if total == 0 && cost == 0.0 {
        // Claude CLI path — tokens not reported
        format!(
            "[{model} | usage: unknown | {time_str}]"
        )
    } else if cost == 0.0 && total > 0 {
        format!(
            "[{model} | {} tokens | free | {time_str}]",
            format_tokens(total)
        )
    } else {
        format!(
            "[{model} | {} tokens | ${cost:.4} | {time_str}]",
            format_tokens(total)
        )
    };

    term.push_lines(&[Line::from(vec![Span::styled(
        format!("  {usage_text}"),
        Style::default().fg(Theme::TEXT_GHOST),
    )])])
}

/// Format a token count with thousands separator (e.g. 1247 → "1,247").
fn format_tokens(n: u64) -> String {
    let s = n.to_string();
    let mut result = String::new();
    let chars: Vec<char> = s.chars().collect();
    for (i, ch) in chars.iter().enumerate() {
        if i > 0 && (chars.len() - i) % 3 == 0 {
            result.push(',');
        }
        result.push(*ch);
    }
    result
}
```

## Target Implementation

The complete `push_usage_line` function (to add to `chat_inline.rs`):

```rust
/// Print a compact per-turn usage summary to the inline terminal scrollback.
///
/// Format: `  [model via provider | N tokens | $X.XXXX | Xs]`
/// When tokens are unavailable (Claude CLI zero-value case): `  [model | usage: unknown | Xs]`
fn push_usage_line(
    term: &mut InlineTerminal,
    theme: &Theme,
    model_name: &str,
    input_tokens: u64,
    output_tokens: u64,
    cost: f64,
    latency_s: f64,
) -> std::io::Result<()> {
    let total = input_tokens + output_tokens;
    let text = if total == 0 {
        format!("  [{model_name} | usage: unknown | {latency_s:.1}s]")
    } else {
        let cost_str = if cost == 0.0 { "free".to_string() } else { format!("${cost:.4}") };
        format!("  [{model_name} | {total} tokens | {cost_str} | {latency_s:.1}s]")
    };
    term.push_lines(&[Line::from(vec![
        Span::styled(text, Style::default().fg(Theme::TEXT_GHOST)),
    ])])
}
```
