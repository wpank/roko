- Print security checklist
- Refuse without auth unless --unsafe-public

   - Templates (Railway, Fly, Docker) include auth configuration
   - Print security checklist before deploying:
     - [x] Auth enabled
     - [x] CORS configured
     - [x] Terminal disabled (or explicitly enabled)
     - [ ] (any missing items)
   - Allow bypass with `--unsafe-public` flag
