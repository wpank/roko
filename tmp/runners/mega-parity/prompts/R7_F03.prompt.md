## Problem Statement (revised based on actual code)

**The 4 slash commands are already implemented.** The original prompt was stale.

**The thread-safety gap is real:** `SessionManager` uses a bare `HashMap<String, AcpSession>`. The current ACP server is single-threaded (stdio loop), but if a WebSocket or HTTP transport is added in the future, concurrent access would race.

However, changing `SessionManager` to use `Arc<RwLock<>>` is a significant refactor that:
1. Changes all methods from sync to async
2. Requires updating every call site in the handler loop
3. Is preemptive (no current concurrency issue)

**Recommendation:** Instead of the full Arc<RwLock> refactor, add `Send + Sync` bounds verification and document the single-threaded contract. If the codebase actually needs concurrent access (e.g. HTTP handler is already spawning tasks), then do the full refactor.

pub async fn with_session_mut<F, R>(&self, id: &str, f: F) -> Option<R>
where
    F: FnOnce(&mut AcpSession) -> R,
{
    self.sessions.write().await.get_mut(id).map(f)
}

pub async fn list_sessions(&self) -> SessionListResult {
    let sessions_guard = self.sessions.read().await;
    let mut sessions: Vec<_> = sessions_guard.values().map(AcpSession::info).collect();
    sessions.sort_by(|l, r| l.created_at.cmp(&r.created_at).then_with(|| l.session_id.cmp(&r.session_id)));
    SessionListResult { sessions }
}

pub async fn load_session(&self, id: &str) -> Result<SessionNewResult, (i32, String)> {
    if let Some(session) = self.sessions.read().await.get(id) {
        return Ok(session.new_result());
    }
    if let Some(session) = self.load_from_disk(id) {
        let result = session.new_result();
        self.sessions.write().await.insert(session.session_id.clone(), session);
        return Ok(result);
    }
    Err((SESSION_NOT_FOUND, format!("ACP session '{id}' was not found")))
}

pub async fn persist_session(&self, session_id: &str) {
    let session = self.sessions.read().await.get(session_id).cloned();
    let Some(session) = session else { return };
    let sessions_dir = self.workdir.join(".roko").join("sessions");
    if let Err(e) = std::fs::create_dir_all(&sessions_dir) {
        tracing::warn!(error = %e, "failed to create sessions directory");
        return;
    }
    let path = sessions_dir.join(format!("{session_id}.json"));
    match serde_json::to_string_pretty(&session) {
        Ok(json) => {
            if let Err(e) = std::fs::write(&path, json) {
                tracing::warn!(path = %path.display(), error = %e, "failed to persist session");
            }
        }
        Err(e) => tracing::warn!(error = %e, "failed to serialize session for persistence"),
    }
}
