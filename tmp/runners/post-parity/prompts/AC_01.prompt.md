# AC_01: Add TokenBudget to DispatchKnowledge

## Task
Add a `TokenBudget` field to `DispatchKnowledge` so the ACP dispatch path can track and enforce token budgets across knowledge retrieval, context assembly, and model calls.

## Runner Context
Runner AC_A (ACP: Session Improvements), batch 1 of 3. No dependencies.

## Problem
`DispatchKnowledge` retrieves knowledge hits and playbooks but has no awareness of token budgets. It can retrieve arbitrarily large context without considering how much space is available in the model's context window.

### Step 1: Add TokenBudget struct (if not already defined)

```rust
/// Token budget for context assembly.
#[derive(Debug, Clone, Copy)]
pub struct TokenBudget {
    /// Total tokens available in the context window.
    pub total: usize,
    /// Tokens reserved for system prompt.
    pub system_prompt: usize,
    /// Tokens reserved for user message.
    pub user_message: usize,
    /// Tokens reserved for model output.
    pub output_reserve: usize,
    /// Tokens available for knowledge/context injection.
    pub available_for_context: usize,
}

impl TokenBudget {
    /// Create a budget for a given context window.
    pub fn for_context_window(window_size: usize) -> Self {
        let output_reserve = (window_size / 4).min(8192);
        let system_prompt = window_size / 6;
        let user_message = window_size / 6;
        let available = window_size
            .saturating_sub(output_reserve)
            .saturating_sub(system_prompt)
            .saturating_sub(user_message);
        Self {
            total: window_size,
            system_prompt,
            user_message,
            output_reserve,
            available_for_context: available,
        }
    }
}
```

### Step 3: Add budget-aware context_text

```rust
impl DispatchKnowledge {
    /// Render prompt context, truncating to fit the token budget.
    pub(crate) fn context_text_budgeted(&self) -> String {
        let full = self.context_text();
        match self.token_budget {
            Some(budget) => {
                // Rough estimate: 1 token ~= 4 chars
                let max_chars = budget.available_for_context * 4;
                if full.len() > max_chars {
                    let boundary = full.char_indices()
                        .take_while(|(i, _)| *i < max_chars)
                        .last()
                        .map(|(i, c)| i + c.len_utf8())
                        .unwrap_or(0);
                    format!("{}\n\n[context truncated to fit token budget]", &full[..boundary])
                } else {
                    full
                }
            }
            None => full,
        }
    }
}
```
