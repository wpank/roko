# AC_02: Build ContextManager with priority queue

## Task
Create a new `ContextManager` in `roko-acp` that manages context sections with priorities, ensuring the most important context is included first when the token budget is limited.

## Runner Context
Runner AC_A (ACP: Session Improvements), batch 2 of 3. No dependencies.

## Problem
Context assembly currently concatenates all available context (knowledge, playbooks, gate feedback, conventions) without prioritization. When the total context exceeds the token budget, there is no strategy for which sections to include. A priority queue ensures the highest-value context is included first.

## Exact Changes

### Step 1: Create `crates/roko-acp/src/context_manager.rs`

```rust
//! ContextManager -- priority-based context section management (AC_02).
//!
//! Manages named context sections with priorities and estimated token
//! counts. When budget is constrained, sections are included in
//! priority order until the budget is exhausted.

use std::collections::BinaryHeap;
use std::cmp::Ordering;

/// A section of context to be included in a prompt.
#[derive(Debug, Clone)]
pub struct ContextSection {
    /// Section name (e.g., "knowledge", "playbooks", "gate_feedback").
    pub name: String,
    /// Section content (markdown or plain text).
    pub content: String,
    /// Priority (higher = included first).
    pub priority: u32,
    /// Estimated token count (rough: chars / 4).
    pub estimated_tokens: usize,
}

impl ContextSection {
    /// Create a section with auto-estimated token count.
    pub fn new(name: impl Into<String>, content: impl Into<String>, priority: u32) -> Self {
        let content = content.into();
        let estimated_tokens = content.len() / 4;
        Self {
            name: name.into(),
            content,
            priority,
            estimated_tokens,
        }
    }
}

impl PartialEq for ContextSection {
    fn eq(&self, other: &Self) -> bool {
        self.priority == other.priority && self.name == other.name
    }
}

impl Eq for ContextSection {}

impl PartialOrd for ContextSection {
    fn partial_cmp(&self, other: &Self) -> Option<Ordering> {
        Some(self.cmp(other))
    }
}

impl Ord for ContextSection {
    fn cmp(&self, other: &Self) -> Ordering {
        self.priority.cmp(&other.priority)
            .then_with(|| self.name.cmp(&other.name))
    }
}

/// Manages context sections and assembles them within a token budget.
pub struct ContextManager {
    sections: BinaryHeap<ContextSection>,
    budget_tokens: Option<usize>,
}

impl ContextManager {
    /// Create a new ContextManager with optional token budget.
    pub fn new(budget_tokens: Option<usize>) -> Self {
        Self {
            sections: BinaryHeap::new(),
            budget_tokens,
        }
    }

    /// Add a context section.
    pub fn add(&mut self, section: ContextSection) {
        self.sections.push(section);
    }

    /// Add a section with the given priority.
    pub fn add_section(
        &mut self,
        name: impl Into<String>,
        content: impl Into<String>,
        priority: u32,
    ) {
        self.add(ContextSection::new(name, content, priority));
    }

    /// Assemble all sections within the budget, returning the combined text
    /// and the list of included section names.
    pub fn assemble(&self) -> (String, Vec<String>) {
        let mut sorted: Vec<_> = self.sections.iter().cloned().collect();
        sorted.sort_by(|a, b| b.cmp(a)); // highest priority first

        let mut result = String::new();
        let mut included = Vec::new();
        let mut tokens_used = 0;

        for section in sorted {
            if let Some(budget) = self.budget_tokens {
                if tokens_used + section.estimated_tokens > budget {
                    // Try to include a truncated version
                    let remaining = budget.saturating_sub(tokens_used);
                    if remaining > 100 {
                        let max_chars = remaining * 4;
                        let truncated = &section.content[..section.content.len().min(max_chars)];
                        result.push_str(&format!(
                            "\n## {} (truncated)\n\n{}\n",
                            section.name, truncated
                        ));
                        included.push(format!("{}(truncated)", section.name));
                    }
                    break;
                }
            }

            result.push_str(&format!("\n## {}\n\n{}\n", section.name, section.content));
            included.push(section.name.clone());
            tokens_used += section.estimated_tokens;
        }

        (result, included)
    }

    /// Total estimated tokens of all sections.
    pub fn total_estimated_tokens(&self) -> usize {
        self.sections.iter().map(|s| s.estimated_tokens).sum()
    }

    /// Number of sections.
    pub fn section_count(&self) -> usize {
        self.sections.len()
    }
}
```
