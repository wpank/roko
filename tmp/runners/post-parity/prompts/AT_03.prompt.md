# AT_03: Spawn N worktrees via WorktreeManager + arm-prefixed event bridge

## Task
Use `roko_orchestrator::worktree::WorktreeManager` to create N isolated git worktrees for an in-flight `TournamentRun`. Then construct a per-arm `mpsc::Sender<CognitiveEvent>` adapter that prefixes every emitted event with the arm id so the editor can render N parallel streams.

## Runner Context
Runner AT_B (ACP: Tournament Execution), batch 1 of 5. Depends on AT_02.

## Problem
After AT_01+AT_02 we have a model and config but no execution. The orchestrator already provides worktree management (`crates/roko-orchestrator/src/worktree.rs:163-269`), and ACP's existing event bridge (`crates/roko-acp/src/acp_adapter.rs::AcpAdapter`) emits events tied to a single `(session_id, run_id)`. For tournaments we need:

1. N worktrees created up front (atomic; if any fails, roll back).
2. An event multiplexer that routes per-arm events through ONE editor session, tagging each with `arm_id`.

### Step 1: Add a worktree-creation helper

In `crates/roko-acp/src/runner.rs` or new file `crates/roko-acp/src/tournament_runner.rs`:

```rust
//! Tournament execution helpers (AT_03).

use std::path::Path;
use std::sync::Arc;

use anyhow::{Context, Result, anyhow};
use roko_orchestrator::worktree::{WorktreeConfig, WorktreeError, WorktreeManager};
use tokio::sync::mpsc;

use crate::session::CognitiveEvent;
use crate::tournament::{ArmStatus, TournamentArm, TournamentRun};

/// Create one worktree per arm. If ANY creation fails, removes all
/// successfully-created worktrees and returns the first error.
///
/// Returns the WorktreeManager so subsequent steps can clean up.
pub async fn provision_worktrees(
    workdir: &Path,
    run: &mut TournamentRun,
) -> Result<WorktreeManager> {
    let manager = WorktreeManager::new(WorktreeConfig {
        repo_root: workdir.to_path_buf(),
        worktrees_root: workdir.join(".roko").join("worktrees"),
        base_branch: detect_base_branch(workdir).unwrap_or_else(|| "main".to_string()),
        max_live: Some(8),
        ..WorktreeConfig::default()
    });

    let mut created: Vec<String> = Vec::with_capacity(run.arms.len());
    for arm in &mut run.arms {
        let id = format!("tournament-{}-{}", run.run_id, arm.arm_id);
        match manager.create(&id, &arm.branch).await {
            Ok(handle) => {
                arm.worktree_path = handle.path;
                created.push(id);
            }
            Err(err) => {
                // Rollback any prior worktrees so we don't leak.
                for prior in &created {
                    if let Err(remove_err) = manager.remove(prior).await {
                        tracing::warn!(
                            id = %prior,
                            error = %remove_err,
                            "tournament rollback: failed to remove worktree"
                        );
                    }
                }
                return Err(anyhow!(err).context(format!(
                    "tournament arm {} worktree creation failed",
                    arm.arm_id
                )));
            }
        }
    }
    Ok(manager)
}

fn detect_base_branch(workdir: &Path) -> Option<String> {
    let out = std::process::Command::new("git")
        .arg("rev-parse")
        .arg("--abbrev-ref")
        .arg("HEAD")
        .current_dir(workdir)
        .output()
        .ok()?;
    if !out.status.success() { return None }
    let s = String::from_utf8_lossy(&out.stdout).trim().to_owned();
    if s.is_empty() || s == "HEAD" { None } else { Some(s) }
}
```

### Step 2: Per-arm event tagging

```rust
/// Wrap the parent event sender with an adapter that tags every
/// emitted event with the arm id, so the editor can render N
/// parallel streams in one session.
pub fn arm_tagged_sender(
    arm: &TournamentArm,
    parent: mpsc::Sender<CognitiveEvent>,
) -> mpsc::Sender<CognitiveEvent> {
    let (tx, mut rx) = mpsc::channel::<CognitiveEvent>(64);
    let arm_id = arm.arm_id.clone();
    tokio::spawn(async move {
        while let Some(mut event) = rx.recv().await {
            tag_with_arm(&mut event, &arm_id);
            if parent.send(event).await.is_err() { break }
        }
    });
    tx
}

/// Mutate an event's metadata to include `arm_id`. Schemas vary across
/// CognitiveEvent variants; this helper handles the common cases.
fn tag_with_arm(event: &mut CognitiveEvent, arm_id: &str) {
    use crate::types::SessionUpdate;
    if let CognitiveEvent::SessionUpdate(update) = event {
        match update {
            SessionUpdate::AgentMessageChunk { _meta, .. }
            | SessionUpdate::ToolCall { .. } => {
                merge_meta(_meta_mut(update), arm_id);
            }
            _ => {}
        }
    }
}

fn _meta_mut(update: &mut crate::types::SessionUpdate) -> &mut Option<serde_json::Value> {
    use crate::types::SessionUpdate as S;
    match update {
        S::AgentMessageChunk { _meta, .. } => _meta,
        // Other variants don't have _meta — return a sentinel via match exhaustion.
        // For variants without metadata we accept that arm_id is lost (logged below).
        _ => {
            tracing::trace!("CognitiveEvent variant without _meta — arm_id not tagged");
            // Fall back: store nowhere. Caller already knows the arm via channel.
            // Return a static None. We don't actually mutate.
            // (Workaround: callers that NEED arm_id should rely on the parent
            // multiplexer below, not on _meta tagging.)
            unreachable!("arm tagging fallthrough — see comment above");
        }
    }
}

fn merge_meta(meta: &mut Option<serde_json::Value>, arm_id: &str) {
    match meta {
        Some(serde_json::Value::Object(m)) => {
            m.insert("armId".into(), serde_json::Value::String(arm_id.to_string()));
        }
        _ => {
            *meta = Some(serde_json::json!({ "armId": arm_id }));
        }
    }
}
```

### Step 3: Cleanup helper

```rust
pub async fn cleanup_worktrees(manager: &WorktreeManager, run: &TournamentRun) {
    for arm in &run.arms {
        let id = format!("tournament-{}-{}", run.run_id, arm.arm_id);
        if let Err(err) = manager.remove(&id).await {
            tracing::warn!(arm = %arm.arm_id, error = %err, "tournament cleanup failed");
        }
    }
}
```

```rust
#[cfg(test)]
mod tests {
    use super::*;
    use crate::config::TournamentConfig;
    use crate::tournament::TournamentRun;

    #[tokio::test]
    async fn provision_worktrees_creates_all_then_paths_exist() {
        let tmp = tempfile::tempdir().unwrap();
        // Init a real git repo so worktree add succeeds
        std::process::Command::new("git").args(["init"]).current_dir(tmp.path()).status().unwrap();
        std::process::Command::new("git").args(["commit", "--allow-empty", "-m", "init"])
            .current_dir(tmp.path()).status().unwrap();

        let mut run = TournamentRun::build(
            "r1", "s1", "test", &tmp.path().join(".roko").join("worktrees"),
            TournamentConfig::default(),
        ).unwrap();

        let manager = provision_worktrees(tmp.path(), &mut run).await.unwrap();
        for arm in &run.arms {
            assert!(arm.worktree_path.exists(), "worktree {:?} should exist", arm.worktree_path);
        }
        cleanup_worktrees(&manager, &run).await;
    }

    #[tokio::test]
    async fn arm_tagged_sender_adds_arm_id_to_meta() {
        let (parent_tx, mut parent_rx) = mpsc::channel(8);
        let arm = TournamentArm {
            arm_id: "arm-1".into(),
            index: 0,
            strategy: "s".into(),
            branch: "b".into(),
            worktree_path: "p".into(),
            status: ArmStatus::Pending,
            workflow: None,
            gate_results: Default::default(),
            cost_usd: 0.0,
            tokens_used: 0,
            files_changed: vec![],
            commit_sha: None,
        };
        let arm_tx = arm_tagged_sender(&arm, parent_tx);

        // Send a chunk
        arm_tx.send(CognitiveEvent::SessionUpdate(
            crate::types::SessionUpdate::AgentMessageChunk {
                content: crate::types::ContentBlock::Text { text: "hi".into() },
                _meta: None,
            }
        )).await.unwrap();

        let event = parent_rx.recv().await.unwrap();
        if let CognitiveEvent::SessionUpdate(crate::types::SessionUpdate::AgentMessageChunk { _meta, .. }) = event {
            let m = _meta.unwrap();
            assert_eq!(m["armId"], "arm-1");
        } else {
            panic!("wrong event variant");
        }
    }
}
```
