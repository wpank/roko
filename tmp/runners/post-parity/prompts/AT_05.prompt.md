# AT_05: Emit comparison ToolCall card + RequestPermission for arm selection

## Task
After AT_04 finishes, render a "🏆 Tournament — N approaches" `ToolCall` card showing per-arm metrics (status, gates, files, cost, tokens) and a recommendation. Then issue a `session/request_permission` RPC asking the user to apply arm A / B / C / cancel. On approval, merge the chosen arm's branch into the host repo.

## Runner Context
Runner AT_C (ACP: Tournament Selection), batch 1 of 5. Depends on AT_04.

## Problem
After AT_04 we have a fully-populated `TournamentRun` but the editor sees only N parallel streams, never a comparison view. The user has no way to choose a winner. This batch:

1. Builds a comparison card matching `tmp/solutions/acp/08-NOVEL-WORKFLOWS.md` Workflow 5 mock.
2. Bridges the existing `RequestPermission` (R3_F04) to ask the user for a winner.
3. On approval, fast-forward-merges the arm branch into the active branch.
4. On rejection, leaves the worktrees as they are (so the user can `git checkout` manually) and emits a "no winner" chunk.

### Step 1: Comparison card renderer

In `crates/roko-acp/src/tournament_runner.rs`:

```rust
use crate::tournament::{ArmStatus, TournamentArm, TournamentRun};
use crate::types::{ContentBlock, SessionUpdate, ToolCallKind, ToolCallStatus};

pub fn render_comparison_card(run: &TournamentRun) -> SessionUpdate {
    let body = render_comparison_body(run);
    let recommended = run.recommended_arm().map(|a| a.arm_id.clone());
    let title = format!(
        "🏆 Tournament — {} approach{} compared",
        run.arms.len(),
        if run.arms.len() == 1 { "" } else { "es" },
    );
    SessionUpdate::ToolCall {
        tool_call_id: format!("tournament-comparison-{}", run.run_id),
        title,
        kind: ToolCallKind::Other,
        status: ToolCallStatus::Completed,
        content: vec![ContentBlock::Text { text: body }],
        locations: None,
        // Note: SessionUpdate::ToolCall may have additional fields — pass them as defaults.
    }
}

fn render_comparison_body(run: &TournamentRun) -> String {
    let mut out = String::with_capacity(1024);
    let recommended = run.recommended_arm().map(|a| a.arm_id.clone());

    for arm in &run.arms {
        let medal = if Some(&arm.arm_id) == recommended.as_ref() { "🏅 " } else { "   " };
        let status_glyph = arm_status_glyph(&arm.status);
        out.push_str(&format!("\n{medal}**{} ({})** {}\n", arm.arm_id, arm.strategy, status_glyph));

        // Gate results inline.
        let mut gates: Vec<_> = arm.gate_results.iter().collect();
        gates.sort_by_key(|(k, _)| k.clone());
        if !gates.is_empty() {
            let parts: Vec<String> = gates.iter()
                .map(|(name, passed)| {
                    if **passed { format!("{} ✓", name) }
                    else        { format!("{} ✗", name) }
                })
                .collect();
            out.push_str(&format!("    Gates: {}\n", parts.join(" · ")));
        }

        // Files + cost summary.
        out.push_str(&format!(
            "    Files: {}\n    Cost: ${:.2}, {} tokens\n",
            files_summary(&arm.files_changed),
            arm.cost_usd,
            arm.tokens_used,
        ));

        if let Some(sha) = &arm.commit_sha {
            out.push_str(&format!("    Commit: `{}`\n", &sha[..sha.len().min(8)]));
        }
    }

    if let Some(rec) = &recommended {
        out.push_str(&format!(
            "\n🏅 **Recommendation:** `{}` (best cost/quality ratio)\n",
            rec
        ));
    } else {
        out.push_str("\n⚠ **No eligible arm** — all arms failed required gates.\n");
    }

    out
}

fn arm_status_glyph(status: &ArmStatus) -> &'static str {
    match status {
        ArmStatus::Passed           => "✅ passed",
        ArmStatus::Failed           => "❌ failed",
        ArmStatus::BudgetExhausted  => "💸 budget",
        ArmStatus::Errored          => "💥 errored",
        ArmStatus::Cancelled        => "⏹ cancelled",
        ArmStatus::Running          => "⏳ running",
        ArmStatus::Pending          => "… pending",
    }
}

fn files_summary(files: &[String]) -> String {
    if files.is_empty() { return "(none)".into() }
    if files.len() <= 3 {
        files.join(", ")
    } else {
        format!("{} files", files.len())
    }
}
```

### Step 2: RequestPermission bridge

In `crates/roko-acp/src/tournament_runner.rs`:

```rust
use crate::types::{PermissionOption, RequestPermissionParams};

pub async fn request_winner_selection(
    session_id: &str,
    run: &TournamentRun,
    transport: &mut crate::transport::StdioTransport<impl tokio::io::AsyncRead + Unpin, impl tokio::io::AsyncWrite + Unpin>,
) -> anyhow::Result<Option<String>> {
    let mut options: Vec<PermissionOption> = run.eligible_arms()
        .iter()
        .map(|arm| PermissionOption {
            id: arm.arm_id.clone(),
            label: format!("Apply {} (${:.2})", arm.arm_id, arm.cost_usd),
            // Other fields default — verify shape.
        })
        .collect();
    options.push(PermissionOption {
        id: "cancel".into(),
        label: "Discard all (no merge)".into(),
    });

    if options.len() == 1 {
        // Only "cancel" available — no eligible arms.
        return Ok(None);
    }

    let recommended = run.recommended_arm().map(|a| a.arm_id.as_str()).unwrap_or("");

    let params = RequestPermissionParams {
        session_id: session_id.to_string(),
        title: "Choose a tournament winner".to_string(),
        description: format!(
            "{} arms passed all required gates. Recommended: `{}`.",
            run.eligible_arms().len(), recommended
        ),
        options,
        // Other fields default — verify shape.
    };

    // Send the request and await the response. The exact RPC plumbing depends
    // on the existing R3_F04 implementation. The shape below assumes there's
    // an `await_permission_response` helper somewhere; if not, we need to
    // wire one up here.
    //
    // FALLBACK: if no response helper exists, log + return None and document
    // the gap in the issue tracker below.
    let chosen = crate::bridge_events::request_permission_blocking(transport, params).await?;

    Ok(if chosen == "cancel" { None } else { Some(chosen) })
}
```

If `request_permission_blocking` doesn't exist in the codebase, add it to `bridge_events.rs` as a thin wrapper around the JSON-RPC `session/request_permission` call. Use the same correlation-id mechanism that R3_F04 already uses.

### Step 3: Apply the winning arm

```rust
use std::process::Command;

pub fn apply_winner(workdir: &std::path::Path, run: &TournamentRun, arm_id: &str) -> anyhow::Result<()> {
    let arm = run.arms.iter().find(|a| a.arm_id == arm_id)
        .ok_or_else(|| anyhow::anyhow!("arm {} not found", arm_id))?;

    // Fast-forward merge the arm's branch into the current branch.
    let status = Command::new("git")
        .args(["merge", "--ff-only", &arm.branch])
        .current_dir(workdir)
        .status()
        .map_err(|e| anyhow::anyhow!("git merge failed to spawn: {e}"))?;
    if !status.success() {
        return Err(anyhow::anyhow!(
            "git merge --ff-only {} failed; resolve manually",
            arm.branch
        ));
    }
    Ok(())
}
```

If fast-forward isn't possible (branches diverged because the user committed during the tournament), fall back to a regular merge with `--no-ff` AND emit a chunk warning the user. Do NOT cherry-pick — that loses arm history.

### Step 5: Wire into tournament driver

Modify the `run_tournament` function from AT_04 (in same file) to:

1. After arm execution finishes:
2. Emit `render_comparison_card(&run)` via the parent sender.
3. If `run.config.auto_apply_winner == false` and `eligible_arms().len() >= 1`:
   - Call `request_winner_selection`.
   - On `Some(arm_id)`: `apply_winner` + emit applied narrative.
   - On `None`: emit "no winner chosen" narrative.
4. If `auto_apply_winner == true` and there's a recommended arm:
   - Skip the prompt; apply directly.

#[test]
fn render_comparison_card_includes_medal_for_recommended() {
    let mut run = TournamentRun::build("r", "s", "p",
        std::path::Path::new("/tmp/no"), TournamentConfig::default()).unwrap();
    for arm in &mut run.arms { arm.status = ArmStatus::Passed;
        arm.gate_results.insert("compile".into(), true);
        arm.gate_results.insert("test".into(), true);
    }
    run.arms[1].cost_usd = 0.10;  // cheapest
    run.arms[0].cost_usd = 0.50;
    run.arms[2].cost_usd = 0.30;

    let update = render_comparison_card(&run);
    if let SessionUpdate::ToolCall { content, .. } = update {
        let text = content.iter().filter_map(|b| match b {
            ContentBlock::Text { text } => Some(text.clone()),
            _ => None,
        }).collect::<Vec<_>>().join("\n");
        assert!(text.contains("🏅"));
        assert!(text.contains("arm-2"));  // recommended
    } else {
        panic!("expected ToolCall");
    }
}
