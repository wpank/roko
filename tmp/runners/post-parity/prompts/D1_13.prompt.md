# D1_13: Implement ProviderHealthTracker

## Goal

Create a new `ProviderHealthTracker` struct that tracks per-provider
health status (success rate, latency, error counts) and provides a
`is_healthy` check for model routing decisions.

## Steps

### 1. Create `crates/roko-agent/src/provider/health.rs`

```rust
//! Provider health tracking for model routing decisions.
//!
//! Tracks per-provider success rates, latency windows, and circuit-breaker
//! state. Used by `ModelCallService` to avoid routing to degraded providers.

use std::collections::HashMap;
use std::sync::Mutex;
use std::time::{Duration, Instant};

/// Rolling window size for health metrics.
const WINDOW_SIZE: usize = 20;
/// Error rate threshold for marking a provider unhealthy.
const UNHEALTHY_THRESHOLD: f64 = 0.5;
/// Cooldown period before retrying an unhealthy provider.
const COOLDOWN: Duration = Duration::from_secs(60);

/// Health status of a single provider.
#[derive(Debug, Clone)]
pub struct ProviderHealth {
    /// Recent call outcomes (true = success, false = failure).
    outcomes: Vec<bool>,
    /// When the provider was last marked unhealthy.
    last_failure_time: Option<Instant>,
    /// Total successful calls.
    total_success: u64,
    /// Total failed calls.
    total_failure: u64,
    /// Rolling average latency in milliseconds.
    avg_latency_ms: f64,
}

impl Default for ProviderHealth {
    fn default() -> Self {
        Self {
            outcomes: Vec::with_capacity(WINDOW_SIZE),
            last_failure_time: None,
            total_success: 0,
            total_failure: 0,
            avg_latency_ms: 0.0,
        }
    }
}

impl ProviderHealth {
    /// Record a call outcome.
    pub fn record(&mut self, success: bool, latency_ms: u64) {
        if self.outcomes.len() >= WINDOW_SIZE {
            self.outcomes.remove(0);
        }
        self.outcomes.push(success);

        if success {
            self.total_success += 1;
        } else {
            self.total_failure += 1;
            self.last_failure_time = Some(Instant::now());
        }

        // EMA for latency
        let alpha = 0.2;
        self.avg_latency_ms =
            (1.0 - alpha) * self.avg_latency_ms + alpha * (latency_ms as f64);
    }

    /// Returns true if the provider is healthy enough for routing.
    pub fn is_healthy(&self) -> bool {
        if self.outcomes.is_empty() {
            return true; // No data = assume healthy
        }

        let recent_failures = self.outcomes.iter().filter(|o| !**o).count();
        let error_rate = recent_failures as f64 / self.outcomes.len() as f64;

        if error_rate >= UNHEALTHY_THRESHOLD {
            // Check cooldown
            if let Some(last_failure) = self.last_failure_time {
                if last_failure.elapsed() < COOLDOWN {
                    return false;
                }
            }
        }

        true
    }

    /// Recent error rate (0.0 to 1.0).
    pub fn error_rate(&self) -> f64 {
        if self.outcomes.is_empty() {
            return 0.0;
        }
        let failures = self.outcomes.iter().filter(|o| !**o).count();
        failures as f64 / self.outcomes.len() as f64
    }

    /// Average latency in milliseconds.
    pub fn avg_latency_ms(&self) -> f64 {
        self.avg_latency_ms
    }
}

/// Tracks health across all providers.
pub struct ProviderHealthTracker {
    providers: Mutex<HashMap<String, ProviderHealth>>,
}

impl ProviderHealthTracker {
    /// Create a new empty tracker.
    pub fn new() -> Self {
        Self {
            providers: Mutex::new(HashMap::new()),
        }
    }

    /// Record a call outcome for a provider.
    pub fn record(&self, provider: &str, success: bool, latency_ms: u64) {
        let mut providers = self.providers.lock().unwrap_or_else(|e| e.into_inner());
        providers
            .entry(provider.to_string())
            .or_default()
            .record(success, latency_ms);
    }

    /// Check if a provider is healthy.
    pub fn is_healthy(&self, provider: &str) -> bool {
        let providers = self.providers.lock().unwrap_or_else(|e| e.into_inner());
        providers
            .get(provider)
            .map_or(true, |h| h.is_healthy())
    }

    /// Get health info for a provider.
    pub fn health(&self, provider: &str) -> Option<ProviderHealth> {
        let providers = self.providers.lock().unwrap_or_else(|e| e.into_inner());
        providers.get(provider).cloned()
    }
}

impl Default for ProviderHealthTracker {
    fn default() -> Self {
        Self::new()
    }
}
```

### 3. Add tests

At the bottom of `health.rs`:

```rust
#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn new_provider_is_healthy() {
        let tracker = ProviderHealthTracker::new();
        assert!(tracker.is_healthy("anthropic"));
    }

    #[test]
    fn provider_becomes_unhealthy_after_failures() {
        let tracker = ProviderHealthTracker::new();
        for _ in 0..WINDOW_SIZE {
            tracker.record("test", false, 100);
        }
        assert!(!tracker.is_healthy("test"));
    }

    #[test]
    fn mixed_results_stay_healthy() {
        let tracker = ProviderHealthTracker::new();
        for i in 0..WINDOW_SIZE {
            tracker.record("test", i % 3 != 0, 100);
        }
        assert!(tracker.is_healthy("test"));
    }
}
```

## Constraints

- Use `std::sync::Mutex` (not `tokio::sync::Mutex`) — health tracking is
  lightweight and does not need async locking.
- Poison recovery with `unwrap_or_else(|e| e.into_inner())` — never panic
  on poisoned mutex.
- The tracker is thread-safe (`Send + Sync`) for use in `Arc`.
