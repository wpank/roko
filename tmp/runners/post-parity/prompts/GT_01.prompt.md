# GT_01: Define RokoAdapter trait (Bevy plugin pattern)

## Task
Define the core `RokoAdapter` trait and `AdapterRegistry` that external integrations (GitHub, Linear, Slack) implement to plug into roko's event system, following a Bevy-style plugin registration pattern.

## Runner Context
Runner GT_A (GTM: Adapter Foundation), batch 1 of 3. No dependencies.

## Problem
External integrations (GitHub issues, Linear tickets, Slack notifications) need a standard way to: (1) receive events from roko (task completed, gate failed), (2) push work items into roko (issue created, ticket assigned), (3) query roko state. There is no unified adapter interface.

## Exact Changes

### Step 1: Create `crates/roko-core/src/adapter.rs`

```rust
//! Adapter trait and registry for external integrations (GT_01).
//!
//! Follows the Bevy plugin pattern: each adapter registers itself
//! with the registry at startup, declaring its capabilities.

use std::collections::HashMap;
use std::sync::Arc;

use async_trait::async_trait;
use serde::{Deserialize, Serialize};

/// Capabilities an adapter can provide.
#[derive(Debug, Clone, Copy, PartialEq, Eq, Hash, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum AdapterCapability {
    /// Can receive work items (issues, tickets, PRs).
    WorkSource,
    /// Can send notifications (Slack, email, webhook).
    Notification,
    /// Can observe events (logging, metrics, audit).
    Observer,
    /// Can provide webhook ingress.
    WebhookReceiver,
}

/// Metadata about a registered adapter.
#[derive(Debug, Clone)]
pub struct AdapterInfo {
    /// Unique adapter name (e.g., "github", "linear", "slack").
    pub name: String,
    /// Human-readable description.
    pub description: String,
    /// Capabilities this adapter provides.
    pub capabilities: Vec<AdapterCapability>,
    /// Version string.
    pub version: String,
}

/// Core adapter trait. All integrations implement this.
#[async_trait]
pub trait RokoAdapter: Send + Sync {
    /// Adapter metadata.
    fn info(&self) -> AdapterInfo;

    /// Initialize the adapter. Called once at startup.
    async fn init(&self) -> Result<(), AdapterError>;

    /// Shutdown the adapter. Called on clean exit.
    async fn shutdown(&self) -> Result<(), AdapterError>;

    /// Health check. Returns Ok if the adapter is operational.
    async fn health(&self) -> Result<(), AdapterError>;
}

/// Error type for adapter operations.
#[derive(Debug, thiserror::Error)]
pub enum AdapterError {
    #[error("adapter configuration error: {0}")]
    Config(String),
    #[error("adapter connection error: {0}")]
    Connection(String),
    #[error("adapter operation failed: {0}")]
    Operation(String),
}

/// Registry of all loaded adapters.
pub struct AdapterRegistry {
    adapters: HashMap<String, Arc<dyn RokoAdapter>>,
}

impl AdapterRegistry {
    /// Create an empty registry.
    pub fn new() -> Self {
        Self {
            adapters: HashMap::new(),
        }
    }

    /// Register an adapter.
    pub fn register(&mut self, adapter: Arc<dyn RokoAdapter>) {
        let info = adapter.info();
        self.adapters.insert(info.name.clone(), adapter);
    }

    /// Get an adapter by name.
    pub fn get(&self, name: &str) -> Option<&Arc<dyn RokoAdapter>> {
        self.adapters.get(name)
    }

    /// List all registered adapters.
    pub fn list(&self) -> Vec<AdapterInfo> {
        self.adapters.values().map(|a| a.info()).collect()
    }

    /// Find adapters with a given capability.
    pub fn with_capability(&self, cap: AdapterCapability) -> Vec<&Arc<dyn RokoAdapter>> {
        self.adapters
            .values()
            .filter(|a| a.info().capabilities.contains(&cap))
            .collect()
    }

    /// Initialize all adapters.
    pub async fn init_all(&self) -> Vec<(String, Result<(), AdapterError>)> {
        let mut results = Vec::new();
        for (name, adapter) in &self.adapters {
            results.push((name.clone(), adapter.init().await));
        }
        results
    }

    /// Shutdown all adapters.
    pub async fn shutdown_all(&self) -> Vec<(String, Result<(), AdapterError>)> {
        let mut results = Vec::new();
        for (name, adapter) in &self.adapters {
            results.push((name.clone(), adapter.shutdown().await));
        }
        results
    }
}

impl Default for AdapterRegistry {
    fn default() -> Self {
        Self::new()
    }
}
```

- `RokoAdapter` trait with init/shutdown/health lifecycle
- `AdapterCapability` enum (WorkSource, Notification, Observer, WebhookReceiver)
- `AdapterRegistry` with register/get/list/with_capability
- `AdapterError` error type
- Module registered in roko-core
