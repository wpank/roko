# GT_02: Define SubAdapter traits (Work/Notification/Observer)

## Task
Define specialized sub-traits (`WorkSourceAdapter`, `NotificationAdapter`, `ObserverAdapter`) that extend `RokoAdapter` with capability-specific methods.

## Runner Context
Runner GT_A (GTM: Adapter Foundation), batch 2 of 3. Depends on GT_01.

## Problem
The base `RokoAdapter` trait handles lifecycle but different capabilities need different methods: work sources need to list/create issues, notification adapters need to send messages, observers need to receive events.

## Current Code

**RokoAdapter** -- from GT_01 (`crates/roko-core/src/adapter.rs`):
Base trait with `info()`, `init()`, `shutdown()`, `health()`.

## Exact Changes

### Step 1: Add sub-traits to `adapter.rs`

```rust
/// A work item from an external source.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct WorkItem {
    /// External ID (e.g., GitHub issue number, Linear ticket ID).
    pub external_id: String,
    /// Source adapter name.
    pub source: String,
    /// Title of the work item.
    pub title: String,
    /// Description / body.
    pub body: String,
    /// Labels or tags.
    pub labels: Vec<String>,
    /// Priority (1=highest, 5=lowest).
    pub priority: Option<u32>,
    /// External URL.
    pub url: Option<String>,
}

/// Adapter that can provide work items.
#[async_trait]
pub trait WorkSourceAdapter: RokoAdapter {
    /// List open work items.
    async fn list_work_items(&self) -> Result<Vec<WorkItem>, AdapterError>;

    /// Get a specific work item by external ID.
    async fn get_work_item(&self, id: &str) -> Result<Option<WorkItem>, AdapterError>;

    /// Update the status of a work item (e.g., close an issue).
    async fn update_work_item_status(
        &self,
        id: &str,
        status: &str,
        comment: Option<&str>,
    ) -> Result<(), AdapterError>;
}

/// A notification to send to an external system.
#[derive(Debug, Clone)]
pub struct Notification {
    /// Channel or recipient.
    pub target: String,
    /// Notification title.
    pub title: String,
    /// Notification body (markdown).
    pub body: String,
    /// Severity level.
    pub severity: NotificationSeverity,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum NotificationSeverity {
    Info,
    Warning,
    Error,
}

/// Adapter that can send notifications.
#[async_trait]
pub trait NotificationAdapter: RokoAdapter {
    /// Send a notification.
    async fn notify(&self, notification: &Notification) -> Result<(), AdapterError>;

    /// List available channels/targets.
    async fn list_targets(&self) -> Result<Vec<String>, AdapterError>;
}

/// Event data passed to observers.
#[derive(Debug, Clone)]
pub struct AdapterEvent {
    /// Event kind (task_completed, gate_failed, plan_started, etc.).
    pub kind: String,
    /// Structured payload.
    pub payload: serde_json::Value,
    /// Timestamp.
    pub timestamp: String,
}

/// Adapter that observes roko events.
#[async_trait]
pub trait ObserverAdapter: RokoAdapter {
    /// Called when a roko event occurs.
    async fn on_event(&self, event: &AdapterEvent) -> Result<(), AdapterError>;
}
```
