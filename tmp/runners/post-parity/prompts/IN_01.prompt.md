# IN_01: Create MemoryLayer wrapping working/episodic/semantic stores

## Task
Create a `MemoryLayer` that provides a unified interface over working memory (current session), episodic memory (episode log), and semantic memory (neuro store).

## Runner Context
Runner IN, batch 1 of 9. No dependencies.

## Problem
Agents currently access memory through three separate APIs: session-local state, `EpisodeLogger` for past experiences, and `NeuroStore` for durable knowledge. A `MemoryLayer` unifies these into a single query interface with automatic tier selection.

### Step 1: Create `crates/roko-learn/src/memory_layer.rs`

```rust
//! MemoryLayer — unified interface over working, episodic, and semantic memory.
//!
//! Provides a single `recall()` method that queries all three tiers and
//! merges results by relevance.

use crate::episode_logger::{Episode, EpisodeLogger};
use serde::{Deserialize, Serialize};
use std::path::PathBuf;

/// A memory retrieval result from any tier.
#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct MemoryItem {
    /// Which tier this item came from.
    pub tier: MemoryTier,
    /// Content text.
    pub content: String,
    /// Relevance score (0.0 to 1.0).
    pub relevance: f64,
    /// Source identifier.
    pub source: String,
    /// When this memory was created/last accessed.
    pub timestamp: Option<String>,
}

/// Memory tier classification.
#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash, Serialize, Deserialize)]
pub enum MemoryTier {
    /// Current session state, very recent, highest relevance.
    Working,
    /// Past task episodes, medium-term.
    Episodic,
    /// Durable knowledge, long-term.
    Semantic,
}

impl MemoryTier {
    /// Default relevance boost for items from this tier.
    pub const fn recency_boost(self) -> f64 {
        match self {
            Self::Working => 0.3,
            Self::Episodic => 0.1,
            Self::Semantic => 0.0,
        }
    }
}

/// Unified memory layer over three storage tiers.
pub struct MemoryLayer {
    /// Working memory: current session items.
    working: Vec<MemoryItem>,
    /// Path to episode log for episodic recall.
    episodes_path: Option<PathBuf>,
    /// Optional knowledge store query function.
    semantic_query: Option<Box<dyn Fn(&str, usize) -> Vec<MemoryItem> + Send + Sync>>,
    /// Maximum items to return per recall.
    max_results: usize,
}

impl MemoryLayer {
    /// Create a new memory layer.
    pub fn new() -> Self {
        Self {
            working: Vec::new(),
            episodes_path: None,
            semantic_query: None,
            max_results: 10,
        }
    }

    /// Create with episode log path.
    pub fn with_episodes(mut self, path: PathBuf) -> Self {
        self.episodes_path = Some(path);
        self
    }

    /// Attach a semantic query function.
    pub fn with_semantic_query<F>(mut self, query: F) -> Self
    where
        F: Fn(&str, usize) -> Vec<MemoryItem> + Send + Sync + 'static,
    {
        self.semantic_query = Some(Box::new(query));
        self
    }

    /// Set maximum results per recall.
    pub fn with_max_results(mut self, n: usize) -> Self {
        self.max_results = n;
        self
    }

    /// Add an item to working memory.
    pub fn remember(&mut self, content: String, source: String) {
        self.working.push(MemoryItem {
            tier: MemoryTier::Working,
            content,
            relevance: 1.0,
            source,
            timestamp: Some(chrono::Utc::now().to_rfc3339()),
        });

        // Keep working memory bounded
        if self.working.len() > 50 {
            self.working.remove(0);
        }
    }

    /// Recall relevant memories across all tiers.
    ///
    /// Results are sorted by relevance (with tier-based recency boost).
    pub fn recall(&self, query: &str, limit: usize) -> Vec<MemoryItem> {
        let effective_limit = limit.min(self.max_results);
        let mut results = Vec::new();

        // 1. Working memory: simple substring match
        for item in &self.working {
            let relevance = if item.content.to_lowercase().contains(&query.to_lowercase()) {
                0.8 + MemoryTier::Working.recency_boost()
            } else {
                0.2 + MemoryTier::Working.recency_boost()
            };
            results.push(MemoryItem {
                relevance,
                ..item.clone()
            });
        }

        // 2. Episodic memory: recent episodes matching query
        if let Some(ref path) = self.episodes_path {
            let logger = EpisodeLogger::new(path.clone());
            if let Ok(episodes) = logger.recent(20) {
                for ep in episodes {
                    let relevance = if ep.task_id.to_lowercase().contains(&query.to_lowercase()) {
                        0.6 + MemoryTier::Episodic.recency_boost()
                    } else {
                        0.3 + MemoryTier::Episodic.recency_boost()
                    };
                    results.push(MemoryItem {
                        tier: MemoryTier::Episodic,
                        content: format!(
                            "Task {}: {} (model: {}, cost: ${:.4})",
                            ep.task_id,
                            if ep.gate_verdict == crate::episode_logger::GateVerdict::Pass { "PASS" } else { "FAIL" },
                            ep.model,
                            ep.cost_usd,
                        ),
                        relevance,
                        source: format!("episode:{}", ep.task_id),
                        timestamp: Some(ep.timestamp.clone()),
                    });
                }
            }
        }

        // 3. Semantic memory: knowledge store query
        if let Some(ref query_fn) = self.semantic_query {
            let semantic_items = query_fn(query, effective_limit);
            results.extend(semantic_items);
        }

        // Sort by relevance descending, take top N
        results.sort_by(|a, b| b.relevance.partial_cmp(&a.relevance).unwrap_or(std::cmp::Ordering::Equal));
        results.truncate(effective_limit);

        results
    }

    /// Clear working memory.
    pub fn clear_working(&mut self) {
        self.working.clear();
    }

    /// Number of items in working memory.
    pub fn working_count(&self) -> usize {
        self.working.len()
    }
}

impl Default for MemoryLayer {
    fn default() -> Self {
        Self::new()
    }
}

#[cfg(test)]
mod tests {
    use super::*;

    #[test]
    fn working_memory_recall() {
        let mut mem = MemoryLayer::new();
        mem.remember("the foo module handles bar".into(), "session".into());
        mem.remember("baz requires qux".into(), "session".into());

        let results = mem.recall("foo", 5);
        assert!(!results.is_empty());
        assert!(results[0].relevance > 0.5);
    }

    #[test]
    fn working_memory_bounded() {
        let mut mem = MemoryLayer::new();
        for i in 0..100 {
            mem.remember(format!("item {i}"), "test".into());
        }
        assert!(mem.working.len() <= 50);
    }
}
```

- Each tier has its own recency boost
- Tests pass for basic recall and bounding
