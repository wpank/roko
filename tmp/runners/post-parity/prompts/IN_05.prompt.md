# IN_05: Wire dynamic gate generation from failure patterns

## Task
Extend `GateEvolver` to generate dynamic gate checks from extracted failure patterns.

## Runner Context
Runner IN, batch 5 of 9. Depends on IN_04.

## Problem
`GateEvolver` extracts failure patterns (IN_04) but doesn't act on them. Dynamic gate generation creates new shell-based checks that catch recurring patterns before they waste agent retries.

## Current Code

**GateEvolver** (IN_04): `extract_patterns()` returns `Vec<FailurePattern>`.
**FailurePattern**: Has `suggested_check`, `hot_files`, `category`.
**GateService** -- `crates/roko-gate/src/gate_service.rs`: Supports shell gates.

## Exact Changes

### Step 1: Add gate generation methods to GateEvolver

In `crates/roko-learn/src/gate_evolver.rs`, add:

```rust
/// A dynamically generated gate check.
#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct DynamicGate {
    /// Gate name for display and tracking.
    pub name: String,
    /// Shell command to run.
    pub command: String,
    /// Arguments.
    pub args: Vec<String>,
    /// Expected exit code for success (typically 0).
    pub expected_exit_code: i32,
    /// Description of what this check catches.
    pub description: String,
    /// Source pattern that generated this gate.
    pub source_pattern: String,
    /// Priority (higher = run earlier in gate pipeline).
    pub priority: u32,
}

impl GateEvolver {
    /// Generate dynamic gates from extracted failure patterns.
    ///
    /// Only generates gates for patterns that have a derivable check command.
    pub fn generate_gates(&self) -> Vec<DynamicGate> {
        self.patterns
            .iter()
            .filter_map(|pattern| {
                let check = pattern.suggested_check.as_ref()?;

                // Parse command and args from the check string
                let parts: Vec<&str> = check.split_whitespace().collect();
                if parts.is_empty() {
                    return None;
                }

                let command = parts[0].to_string();
                let args: Vec<String> = parts[1..].iter().map(|s| s.to_string()).collect();

                Some(DynamicGate {
                    name: format!("dynamic-{}-{}", pattern.category, sanitize_label(&pattern.label)),
                    command,
                    args,
                    expected_exit_code: 0,
                    description: format!(
                        "Auto-generated from failure pattern (seen {}x): {}",
                        pattern.frequency,
                        pattern.label
                    ),
                    source_pattern: pattern.label.clone(),
                    priority: pattern.frequency,
                })
            })
            .collect()
    }

    /// Generate gates and persist them to a TOML file for the gate pipeline.
    pub fn generate_and_persist(&self, output_path: &Path) -> std::io::Result<usize> {
        let gates = self.generate_gates();
        if gates.is_empty() {
            return Ok(0);
        }

        let mut toml_content = String::from("# Auto-generated gates from failure pattern analysis\n\n");

        for gate in &gates {
            toml_content.push_str(&format!(
                "[[gate]]\nname = \"{}\"\ncommand = \"{}\"\nargs = {:?}\ndescription = \"{}\"\n\n",
                gate.name, gate.command, gate.args,
                gate.description.replace('"', "'")
            ));
        }

        std::fs::write(output_path, toml_content)?;
        Ok(gates.len())
    }

    /// Generate gates suitable for injection into GateConfig.
    ///
    /// Returns tuples of (name, command) that can be added to the
    /// gate pipeline as shell gates.
    pub fn gate_configs(&self) -> Vec<(String, String)> {
        self.generate_gates()
            .into_iter()
            .map(|g| {
                let full_cmd = if g.args.is_empty() {
                    g.command
                } else {
                    format!("{} {}", g.command, g.args.join(" "))
                };
                (g.name, full_cmd)
            })
            .collect()
    }
}

fn sanitize_label(label: &str) -> String {
    label
        .chars()
        .take(20)
        .map(|c| if c.is_alphanumeric() { c } else { '-' })
        .collect::<String>()
        .to_lowercase()
}
```
