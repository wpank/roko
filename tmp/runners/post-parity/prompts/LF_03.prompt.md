# LF_03: Wire episode logging into runner v2 event loop

## Task
Ensure the runner v2 event loop emits full `Episode` records per task to the `EpisodeLogger`.

## Runner Context
Runner LF, batch 3 of 6. No dependencies.

## Problem
The runner event loop dispatches agents for tasks and receives completion events, but may not be constructing and recording complete `Episode` records with all fields (tokens, cost, model, role, gate verdict, duration, HDC fingerprint). The `EpisodeLogger` exists and is referenced but episodes may be incomplete or missing fields.

## Exact Changes

### Step 1: Ensure EpisodeLogger is available in run state

In `crates/roko-cli/src/runner/event_loop.rs`, verify the run loop has access to an `EpisodeLogger`. If not already present, add it to the run configuration or state:

```rust
let episode_logger = EpisodeLogger::new(roko_dir.join("episodes.jsonl"));
```

### Step 2: Build complete Episode on task completion

After a task completes (agent result + gate result available), construct a full episode:

```rust
    let episode = Episode {
        task_id: task_id.clone(),
        role: agent_role.to_string(),
        model: model_slug.clone(),
        tokens_in: completion.tokens_in,
        tokens_out: completion.tokens_out,
        cost_usd: completion.cost_usd,
        duration_ms: completion.duration.as_millis() as u64,
        gate_verdict: match &gate_result {
            Some(gr) if gr.passed => GateVerdict::Pass,
            Some(_) => GateVerdict::Fail,
            None => GateVerdict::Skip,
        },
        hdc_fingerprint: compute_hdc_fingerprint(&task_id, &completion.output_text),
        timestamp: chrono::Utc::now().to_rfc3339(),
    };
```

### Step 3: Log the episode

```rust
    if let Err(e) = episode_logger.log(&episode) {
        warn!(error = %e, task = %task_id, "failed to log episode");
    }
```

### Step 4: Add HDC fingerprint helper if missing

```rust
fn compute_hdc_fingerprint(task_id: &str, output: &str) -> Option<String> {
    use std::collections::hash_map::DefaultHasher;
    use std::hash::{Hash, Hasher};
    let mut hasher = DefaultHasher::new();
    task_id.hash(&mut hasher);
    output.hash(&mut hasher);
    Some(format!("{:016x}", hasher.finish()))
}
```
