# LF_07: Wire AnomalyDetector into dispatch path

## Task
Instantiate `AnomalyDetector` at session start and check each dispatch for prompt loops, cost spikes, and quality drift.

## Runner Context
Runner LF, batch 7 of 9. No dependencies.

## Problem
`AnomalyDetector` exists with `check_prompt()`, `check_cost()`, `check_quality()`, and `check_budget()` methods. It detects runaway loops, cost spikes, and quality degradation. But no live code path instantiates it or calls its check methods. Anomalies go undetected.

    if let Some(anomaly) = anomaly_detector.check_prompt(prompt_hash) {
