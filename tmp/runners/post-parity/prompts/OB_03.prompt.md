# OB_03: Emit gen_ai.* spans for model calls

## Task
Add OpenTelemetry-compatible `gen_ai.*` semantic convention spans to `GatewayEventWriter` so model calls are observable via standard LLM observability tooling.

## Runner Context
Runner OB_A (Observability: Tracing), batch 3 of 4. Depends on OB_02.

## Problem
Model calls are logged to `gateway.jsonl` but not emitted as structured tracing spans with the `gen_ai.*` semantic conventions used by OpenTelemetry LLM observability tools. Adding these spans enables integration with Langfuse, Helicone, and OTLP collectors.

## Exact Changes

### Step 1: Add span emission function

```rust
/// Emit a tracing span following the OpenTelemetry gen_ai.* semantic conventions.
///
/// See: https://opentelemetry.io/docs/specs/semconv/gen-ai/
///
/// This function creates and immediately closes a span with the correct
/// attributes for LLM observability tooling.
pub fn emit_gen_ai_span(event: &GatewayEvent) {
    // Create span with gen_ai.* attributes
    let span = tracing::info_span!(
        "gen_ai.chat",
        "gen_ai.system" = event.provider.as_deref().unwrap_or("unknown"),
        "gen_ai.request.model" = %event.model,
        "gen_ai.response.model" = %event.model,
        "gen_ai.usage.input_tokens" = event.input_tokens,
        "gen_ai.usage.output_tokens" = event.output_tokens,
        "gen_ai.usage.total_tokens" = event.input_tokens + event.output_tokens,
        "gen_ai.response.finish_reason" = if event.success { "stop" } else { "error" },
        // Roko-specific extensions
        "roko.request_id" = %event.request_id,
        "roko.caller" = %event.caller,
        "roko.cost_usd" = event.cost_usd,
        "roko.latency_ms" = event.latency_ms,
        "roko.cache_hit" = event.cache_hit,
    );

    let _guard = span.enter();

    if let Some(ref error) = event.error {
        tracing::error!(error = %error, "gen_ai.chat failed");
    }
}
```

### Step 2: Wire into GatewayEventWriter::write

After the existing JSONL write, emit the span:

```rust
impl GatewayEventWriter {
    pub fn write(&self, event: &GatewayEvent) -> std::io::Result<()> {
        // existing JSONL write
        self.write_jsonl(event)?;

        // Emit gen_ai.* span for observability
        emit_gen_ai_span(event);

        Ok(())
    }
}
```

## Acceptance Criteria
- `emit_gen_ai_span()` creates spans with `gen_ai.*` attributes
- Span covers: system, model, token counts, finish reason, cost, latency
- Emitted on every `GatewayEventWriter::write()` call
- No behavior change to existing JSONL writing
- Works without OpenTelemetry (tracing spans are no-ops without a subscriber)

## Do NOT
- Add OpenTelemetry as a hard dependency (use tracing spans only)
- Modify the `GatewayEvent` struct
- Change the JSONL format
