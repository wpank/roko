# OB_05: Add RouterTrace widget to TUI

## Task
Create a `RouterTrace` widget for the ratatui TUI that displays the CascadeRouter's model selection decisions in real time.

## Runner Context
Runner OB_B (Observability: Dashboard), batch 1 of 4. No dependencies.

## Problem
The CascadeRouter makes model selection decisions based on task complexity, cost, and routing history, but these decisions are invisible in the TUI. Operators cannot see why a particular model was chosen or how routing is performing.

## Current Code

**TUI widgets** -- `crates/roko-cli/src/tui/widgets/`:
Contains existing widget implementations. Check what widgets exist and follow the pattern.

**CascadeRouter** -- `crates/roko-learn/src/cascade_router.rs`:
Has `route()` method returning `CascadeRouteExplanation` with model, reason, cost estimate.

## Exact Changes

### Step 1: Create `crates/roko-cli/src/tui/widgets/router_trace.rs`

```rust
//! RouterTrace widget — displays CascadeRouter decisions (OB_05).

use ratatui::prelude::*;
use ratatui::widgets::{Block, Borders, Paragraph, Row, Table};

/// A single routing decision for display.
#[derive(Debug, Clone)]
pub struct RouterTraceEntry {
    pub timestamp: String,
    pub task: String,
    pub model: String,
    pub reason: String,
    pub cost_estimate: f64,
}

/// Widget that displays a table of recent routing decisions.
pub struct RouterTrace<'a> {
    entries: &'a [RouterTraceEntry],
    title: &'a str,
}

impl<'a> RouterTrace<'a> {
    pub fn new(entries: &'a [RouterTraceEntry]) -> Self {
        Self {
            entries,
            title: "Router Trace",
        }
    }

    pub fn title(mut self, title: &'a str) -> Self {
        self.title = title;
        self
    }
}

impl Widget for RouterTrace<'_> {
    fn render(self, area: Rect, buf: &mut Buffer) {
        let header = Row::new(vec!["Time", "Task", "Model", "Reason", "Est. Cost"])
            .style(Style::default().fg(Color::Yellow).add_modifier(Modifier::BOLD));

        let rows: Vec<Row> = self.entries.iter().rev().take(20).map(|e| {
            Row::new(vec![
                e.timestamp.clone(),
                truncate_str(&e.task, 30),
                e.model.clone(),
                truncate_str(&e.reason, 40),
                format!("${:.4}", e.cost_estimate),
            ])
        }).collect();

        let widths = [
            Constraint::Length(12),
            Constraint::Length(30),
            Constraint::Length(20),
            Constraint::Fill(1),
            Constraint::Length(10),
        ];

        let table = Table::new(rows, widths)
            .header(header)
            .block(Block::default().borders(Borders::ALL).title(self.title));

        Widget::render(table, area, buf);
    }
}

fn truncate_str(s: &str, max: usize) -> String {
    if s.len() <= max {
        s.to_string()
    } else {
        format!("{}...", &s[..max.saturating_sub(3)])
    }
}
```
