# OB_06: Add CostPanel widget to TUI

## Task
Create a `CostPanel` widget for the ratatui TUI that displays running cost totals, per-model breakdown, and budget utilization.

## Runner Context
Runner OB_B (Observability: Dashboard), batch 2 of 4. No dependencies.

## Problem
Cost data is tracked in `CostTable` and gateway events but not displayed in the TUI. Operators need real-time visibility into spending to make routing decisions and set budgets.

## Current Code

**CostTable** -- `crates/roko-learn/src/cost_table.rs`:
Has per-model cost tracking with `total_cost()`, `per_model_costs()`.

## Exact Changes

### Step 1: Create `crates/roko-cli/src/tui/widgets/cost_panel.rs`

```rust
//! CostPanel widget — displays running cost totals (OB_06).

use ratatui::prelude::*;
use ratatui::widgets::{Bar, BarChart, BarGroup, Block, Borders, Paragraph};

/// Cost data for display.
#[derive(Debug, Clone)]
pub struct CostData {
    pub total_cost_usd: f64,
    pub budget_usd: Option<f64>,
    pub per_model: Vec<(String, f64)>,
    pub per_role: Vec<(String, f64)>,
}

/// Widget that displays cost information.
pub struct CostPanel<'a> {
    data: &'a CostData,
}

impl<'a> CostPanel<'a> {
    pub fn new(data: &'a CostData) -> Self {
        Self { data }
    }
}

impl Widget for CostPanel<'_> {
    fn render(self, area: Rect, buf: &mut Buffer) {
        let chunks = Layout::default()
            .direction(Direction::Vertical)
            .constraints([
                Constraint::Length(3), // summary line
                Constraint::Fill(1),   // bar chart
            ])
            .split(area);

        // Summary
        let budget_str = self.data.budget_usd
            .map(|b| format!(" / ${b:.2} budget ({:.0}%)", self.data.total_cost_usd / b * 100.0))
            .unwrap_or_default();
        let summary = Paragraph::new(format!(
            "Total: ${:.4}{budget_str}",
            self.data.total_cost_usd
        ))
        .block(Block::default().borders(Borders::ALL).title("Cost"));
        Widget::render(summary, chunks[0], buf);

        // Per-model bar chart
        if !self.data.per_model.is_empty() {
            let max_cost = self.data.per_model.iter()
                .map(|(_, c)| *c)
                .fold(0.0_f64, f64::max);
            let scale = if max_cost > 0.0 { 100.0 / max_cost } else { 1.0 };

            let bars: Vec<Bar> = self.data.per_model.iter().map(|(model, cost)| {
                Bar::default()
                    .label(Line::from(model.clone()))
                    .value((cost * scale) as u64)
                    .text_value(format!("${cost:.4}"))
            }).collect();

            let chart = BarChart::default()
                .block(Block::default().borders(Borders::ALL).title("Per-Model Cost"))
                .data(BarGroup::default().bars(&bars))
                .bar_width(12)
                .bar_gap(1)
                .bar_style(Style::default().fg(Color::Cyan));

            Widget::render(chart, chunks[1], buf);
        }
    }
}
```
