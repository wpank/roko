# OB_09: Cost spike anomaly detector

## Task
Add a cost spike anomaly detector that subscribes to the runtime event bus and emits alerts when per-call or per-run costs exceed a configurable threshold above the running average.

## Runner Context
Runner OB_C (Observability: Alerting), batch 1 of 3. No dependencies.

## Problem
Cost spikes (e.g., a model call that costs 10x the average) can indicate prompt injection, runaway loops, or model misconfiguration. There is no automated detection -- operators only discover cost issues after reviewing gateway.jsonl.

## Current Code

**EventBus** -- `crates/roko-runtime/src/event_bus.rs`:
Has `subscribe()` and `emit()`. Events include `RuntimeEvent` variants.

## Exact Changes

### Step 1: Add anomaly detector types

In `crates/roko-runtime/src/event_bus.rs` (or a new `alerting.rs`):

```rust
/// Configuration for cost spike detection.
#[derive(Debug, Clone)]
pub struct CostSpikeConfig {
    /// Alert when a single call costs more than this multiple of the EMA.
    pub spike_multiplier: f64,
    /// Minimum number of samples before alerting (avoid false positives on startup).
    pub min_samples: usize,
    /// EMA decay factor (0.0-1.0). Higher = more responsive.
    pub ema_alpha: f64,
}

impl Default for CostSpikeConfig {
    fn default() -> Self {
        Self {
            spike_multiplier: 5.0,
            min_samples: 10,
            ema_alpha: 0.1,
        }
    }
}

/// Detects cost spikes using exponential moving average.
pub struct CostSpikeDetector {
    config: CostSpikeConfig,
    ema: f64,
    sample_count: usize,
}

impl CostSpikeDetector {
    pub fn new(config: CostSpikeConfig) -> Self {
        Self {
            config,
            ema: 0.0,
            sample_count: 0,
        }
    }

    /// Feed a new cost observation. Returns `Some(alert_message)` if spike detected.
    pub fn observe(&mut self, cost_usd: f64, context: &str) -> Option<String> {
        self.sample_count += 1;

        if self.sample_count <= 1 {
            self.ema = cost_usd;
            return None;
        }

        // Update EMA
        self.ema = self.config.ema_alpha * cost_usd + (1.0 - self.config.ema_alpha) * self.ema;

        // Check for spike
        if self.sample_count >= self.config.min_samples
            && cost_usd > self.ema * self.config.spike_multiplier
        {
            Some(format!(
                "Cost spike: ${cost_usd:.4} is {:.1}x the average ${:.4} ({context})",
                cost_usd / self.ema,
                self.ema,
            ))
        } else {
            None
        }
    }

    /// Current EMA value.
    pub fn current_ema(&self) -> f64 {
        self.ema
    }
}
```

### Step 2: Add alert emission

```rust
/// Alert event emitted when an anomaly is detected.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AlertEvent {
    pub kind: AlertKind,
    pub message: String,
    pub timestamp: String,
    pub severity: AlertSeverity,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum AlertKind {
    CostSpike,
    PassRateDrop,
    LatencySurge,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum AlertSeverity {
    Warning,
    Critical,
}
```
