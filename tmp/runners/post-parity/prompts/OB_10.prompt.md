# OB_10: Pass-rate drop detector

## Task
Add a pass-rate drop detector that monitors gate pass rates and alerts when the rate drops below a configurable threshold over a sliding window.

## Runner Context
Runner OB_C (Observability: Alerting), batch 2 of 3. No dependencies.

## Problem
A sudden drop in gate pass rates (e.g., from 90% to 30%) indicates a systemic issue -- bad model output, broken build environment, or a configuration change. There is no automated detection.

## Current Code

**AdaptiveThresholds** -- `crates/roko-gate/src/adaptive_threshold.rs`:
Tracks per-rung pass streaks and EMA, but does not emit alerts.

## Exact Changes

### Step 1: Add pass-rate detector

In `crates/roko-runtime/src/event_bus.rs` (alongside CostSpikeDetector):

```rust
/// Detects drops in gate pass rate over a sliding window.
pub struct PassRateDetector {
    /// Circular buffer of recent gate outcomes (true=pass, false=fail).
    window: std::collections::VecDeque<bool>,
    /// Window size.
    window_size: usize,
    /// Alert when pass rate drops below this threshold.
    alert_threshold: f64,
    /// Minimum observations before alerting.
    min_samples: usize,
}

impl PassRateDetector {
    pub fn new(window_size: usize, alert_threshold: f64) -> Self {
        Self {
            window: std::collections::VecDeque::with_capacity(window_size),
            window_size,
            alert_threshold,
            min_samples: window_size / 2,
        }
    }

    /// Record a gate outcome. Returns alert if pass rate dropped below threshold.
    pub fn observe(&mut self, passed: bool, gate_name: &str) -> Option<String> {
        if self.window.len() >= self.window_size {
            self.window.pop_front();
        }
        self.window.push_back(passed);

        if self.window.len() < self.min_samples {
            return None;
        }

        let pass_count = self.window.iter().filter(|&&p| p).count();
        let rate = pass_count as f64 / self.window.len() as f64;

        if rate < self.alert_threshold {
            Some(format!(
                "Pass rate drop: {:.0}% over last {} gates (threshold: {:.0}%, triggered by '{gate_name}')",
                rate * 100.0,
                self.window.len(),
                self.alert_threshold * 100.0,
            ))
        } else {
            None
        }
    }

    /// Current pass rate (0.0-1.0).
    pub fn current_rate(&self) -> f64 {
        if self.window.is_empty() {
            return 1.0;
        }
        let pass_count = self.window.iter().filter(|&&p| p).count();
        pass_count as f64 / self.window.len() as f64
    }
}
```
