# PA_01: Create shared_http_client() factory in roko-agent

## Task
Add a `shared_http_client()` factory function to `roko-agent/src/http.rs` that returns `Arc<ReqwestPoster>`.

## Runner Context
Runner PA, batch 1 of 5. No dependencies.

## Problem
Every LLM call, health check, and API request creates a fresh `reqwest::Client::new()`. This destroys connection pooling, TLS session reuse, and HTTP/2 multiplexing (7-43s latency regression observed).

## Current Code

`crates/roko-agent/src/http.rs:116-134` — `ReqwestPoster` struct:
```rust
pub struct ReqwestPoster {
    client: reqwest::Client,
}

impl ReqwestPoster {
    pub fn new() -> Self {
        Self {
            client: reqwest::Client::new(),
        }
    }

    pub const fn with_client(client: reqwest::Client) -> Self {
        Self { client }
    }
}
```

`crates/roko-agent/src/http.rs:71-112` — `HttpPoster` trait (the abstraction seam):
```rust
#[async_trait]
pub trait HttpPoster: Send + Sync {
    async fn post_json(&self, url: &str, headers: &[(String, String)], body: &[u8], timeout_ms: u64) -> Result<String, HttpPostError>;
    async fn get_json(...) -> Result<String, HttpPostError> { ... }
    async fn delete_json(...) -> Result<String, HttpPostError> { ... }
}
```

## Exact Changes

### Step 1: Add factory functions at bottom of `http.rs`

After the existing `impl HttpPoster for ReqwestPoster` block, add:

```rust
use std::sync::Arc;

/// Create a shared HTTP client for the entire process.
/// Call once at startup, clone the Arc everywhere.
pub fn shared_http_client() -> Arc<ReqwestPoster> {
    Arc::new(ReqwestPoster::new())
}

/// Create a shared HTTP client wrapping an existing reqwest::Client.
pub fn shared_http_client_from(client: reqwest::Client) -> Arc<ReqwestPoster> {
    Arc::new(ReqwestPoster::with_client(client))
}
```

### Step 2: Re-export from `crates/roko-agent/src/lib.rs`

Find the existing `pub use http::` line and add the new exports:
```rust
pub use http::{shared_http_client, shared_http_client_from, HttpPoster, ReqwestPoster};
```

If no `pub use http::` line exists, find where other modules are re-exported and add one.
