# PA_04: Wire shared HTTP client into roko-agent-server sidecar

## Task
Replace `reqwest::Client::new()` in `roko-agent-server` with a shared client stored in sidecar AppState.

## Runner Context
Runner PA, batch 4 of 5. Depends on PA_01.

## Problem
The per-agent sidecar (`roko-agent-server`) has one known `reqwest::Client::new()` call in `lib.rs:417` inside `heartbeat_loop()`. This creates a new client per agent spawn.

## Current Code

`crates/roko-agent-server/src/lib.rs:417` — in `heartbeat_loop()`:
```rust
let client = reqwest::Client::new();
```

This function runs once per spawned agent and sends periodic heartbeats to the control plane. Each spawn gets its own client.
