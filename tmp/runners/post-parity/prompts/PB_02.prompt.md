### Step 1: Define `ChatDispatchRequest` struct in `chat_session.rs`

Add after the `ChatAgentSession` struct definition (after line 68):

```rust
/// Everything needed to make one LLM call, assembled from session state.
#[derive(Debug, Clone)]
pub struct ChatDispatchRequest {
    /// System prompt (from /system command or SystemPromptBuilder).
    pub system_prompt: String,
    /// Conversation history for multi-turn.
    pub messages: Vec<ChatMessage>,
    /// Tool allowlist as comma-separated names.
    pub allowed_tools_csv: String,
    /// Resolved model slug.
    pub model: String,
    /// Provider identifier.
    pub provider: String,
    /// Reasoning effort level.
    pub effort: String,
    /// MCP config file path.
    pub mcp_config: Option<PathBuf>,
    /// Session ID for --resume (multi-turn CLI).
    pub session_id: Option<String>,
    /// Working directory.
    pub workdir: PathBuf,
    /// Shared HTTP client.
    pub http_client: reqwest::Client,
    /// Settings JSON path.
    pub settings_json: Option<PathBuf>,
    /// Per-turn timeout.
    pub timeout: Option<Duration>,
}
```

### Step 2: Add `build_dispatch_request()` method

Add to `impl ChatAgentSession`:

```rust
/// Assemble all session state into a dispatch request.
pub fn build_dispatch_request(&self) -> ChatDispatchRequest {
    ChatDispatchRequest {
        system_prompt: self.system_prompt.clone(),
        messages: self.api_history.clone(),
        allowed_tools_csv: self.allowed_tools_csv.clone(),
        model: self.model_selection.model_slug().to_string(),
        provider: self.model_selection.provider().to_string(),
        effort: self.effort.clone(),
        mcp_config: self.mcp_config.clone(),
        session_id: self.session_id.clone(),
        workdir: self.workdir.clone(),
        http_client: self.http_client.clone(),
        settings_json: self.settings_json.clone(),
        timeout: self.timeout,
    }
}
```

### Step 3: Add `update_from_response()` method

Add to `impl ChatAgentSession`:

```rust
/// Update session state from a dispatch response.
pub fn update_from_response(&mut self, session_id: Option<String>, assistant_text: &str) {
    // Capture session_id for --resume on next turn
    if let Some(id) = session_id {
        self.session_id = Some(id);
    }
    // Append assistant response to history
    self.api_history.push(ChatMessage {
        role: "assistant".to_string(),
        content: assistant_text.to_string(),
    });
}
```
