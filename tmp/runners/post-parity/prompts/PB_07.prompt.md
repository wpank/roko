# PB_07: Replace dispatch_direct with adapter-layer dispatch

## Task
Replace the hand-rolled HTTP dispatch in `dispatch_direct.rs` with calls through `ClaudeCliAgent` / API adapter, using `ChatDispatchRequest` from PB_02.

## Runner Context
Runner PB, batch 7 of 8. Depends on PB_03, PB_04, PB_05, PB_06. This is the keystone batch.

## Problem

**dispatch_direct.rs** (line 10: `#![cfg(feature = "legacy-orchestrate")]`) hand-rolls provider HTTP:
- Lines 22-35: `dispatch_prompt()` routes by `AuthMethod`
- `dispatch_claude_cli()` builds bare `claude --print --output-format stream-json` with NO session state
- `dispatch_anthropic_api()` builds raw JSON with NO system prompt, tools, history
- `dispatch_openai_compat()` same gaps

**dispatch_v2.rs** — `dispatch_via_model_call_service()` is the non-legacy path, called at chat_inline.rs:1490:
```rust

let result = crate::dispatch_v2::dispatch_via_model_call_service(&text).await;
```
This also only receives the text prompt, not the full session state.

**chat_inline.rs:1459-1496** — `dispatch_prompt()` spawns dispatch:

## Exact Changes

### Step 1: Create `dispatch_chat_request()` function

In `chat_inline.rs` (or a new `chat_dispatch.rs` if the file is too large), add:

```rust
async fn dispatch_chat_request(request: ChatDispatchRequest) -> Result<DispatchResult> {
    // Route based on provider
    match request.provider.as_str() {
        "claude-cli" | "claude" | "" => dispatch_via_cli(&request).await,
        _ => dispatch_via_api(&request).await,
    }
}

async fn dispatch_via_cli(request: &ChatDispatchRequest) -> Result<DispatchResult> {
    let mut agent = ClaudeCliAgent::new(
        "claude",
        &request.workdir,
        &request.model,
    )
    .with_system_prompt(&request.system_prompt)
    .with_effort(&request.effort);

    if !request.allowed_tools_csv.is_empty() {
        agent = agent.with_tools(&request.allowed_tools_csv);
    }

    if let Some(mcp) = &request.mcp_config {
        agent = agent.with_mcp_config(mcp.clone());
    }

    if let Some(sid) = &request.session_id {
        agent = agent.with_resume(sid.clone());
    }

    // Run and convert to DispatchResult
    let output = agent.run_print(&request.messages.last()
        .map(|m| m.content.as_str())
        .unwrap_or("")).await?;
    // ... convert AgentOutput to DispatchResult
}

async fn dispatch_via_api(request: &ChatDispatchRequest) -> Result<DispatchResult> {
    // Use ModelCallService or direct API with full request state
    crate::dispatch_v2::dispatch_with_request(request).await
}
```
