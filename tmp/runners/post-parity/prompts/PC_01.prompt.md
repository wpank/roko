# PC_01: Document current streaming data flow

## Task
Read and document how streaming token deltas flow from LLM to TUI, identifying the gap where `StreamingState.append()` is never called with live data.

## Runner Context
Runner PC, batch 1 of 6. Depends on PB_07.

## Problem
`StreamingState` exists with an `append()` method. SSE/streaming events arrive from the LLM. But the two are never connected — `append()` is never called with live token deltas.
