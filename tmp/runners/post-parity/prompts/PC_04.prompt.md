# PC_04: Wire Claude API SSE events to StreamingCallback

## Task
When using Claude API backend (Anthropic API or OpenAI-compat), pipe SSE `content_block_delta` events through `StreamingCallback`.

## Runner Context
Runner PC, batch 4 of 6. Depends on PC_02.

## Problem
API providers return SSE streams with `content_block_delta` events containing token text. The `dispatch_v2` path (or `ModelCallService`) processes these internally but doesn't forward deltas in real-time.

## Current Code

**dispatch_v2.rs** — `dispatch_via_model_call_service()` makes API calls through `ModelCallService`. Search for where SSE response bytes are read and parsed.

**Anthropic API provider** — `crates/roko-agent/src/provider/anthropic_api.rs` — find the SSE event loop where `content_block_delta` events are parsed.

**OpenAI-compat backend** — `crates/roko-agent/src/openai_compat_backend.rs:389` — creates `reqwest::Client::new()` for streaming. The streaming loop parses `data: {...}` lines with `choices[0].delta.content`.

**ACP bridge streaming** — `crates/roko-acp/src/bridge_events.rs:687-745` — streams SSE chunks, tracks `total_input` and `total_output` tokens.

## Exact Changes

```rust
pub async fn dispatch_with_request_streaming(
    request: &ChatDispatchRequest,
    callback: Option<&dyn StreamingCallback>,
) -> Result<DispatchResult> {
    // ... existing setup ...
    // In the SSE reading loop:
    // When content_block_delta with text_delta arrives:
    if let Some(cb) = callback {

```

### Step 2: Wire into Anthropic API SSE parser

Find where `content_block_delta` events are parsed in the Anthropic API provider. Add callback forwarding:

```rust
// In the SSE event loop:
"content_block_delta" => {
    if let Some(text) = delta.get("text").and_then(|v| v.as_str()) {
        buffer.push_str(text);
        if let Some(cb) = callback {
            cb.on_token(text);
        }
    }
}
```

### Step 3: Wire into OpenAI-compat SSE parser

In `openai_compat_backend.rs`, find the streaming response loop:

```rust
// In the SSE chunk parsing:
if let Some(content) = choice.get("delta").and_then(|d| d.get("content")).and_then(|c| c.as_str()) {
    buffer.push_str(content);
    if let Some(cb) = callback {
        cb.on_token(content);
    }
}
```
