# PE_02: Flip dangerously_skip_permissions defaults to false

## Task
Change all hardcoded `dangerously_skip_permissions: true` to `false` at every NEEDS FIX site from PE_01.

## Runner Context
Runner PE, batch 2 of 8. Depends on PE_01.

## Problem
From PE_01's audit, 7 production sites hardcode `dangerously_skip_permissions: true` without user opt-in. SA-1 anti-pattern: "Default is safe."

## Exact Sites to Fix

### Site 1: `crates/roko-agent/src/claude_cli_agent.rs:128`
In `ClaudeCliAgent::new()` constructor:
```rust
// BEFORE
dangerously_skip_permissions: true,

// AFTER
dangerously_skip_permissions: false,
```

### Site 2: `crates/roko-cli/src/runner/types.rs:1337`
In `RunConfig::from_roko_config()`:
```rust
// BEFORE
dangerously_skip_permissions: true,

// AFTER
dangerously_skip_permissions: roko_config.runner.as_ref()
    .and_then(|r| r.dangerously_skip_permissions)
    .unwrap_or(false),
```

### Site 3: `crates/roko-cli/src/runner/types.rs:1382`
In `impl Default for RunConfig`:
```rust
// BEFORE
dangerously_skip_permissions: true,

// AFTER
dangerously_skip_permissions: false,
```

### Site 4: `crates/roko-cli/src/agent_exec.rs:145`
```rust
// BEFORE
dangerously_skip_permissions: true,

// AFTER
dangerously_skip_permissions: false,
```

### Site 5: `crates/roko-cli/src/serve_runtime.rs:543`
```rust
// BEFORE
dangerously_skip_permissions: true,

// AFTER
dangerously_skip_permissions: false,
```

### Site 6: `crates/roko-cli/src/commands/plan.rs:291`
```rust
// BEFORE
dangerously_skip_permissions: true,

// AFTER — read from CLI flag (PE_03 adds the flag)
dangerously_skip_permissions: args.dangerously_skip_permissions,
```
If PE_03 hasn't run yet, use `false` for now.

### Site 7: `crates/roko-acp/src/runner.rs:1763`
Raw `Command::arg("--dangerously-skip-permissions")`. Make conditional:
```rust
// BEFORE
let output = Command::new(CLAUDE_CLI_BIN)
    .arg("--print")
    .arg("--dangerously-skip-permissions")
    .arg(prompt)

// AFTER
let mut cmd = Command::new(CLAUDE_CLI_BIN);
cmd.arg("--print");
if skip_permissions {
    cmd.arg("--dangerously-skip-permissions");
}
cmd.arg(prompt);
```
Thread `skip_permissions: bool` from the caller.
