# PE_06: Make safety required (not Option) in ToolDispatcher

## Task
Change `ToolDispatcher::new()` to require a `SafetyLayer` parameter instead of `safety: None`.

## Runner Context
Runner PE, batch 6 of 8. Depends on PE_05.

## Problem
`ToolDispatcher::new()` initializes with `safety: None` by default. Any code that constructs without `.with_safety()` gets zero safety enforcement silently.

## Current Code

**ToolDispatcher::new()** — `crates/roko-agent/src/dispatcher/mod.rs`:
Search for `pub fn new(` in the dispatcher. Expected around line 108:
```rust
pub fn new(registry: ToolRegistry, resolver: ToolResolver) -> Self {
    Self {
        registry,
        resolver,
        safety: None,  // UNSAFE
    }
}
```

**`.with_safety()` builder** — around line 127:
```rust
pub fn with_safety(mut self, layer: SafetyLayer) -> Self {
    self.safety = Some(layer);
    self
}
```

## Exact Changes

### Step 1: Change constructor signature

```rust
// BEFORE
pub fn new(registry: ToolRegistry, resolver: ToolResolver) -> Self {
    Self { registry, resolver, safety: None }
}

// AFTER
pub fn new(registry: ToolRegistry, resolver: ToolResolver, safety: SafetyLayer) -> Self {
    Self { registry, resolver, safety: Some(safety) }
}
```

### Step 2: Update all call sites

Search for `ToolDispatcher::new(` across the codebase. At each site:

```rust
// BEFORE
let dispatcher = ToolDispatcher::new(registry, resolver);

// AFTER — use RestrictedFallback contract
let contract = AgentContract::load_for_role_with_mode(role, ContractLoadMode::RestrictedFallback)
    .expect("RestrictedFallback never returns Err");
let safety = SafetyLayer::new(contract);
let dispatcher = ToolDispatcher::new(registry, resolver, safety);
```

### Step 3: Keep `.with_safety()` for override

Keep the builder method for cases where safety needs to be replaced after construction:
```rust
pub fn with_safety(mut self, layer: SafetyLayer) -> Self {
    self.safety = Some(layer);
    self
}
```

### Step 4: Handle test code

In test code, pass a permissive safety layer:
```rust
#[cfg(test)]
let safety = SafetyLayer::new(AgentContract::permissive("test"));
let dispatcher = ToolDispatcher::new(registry, resolver, safety);
```
