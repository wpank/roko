# PF_02: Add #[deprecated] warning to orchestrate.rs public API

## Task
Mark orchestrate.rs public entry points with `#[deprecated]` so anyone enabling the feature gets compiler warnings.

## Runner Context
Runner PF, batch 2 of 3. Depends on PF_01.

## Problem
FZ-2 anti-pattern: "Compile warning on opt-in." If someone enables `legacy-orchestrate`, they should see deprecation warnings discouraging new dependencies.

## Current Code

**orchestrate.rs** — 22,096 lines. Not actually gated by `#[cfg(feature = "legacy-orchestrate")]` at the module level (the research found no `#[cfg]` in orchestrate.rs itself — it's gated at import sites in lib.rs, run.rs, unified.rs).

**lib.rs imports** — search for `mod orchestrate` or `use orchestrate`:
- `crates/roko-cli/src/lib.rs:62,83,134` — `#[cfg(feature = "legacy-orchestrate")]` around imports

**Key public entry points** in orchestrate.rs to deprecate:
- `PlanRunner` struct (the main executor)
- `run_orchestration` or equivalent entry function
- Any types re-exported from lib.rs

## Exact Changes

### Step 1: Find public entry points

In orchestrate.rs, search for `pub struct PlanRunner` and `pub fn` at the top level. Expected:
- `pub struct PlanRunner` — main executor
- `pub async fn run_orchestration(...)` or similar entry point
- Any `pub struct` or `pub enum` that lib.rs re-exports

### Step 2: Add #[deprecated] to each entry point

```rust
#[deprecated(since = "0.2.0", note = "Use v2 runner (crate::runner::run) instead")]
pub struct PlanRunner { ... }

#[deprecated(since = "0.2.0", note = "Use v2 runner (crate::runner::run) instead")]
pub async fn run_orchestration(...) { ... }
```

### Step 3: Allow deprecated within the module

At the top of orchestrate.rs:
```rust
#![allow(deprecated)]  // Internal usage within deprecated module is fine
```

Or if it's not a file-level module:
```rust
#[allow(deprecated)]
mod orchestrate { ... }
```

### Step 4: Verify lib.rs re-exports are gated

In lib.rs, verify that all `use orchestrate::` lines have `#[cfg(feature = "legacy-orchestrate")]`:
```rust
#[cfg(feature = "legacy-orchestrate")]
pub mod orchestrate;
```

## Write Scope
- `crates/roko-cli/src/orchestrate.rs` (add #[deprecated] + #![allow(deprecated)])

```bash
cargo build -p roko-cli 2>&1 | head -30

```
## Acceptance Criteria
- `cargo build --features legacy-orchestrate` produces deprecation warnings at call sites
- Warnings name the replacement (v2 runner)
- Internal orchestrate.rs code compiles without warnings

## Do NOT
- Change any behavior in orchestrate.rs
- Refactor or remove code
