# PL_02: Add API provider fallback for ACP pipeline phases

## Task
Allow ACP pipeline phases to use API providers (not just Claude CLI) when the CLI is unavailable.

## Runner Context
Runner PL, batch 2 of 4. Depends on PL_01.

## Problem
ACP-2 anti-pattern: "Single dispatch path." `run_claude_cli()` at runner.rs:1752-1785 hardcodes the Claude CLI subprocess. Users without the Claude CLI installed cannot use ACP workflows. The single-agent path already supports API providers via `ModelCallService`, but the pipeline phases don't.

```rust

    let output = Command::new("claude")
        .arg("--print")
        // ...
        .output().await?;
}
```

**ModelCallService** — `model_call_service.rs:49-80`:

## Exact Changes

### Step 1: Add has_claude_cli() check

```rust
fn has_claude_cli() -> bool {
    std::process::Command::new("claude")
        .arg("--version")
        .stdout(std::process::Stdio::null())
        .stderr(std::process::Stdio::null())
        .status()
        .map(|s| s.success())
        .unwrap_or(false)
}
```

Cache the result at pipeline startup (don't check per-phase):


### Step 2: Add run_via_model_service()

```rust
async fn run_via_model_service(
    prompt: &str,
    model_service: &ModelCallService,
    model: Option<&str>,
) -> Result<String> {
    let request = ModelCallRequest {
        system_prompt: None,
        user_content: prompt.to_string(),
        model: model.map(|m| m.to_string()),
        caller_id: "acp_pipeline".into(),
        // ...
    };
    let response = model_service.call(request).await?;
    Ok(response.content)
}
```

### Step 3: Route in run_agent_phase

```rust
async fn run_agent_phase(
    &self,
    phase: &Phase,
    workdir: &Path,
    cancel_token: &CancelToken,
    mcp_config_path: Option<&Path>,
    model_service: Option<&ModelCallService>,
    cli_available: bool,
) -> Result<String> {
    if cli_available {
        run_claude_cli(&phase.prompt, workdir, cancel_token, mcp_config_path).await
    } else if let Some(service) = model_service {
        run_via_model_service(&phase.prompt, service, phase.model.as_deref()).await
    } else {
        Err(anyhow!("neither Claude CLI nor API provider available for ACP dispatch"))
    }
}
```
