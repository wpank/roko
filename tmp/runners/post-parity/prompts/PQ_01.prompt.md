# PQ_01: Wire HDC similarity into code_context_for_task prompt assembly

## Task
Enable the HDC arm in `code_context_for_task`'s `SearchStrategy::Hybrid` so code context benefits from semantic similarity.

## Runner Context
Runner PQ, batch 1 of 4. No dependencies.

## Problem
CI-1 anti-pattern: "Feature arm always None." `code_context_for_task` (prompt_helpers.rs:206-239) constructs a `SearchStrategy::Hybrid` with `hdc: None` (line 237). The `WorkspaceIndex` already computes HDC fingerprints and has `hdc_search` (workspace.rs:558). The RRF merge in `search()` (workspace.rs:618-632) handles combining keyword + HDC results. The only missing step: constructing an `HdcQuery` from the task description.

## Exact Changes

### Step 1: Generate HDC fingerprint from task description

In `prompt_helpers.rs:206-239`, before constructing `SearchStrategy::Hybrid`:

```rust
use roko_index::hdc::fingerprint_text;

// Generate HDC vector from task description
let hdc_query = match fingerprint_text(task_description.as_bytes()) {
    Ok(fingerprint) => Some(HdcQuery {
        anchor: fingerprint,
        min_similarity: 0.3,
        max_results: 20,
    }),
    Err(err) => {
        debug!(%err, "HDC fingerprint generation failed — using keyword only");
        None
    }
};
```

### Step 2: Pass HdcQuery into the Hybrid strategy

```rust
let strategy = SearchStrategy::Hybrid {
    keyword: Some(KeywordQuery {
        text: query.clone(),
        scope: SearchScope::All,
        case_sensitive: false,
        whole_word: false,
    }),
    structural: None,
    hdc: hdc_query,  // ← was None, now populated
};
```

### Step 3: Verify fingerprint_text exists

If `fingerprint_text` doesn't exist (maybe it's `fingerprint_file` or `fingerprint_symbol`), use the appropriate variant:

```rust
// If fingerprint_file is the available API:
let hdc_query = match roko_index::hdc::fingerprint_file(task_description.as_bytes(), "task") {
    Ok(fp) => Some(HdcQuery { anchor: fp, min_similarity: 0.3, max_results: 20 }),
    Err(_) => None,
};
```

### Step 4: No other changes needed

The RRF merge in `WorkspaceIndex::search()` automatically fuses HDC scores with keyword scores. No changes to the merge logic needed.
