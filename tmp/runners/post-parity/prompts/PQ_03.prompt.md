# PQ_03: Wire personalized PageRank for task-specific code ranking

## Task
Use `personalized_pagerank` seeded on task-relevant symbols to boost structurally connected code context results.

## Runner Context
Runner PQ, batch 3 of 4. Depends on PQ_01.

## Problem
CI-3 anti-pattern: "Global ranking for task-specific queries." `WorkspaceIndex` uses global `pagerank(graph, 30, 0.85)` at construction (workspace.rs:1018), producing uniform scores. `personalized_pagerank` (graph.rs:709-744) is exported but never called. For a task touching `CascadeRouter`, symbols connected to it should rank higher.

## Exact Changes

### Step 1: Extract seed nodes from initial search results

In `code_context_for_task` (prompt_helpers.rs), after the initial Hybrid search:

```rust
let initial_results = index.search(&strategy, 20)?;

// Extract top-5 symbol IDs as PPR seeds
let seed_ids: Vec<SymbolId> = initial_results.iter()
    .take(5)
    .filter_map(|r| r.symbol_id.clone())
    .collect();
```

### Step 2: Compute personalized PageRank

```rust
let ppr_scores = if !seed_ids.is_empty() {
    if let Some(graph) = index.graph() {
        roko_index::graph::personalized_pagerank(graph, &seed_ids, 0.85, 15)
    } else {
        HashMap::new()
    }
} else {
    HashMap::new()
};
```

### Step 3: Re-rank results using PPR scores

```rust
// Apply PPR as multiplicative boost
for result in &mut initial_results {
    if let Some(sym_id) = &result.symbol_id {
        if let Some(&ppr) = ppr_scores.get(sym_id) {
            result.score *= 1.0 + ppr * 2.0;  // up to ~2x boost for structurally related
        }
    }
}
initial_results.sort_by(|a, b| b.score.partial_cmp(&a.score).unwrap_or(std::cmp::Ordering::Equal));
```

Or add a combined method:
```rust
pub fn rerank_with_ppr(&self, seeds: &[SymbolId], results: &mut Vec<SearchResult>) {
    let ppr = personalized_pagerank(&self.graph, seeds, 0.85, 15);
    for r in results.iter_mut() {
        if let Some(id) = &r.symbol_id {
            if let Some(&score) = ppr.get(id) {
                r.score *= 1.0 + score * 2.0;
            }
        }
    }
    results.sort_by(|a, b| b.score.partial_cmp(&a.score).unwrap_or(std::cmp::Ordering::Equal));
}
```
