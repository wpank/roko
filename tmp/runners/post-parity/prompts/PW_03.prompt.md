# PW_03: Wire SSE /api/bench/events for live bench run streaming

## Task
Implement the SSE endpoint that streams bench run progress events to the demo app.

## Runner Context
Runner PW, batch 3 of 6. Depends on PW_02.

## Problem
`demo/demo-app/src/hooks/useBenchSSE.ts:28`:
```ts
const es = new EventSource(`${SERVE_URL}/api/bench/events${params}`);
```
Expects event types: `BenchTaskStarted`, `BenchTaskCompleted`, `BenchProgress`, `BenchRunCompleted`.

No SSE endpoint exists at `/api/bench/events`. The demo `CostRace.tsx` also subscribes when `live=true`.

## Exact Changes

### Step 1: Add SSE handler

```rust
async fn bench_events(
    State(state): State<Arc<AppState>>,
    Query(params): Query<BenchEventParams>,
) -> Sse<impl Stream<Item = Result<Event, Infallible>>> {
    let mut rx = state.bench_event_tx.subscribe();
    let bench_id = params.bench_id.clone();

    let stream = async_stream::stream! {
        while let Ok(event) = rx.recv().await {
            // Filter by bench_id if specified
            if let Some(ref id) = bench_id {
                if event.bench_id() != id.as_str() { continue; }
            }
            let event_type = event.event_type();
            let data = serde_json::to_string(&event).unwrap_or_default();
            yield Ok(Event::default().event(event_type).data(data));
        }
    };
    Sse::new(stream)
}
```

### Step 2: Add BenchEvent enum if it doesn't exist

```rust
#[derive(Serialize, Clone)]
#[serde(tag = "type")]
pub enum BenchEvent {
    BenchTaskStarted { bench_id: String, task_name: String },
    BenchTaskCompleted { bench_id: String, result: BenchTaskResult },
    BenchProgress { bench_id: String, completed: u32, total: u32, cost_so_far: f64 },
    BenchRunCompleted { bench_id: String, summary: BenchRunSummary },
}
```

### Step 3: Add broadcast channel to AppState

Add `bench_event_tx: broadcast::Sender<BenchEvent>` to `AppState`. Initialize with `broadcast::channel(256)`.

### Step 4: Register route

```rust
.route("/bench/events", get(bench_events))
```
