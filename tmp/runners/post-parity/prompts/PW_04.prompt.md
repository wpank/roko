# PW_04: Wire GET /api/dashboard summary and GET /api/gates/history endpoints

## Task
Implement the dashboard summary and gate history endpoints.

## Runner Context
Runner PW, batch 4 of 6. No dependencies.

## Problem
`demo/demo-app/src/hooks/useDashboard.ts:19` calls `GET /api/dashboard`.
`demo/demo-app/src/pages/dashboard/ChainView.tsx:56` calls `GET /api/gates/history?limit=20`.

Expected shapes:
```ts
// /api/dashboard
{ total_cost?: number; cache_hit_rate?: number; routing_distribution?: Record<string,number>; gate_pass_rate?: number; }

// /api/gates/history
GateRun[] // task_name, gate verdicts, timing
```

## Exact Changes

### Step 1: Add dashboard summary handler

Read from `LearningRuntime` or learning files on disk:

```rust
async fn dashboard_summary(
    State(state): State<Arc<AppState>>,
) -> Result<impl IntoResponse, ApiError> {
    let learn_root = state.workdir.join(".roko/learn");
    // Read costs from costs.jsonl, routing from cascade-router.json, gates from gate-thresholds.json
    let total_cost = read_total_cost(&learn_root);
    let routing = read_routing_distribution(&learn_root);
    let gate_pass_rate = read_gate_pass_rate(&learn_root);
    Ok(Json(json!({
        "total_cost": total_cost,
        "cache_hit_rate": null,
        "routing_distribution": routing,
        "gate_pass_rate": gate_pass_rate,
    })))
}
```

### Step 2: Add gate history handler

```rust
async fn gate_history(
    State(state): State<Arc<AppState>>,
    Query(params): Query<LimitParams>,
) -> Result<impl IntoResponse, ApiError> {
    let limit = params.limit.unwrap_or(20);
    // Read from .roko/episodes.jsonl, extract gate verdicts per episode
    let episodes = read_recent_episodes(&state.workdir, limit)?;
    let gate_runs: Vec<_> = episodes.iter().map(|ep| {
        json!({
            "task_name": ep.task_id,
            "gates": ep.gate_results,
            "timestamp": ep.timestamp,
        })
    }).collect();
    Ok(Json(gate_runs))
}
```

### Step 3: Register routes

```rust
.route("/dashboard", get(dashboard_summary))
.route("/gates/history", get(gate_history))
```
