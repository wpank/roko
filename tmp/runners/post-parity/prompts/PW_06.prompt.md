# PW_06: Wire GET /api/learn/adaptive-thresholds and GET /api/agents/topology

## Task
Implement the adaptive thresholds and agent topology endpoints the demo dashboard expects.

## Runner Context
Runner PW, batch 6 of 6. No dependencies.

## Problem
`CostDashboard.tsx:143` calls `GET /api/learn/adaptive-thresholds`.
`AgentFleet.tsx:439` calls `GET /api/agents/topology`.

Expected shapes:
```ts
// /api/learn/adaptive-thresholds
{ thresholds: Record<string, { rung: string; current: number; target?: number; }> }

// /api/agents/topology
{ nodes: { agent_id: string; role: string; endpoints: string[]; }[]; edges: { from: string; to: string; }[]; }
```

## Exact Changes

### Step 1: Adaptive thresholds handler

Read from `.roko/learn/gate-thresholds.json`:

```rust
async fn adaptive_thresholds(
    State(state): State<Arc<AppState>>,
) -> Result<impl IntoResponse, ApiError> {
    let path = state.workdir.join(".roko/learn/gate-thresholds.json");
    let thresholds = match std::fs::read_to_string(&path) {
        Ok(content) => serde_json::from_str::<Value>(&content)
            .unwrap_or(json!({})),
        Err(_) => json!({}),
    };
    Ok(Json(json!({ "thresholds": thresholds })))
}
```

### Step 2: Agent topology handler

Build from registered agents in AppState:

```rust
async fn agent_topology(
    State(state): State<Arc<AppState>>,
) -> Result<impl IntoResponse, ApiError> {
    let agents = state.agents.read().await;
    let nodes: Vec<_> = agents.values().map(|a| json!({
        "agent_id": a.id,
        "role": a.role,
        "endpoints": a.endpoints_list(),
    })).collect();
    // Edges from agent communication patterns (if tracked)
    let edges: Vec<Value> = vec![];
    Ok(Json(json!({ "nodes": nodes, "edges": edges })))
}
```

### Step 3: Register routes

```rust
.route("/learn/adaptive-thresholds", get(adaptive_thresholds))
.route("/agents/topology", get(agent_topology))
```
