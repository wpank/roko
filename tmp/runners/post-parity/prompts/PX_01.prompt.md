# PX_01: Wire DaimonPolicy as real AffectPolicy implementor

## Task
Implement `AffectPolicy` for the daimon-backed affect engine so production code uses real affect modulation instead of `NoOpAffectPolicy`.

## Runner Context
Runner PX (Critical Fixes), batch 1 of 4. No dependencies.

## Problem
The `AffectPolicy` trait exists at `foundation.rs:449-472` with 6 methods, but only `NoOpAffectPolicy` (foundation.rs:474-509) implements it. `DaimonPolicy` at `affect.rs:67-89` is a data struct that does NOT implement the trait. In production, `EffectServices.affect_policy` (effect_driver.rs:49) is always `None` or `NoOpAffectPolicy`. The entire affect system — somatic markers, PAD state, dispatch modulation — is built but never activated.

### Step 1: Create DaimonAffectPolicy wrapper

In `crates/roko-daimon/src/policy.rs` (new file):

```rust
use roko_core::foundation::{AffectContext, AffectPolicy, DispatchModulation};
use roko_core::BehavioralState;
use crate::DaimonState;
use std::path::PathBuf;

pub struct DaimonAffectPolicy {
    state: DaimonState,
    persist_path: PathBuf,
}

impl DaimonAffectPolicy {
    pub fn new(state: DaimonState, persist_path: PathBuf) -> Self {
        Self { state, persist_path }
    }

    pub fn load_or_default(path: &std::path::Path) -> Self {
        let state = DaimonState::load(path).unwrap_or_default();
        Self { state, persist_path: path.to_path_buf() }
    }
}
```

### Step 2: Implement AffectPolicy for DaimonAffectPolicy

```rust
impl AffectPolicy for DaimonAffectPolicy {
    fn pre_dispatch(&self, task_id: &str, role: &str) -> AffectContext {
        // Use DaimonState's PAD model to create context
        let pad = self.state.pad_state();
        AffectContext {
            arousal: pad.arousal,
            pleasure: pad.pleasure,
            dominance: pad.dominance,
            behavioral_state: self.state.behavioral_state(),
        }
    }

    fn on_task_outcome(&mut self, _task_id: &str, succeeded: bool, tokens_used: u64, cost_usd: f64) {
        if succeeded {
            self.state.record_success(tokens_used, cost_usd);
        } else {
            self.state.record_failure(tokens_used, cost_usd);
        }
    }

    fn on_gate_result(&mut self, _gate_name: &str, passed: bool, _rung: u8, confidence: f64) {
        self.state.record_gate_outcome(passed, confidence);
    }

    fn modulate_dispatch(&self, role: &str, params: &mut DispatchModulation) {
        self.state.modulate_dispatch(role, params);
    }

    fn behavioral_state(&self) -> BehavioralState {
        self.state.behavioral_state()
    }

    async fn persist(&self) -> Result<()> {
        self.state.save(&self.persist_path)
    }
}
```
