# PY_03: Route research commands through WorkflowEngine dispatch

## Task
Replace 5 `run_agent_capture_silent` calls in research commands with `run_with_workflow_engine` so research work gets episodes, gate checks, and learning feedback.

## Runner Context
Runner PY (Workflow Routing), batch 3 of 4. Depends on PY_01.

## Problem
`commands/research.rs` calls `run_agent_capture_silent` directly at 5 sites, bypassing WorkflowEngine. This means research agent runs get no episode logging, no gate validation, no efficiency tracking, and no cascade router feedback.
