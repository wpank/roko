# PY_04: Route plan generate/regenerate through WorkflowEngine dispatch

## Task
Replace 2 `run_agent_logged` calls in plan commands with `run_with_workflow_engine` so plan generation gets episodes, gate checks, and learning feedback.

## Runner Context
Runner PY (Workflow Routing), batch 4 of 4. Depends on PY_01.

## Problem
`commands/plan.rs` calls `run_agent_logged` directly at 2 sites for `plan generate` and `plan regenerate`. These bypass WorkflowEngine — no gate validation, no cascade router feedback, no efficiency tracking.

**AFTER**:

    &task_prompt,
    &workdir,

    anyhow::bail!("plan generation failed");
}
report.exit_code.unwrap_or(0)
