# RF_05: Lazy event serialization -- defer JSON encoding

## Task
Make `JsonlLogger` defer JSON serialization until the write moment, avoiding wasted encoding for events that may never be flushed.

## Runner Context
Runner RF, batch 5 of 11. No dependencies.

## Problem
`JsonlLogger::write_event()` serializes every event to JSON immediately via `serde_json::to_string()`. If events are produced faster than they can be written (or if the logger is used in a buffered mode), serialization is wasted work. Deferring encoding until the actual write moment saves CPU on the hot path.

## Exact Changes

### Step 1: Add batch buffer option

In `crates/roko-runtime/src/jsonl_logger.rs`, add a batch buffer:

```rust
pub struct JsonlLogger {
    path: PathBuf,
    seq: AtomicU64,
    writer: Mutex<Option<std::io::BufWriter<std::fs::File>>>,
    /// Optional event buffer for batched writes.
    buffer: Mutex<Vec<RuntimeEvent>>,
    /// Maximum buffer size before auto-flush.
    buffer_capacity: usize,
}
```

### Step 2: Update constructor

```rust
impl JsonlLogger {
    pub fn new(path: PathBuf) -> Self {
        Self {
            path,
            seq: AtomicU64::new(0),
            writer: Mutex::new(None),
            buffer: Mutex::new(Vec::new()),
            buffer_capacity: 0, // 0 = no buffering (immediate write, backward compat)
        }

    /// Create a buffered logger that defers serialization until flush.
    pub fn buffered(path: PathBuf, capacity: usize) -> Self {
        Self {
            path,
            seq: AtomicU64::new(0),
            writer: Mutex::new(None),
            buffer: Mutex::new(Vec::with_capacity(capacity)),
            buffer_capacity: capacity,
        }
    }
```

### Step 3: Add buffered write path

```rust
    fn write_event_buffered(&self, event: &RuntimeEvent) -> std::io::Result<()> {
        if self.buffer_capacity == 0 {
            // Unbuffered: serialize and write immediately (existing behavior)
            return self.write_event_immediate(event);
        }

        let mut buf = self.buffer.lock().unwrap_or_else(|e| e.into_inner());
        buf.push(event.clone());

        if buf.len() >= self.buffer_capacity {
            let events: Vec<_> = buf.drain(..).collect();
            drop(buf); // Release lock before I/O
            self.flush_events(&events)?;
        }

        Ok(())
    }

    fn write_event_immediate(&self, event: &RuntimeEvent) -> std::io::Result<()> {
        // Existing write_event logic
        self.ensure_writer()?;
        let envelope = RuntimeEventEnvelope::new(
            event.run_id(),
            self.seq.fetch_add(1, Ordering::Relaxed),
            "jsonl_logger",
            event.clone(),
        );
        let json = serde_json::to_string(&envelope)
            .map_err(|err| std::io::Error::new(std::io::ErrorKind::InvalidData, err))?;
        let mut writer = self.writer.lock().unwrap_or_else(std::sync::PoisonError::into_inner);
        if let Some(ref mut w) = *writer {
            writeln!(w, "{json}")?;
            w.flush()?;
        }
        Ok(())
    }

    fn flush_events(&self, events: &[RuntimeEvent]) -> std::io::Result<()> {
        self.ensure_writer()?;
        let mut writer = self.writer.lock().unwrap_or_else(std::sync::PoisonError::into_inner);
        if let Some(ref mut w) = *writer {
            for event in events {
                let envelope = RuntimeEventEnvelope::new(
                    event.run_id(),
                    self.seq.fetch_add(1, Ordering::Relaxed),
                    "jsonl_logger",
                    event.clone(),
                );
                let json = serde_json::to_string(&envelope)
                    .map_err(|err| std::io::Error::new(std::io::ErrorKind::InvalidData, err))?;
                writeln!(w, "{json}")?;
            }
            w.flush()?;
        }
        Ok(())
    }

    /// Flush any buffered events to disk.
    pub fn flush(&self) -> std::io::Result<()> {
        let mut buf = self.buffer.lock().unwrap_or_else(|e| e.into_inner());
        if buf.is_empty() {
            return Ok(());
        }
        let events: Vec<_> = buf.drain(..).collect();
        drop(buf);
        self.flush_events(&events)
    }
```

### Step 4: Update EventConsumer impl

```rust

    fn consume(&self, event: &RuntimeEvent) {
        let _ = self.write_event_buffered(event);
    }
```
