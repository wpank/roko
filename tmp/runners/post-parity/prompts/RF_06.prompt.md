# RF_06: Define GateMode enum (Express/Full/Quick)

## Task
Define a `GateMode` enum in `pipeline_state.rs` that controls how many gates are run: Express (compile only), Quick (compile + test), Full (all gates).

## Runner Context
Runner RF, batch 6 of 11. No dependencies.

## Problem
The gate pipeline always runs all configured gates. For rapid iteration, users need lighter modes that skip expensive gates (clippy, fmt, LLM judge) while keeping the compile check. Express mode gives the fastest feedback; Full mode gives the most thorough.

### Step 1: Define GateMode enum

In `crates/roko-runtime/src/pipeline_state.rs`, add:

```rust
/// Controls which gates are executed during verification.
#[derive(Clone, Copy, Debug, PartialEq, Eq, Hash, Serialize, Deserialize)]
pub enum GateMode {
    /// Compile only. Fastest feedback for rapid iteration.
    Express,
    /// Compile + test. Reasonable coverage without lint overhead.
    Quick,
    /// All configured gates (compile, clippy, test, fmt, diff, custom, judge).
    Full,
}

impl GateMode {
    /// Filter a list of gate names to only those enabled by this mode.
    pub fn filter_gates<'a>(&self, gates: &'a [String]) -> Vec<&'a str> {
        match self {
            Self::Express => gates
                .iter()
                .filter(|g| g.starts_with("compile"))
                .map(|g| g.as_str())
                .collect(),
            Self::Quick => gates
                .iter()
                .filter(|g| g.starts_with("compile") || g.starts_with("test"))
                .map(|g| g.as_str())
                .collect(),
            Self::Full => gates.iter().map(|g| g.as_str()).collect(),
        }
    }

    /// Human-readable label for this mode.
    pub const fn label(self) -> &'static str {
        match self {
            Self::Express => "express",
            Self::Quick => "quick",
            Self::Full => "full",
        }
    }

    /// Parse a mode from a string.
    pub fn from_str_loose(s: &str) -> Self {
        match s.to_lowercase().as_str() {
            "express" | "fast" | "compile-only" => Self::Express,
            "quick" | "standard" => Self::Quick,
            "full" | "all" | "thorough" => Self::Full,
            _ => Self::Full, // default to full
        }
    }

    /// Number of gate types enabled by this mode.
    pub const fn gate_count(self) -> usize {
        match self {
            Self::Express => 1,
            Self::Quick => 2,
            Self::Full => 7,
        }
    }
}

impl Default for GateMode {
    fn default() -> Self {
        Self::Full
    }
}

impl std::fmt::Display for GateMode {
    fn fmt(&self, f: &mut std::fmt::Formatter<'_>) -> std::fmt::Result {
        f.write_str(self.label())
    }
}

impl std::str::FromStr for GateMode {
    type Err = String;
    fn from_str(s: &str) -> Result<Self, Self::Err> {
        Ok(Self::from_str_loose(s))
    }
}
```

### Step 3: Add tests

```rust
#[cfg(test)]
mod gate_mode_tests {
    use super::*;

    #[test]
    fn express_filters_to_compile() {
        let gates = vec!["compile".into(), "clippy".into(), "test".into()];
        let filtered = GateMode::Express.filter_gates(&gates);
        assert_eq!(filtered, vec!["compile"]);
    }

    #[test]
    fn quick_filters_to_compile_and_test() {
        let gates = vec!["compile".into(), "clippy".into(), "test".into(), "fmt".into()];
        let filtered = GateMode::Quick.filter_gates(&gates);
        assert_eq!(filtered, vec!["compile", "test"]);
    }

    #[test]
    fn full_keeps_all() {
        let gates = vec!["compile".into(), "clippy".into(), "test".into()];
        let filtered = GateMode::Full.filter_gates(&gates);
        assert_eq!(filtered.len(), 3);
    }
}
```
