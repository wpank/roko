# RP_03: Add cumulative context section generator

## Task
Add a cumulative context section to `PromptAssemblyService` that includes summaries of previously completed tasks in the same run, giving agents awareness of what has already been done.

## Runner Context
Runner RP_A (Runner Patterns), batch 3 of 5. No dependencies.

## Problem
When a multi-task plan runs, each agent starts with no knowledge of what prior tasks accomplished. Agents may redo work, create conflicts, or miss dependencies. A cumulative context section summarizes prior task outputs.

### Step 2: Add methods to manage cumulative context

```rust
impl PromptAssemblyService {
    /// Record a completed task for cumulative context.
    pub fn record_completed_task(&self, summary: TaskSummary) {
        let mut ctx = self.cumulative_context.lock().unwrap();
        ctx.push(summary);
    }

    /// Clear cumulative context (e.g., at the start of a new run).
    pub fn clear_cumulative_context(&self) {
        let mut ctx = self.cumulative_context.lock().unwrap();
        ctx.clear();
    }

    /// Generate the cumulative context section for the system prompt.
    fn cumulative_context_section(&self) -> Option<String> {
        let ctx = self.cumulative_context.lock().unwrap();
        if ctx.is_empty() {
            return None;
        }

        let mut section = String::from("## Previously Completed Tasks\n\n");
        section.push_str("The following tasks have already been completed in this run. ");
        section.push_str("Do NOT redo this work. Build on it.\n\n");

        for (i, task) in ctx.iter().enumerate() {
            section.push_str(&format!(
                "### Task {} ({})\n{}\n",
                i + 1,
                task.role,
                task.summary
            ));
            if !task.files_changed.is_empty() {
                section.push_str("Files changed: ");
                section.push_str(&task.files_changed.join(", "));
                section.push('\n');
            }
            section.push('\n');
        }

        Some(section)
    }
}
```

### Step 3: Inject into prompt assembly

In the `assemble()` method, after building the system prompt sections, add the cumulative context:

```rust
// In assemble(), after other sections:
if let Some(cumulative) = self.cumulative_context_section() {
    builder = builder.with_additional_context("cumulative", &cumulative);
}
```
