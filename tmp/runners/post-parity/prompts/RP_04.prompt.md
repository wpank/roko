# RP_04: Add AntiPatternChecker rules and dispatch

## Task
Create an `AntiPatternChecker` gate that detects common code anti-patterns in agent output (duplicate implementations, unused imports, oversized functions) and reports them as gate verdicts.

## Runner Context
Runner RP_A (Runner Patterns), batch 4 of 5. No dependencies.

## Problem
Agents frequently produce code with anti-patterns: duplicate implementations of existing functions, excessively long functions, unused imports, commented-out code blocks. These should be caught by a gate before merging.

## Exact Changes

### Step 1: Create `crates/roko-gate/src/anti_pattern_checker.rs`

```rust
//! Anti-pattern detection gate (RP_04).
//!
//! Scans agent-modified files for common code anti-patterns and
//! reports findings as gate verdicts.

use std::path::Path;

use roko_core::{Body, Context, Engram, Kind, Result, Verdict, Verify};

/// Anti-pattern rule.
#[derive(Debug, Clone)]
pub struct AntiPatternRule {
    pub name: String,
    pub description: String,
    pub severity: AntiPatternSeverity,
    pub checker: AntiPatternCheck,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum AntiPatternSeverity {
    /// Informational -- do not fail the gate.
    Info,
    /// Warning -- fail if count exceeds threshold.
    Warning,
    /// Error -- always fail.
    Error,
}

#[derive(Debug, Clone)]
pub enum AntiPatternCheck {
    /// Function body exceeds this many lines.
    MaxFunctionLength(usize),
    /// File exceeds this many lines.
    MaxFileLength(usize),
    /// Regex pattern that should not appear.
    ForbiddenPattern(String),
    /// Duplicate of an existing function name.
    DuplicateFunction,
}

/// Result of checking one file.
#[derive(Debug, Clone)]
pub struct AntiPatternFinding {
    pub rule: String,
    pub file: String,
    pub line: Option<usize>,
    pub message: String,
    pub severity: AntiPatternSeverity,
}

/// The anti-pattern checker gate.
pub struct AntiPatternChecker {
    rules: Vec<AntiPatternRule>,
}

impl AntiPatternChecker {
    /// Create with default rules.
    pub fn with_defaults() -> Self {
        Self {
            rules: default_rules(),
        }
    }

    /// Check a single file's contents.
    pub fn check_file(&self, path: &str, content: &str) -> Vec<AntiPatternFinding> {
        let mut findings = Vec::new();

        for rule in &self.rules {
            match &rule.checker {
                AntiPatternCheck::MaxFunctionLength(max) => {
                    // Simple heuristic: count lines between fn..{ and closing }
                    let mut in_fn = false;
                    let mut fn_start = 0;
                    let mut fn_name = String::new();
                    let mut brace_depth = 0;

                    for (i, line) in content.lines().enumerate() {
                        let trimmed = line.trim();
                        if !in_fn && (trimmed.starts_with("pub fn ")
                            || trimmed.starts_with("fn ")
                            || trimmed.starts_with("pub async fn ")
                            || trimmed.starts_with("async fn "))
                        {
                            in_fn = true;
                            fn_start = i;
                            fn_name = trimmed.split('(').next().unwrap_or("").to_string();
                            brace_depth = 0;
                        }
                        if in_fn {
                            brace_depth += trimmed.matches('{').count();
                            brace_depth = brace_depth.saturating_sub(trimmed.matches('}').count());
                            if brace_depth == 0 && i > fn_start {
                                let length = i - fn_start;
                                if length > *max {
                                    findings.push(AntiPatternFinding {
                                        rule: rule.name.clone(),
                                        file: path.to_string(),
                                        line: Some(fn_start + 1),
                                        message: format!(
                                            "{fn_name}: {length} lines (max {max})"
                                        ),
                                        severity: rule.severity,
                                    });
                                }
                                in_fn = false;
                            }
                        }
                    }
                }
                AntiPatternCheck::MaxFileLength(max) => {
                    let lines = content.lines().count();
                    if lines > *max {
                        findings.push(AntiPatternFinding {
                            rule: rule.name.clone(),
                            file: path.to_string(),
                            line: None,
                            message: format!("{lines} lines (max {max})"),
                            severity: rule.severity,
                        });
                    }
                }
                AntiPatternCheck::ForbiddenPattern(pattern) => {
                    for (i, line) in content.lines().enumerate() {
                        if line.contains(pattern.as_str()) {
                            findings.push(AntiPatternFinding {
                                rule: rule.name.clone(),
                                file: path.to_string(),
                                line: Some(i + 1),
                                message: format!("forbidden pattern: {pattern}"),
                                severity: rule.severity,
                            });
                        }
                    }
                }
                AntiPatternCheck::DuplicateFunction => {
                    // Simple check: this is a placeholder for cross-file analysis
                }
            }
        }

        findings
    }
}

fn default_rules() -> Vec<AntiPatternRule> {
    vec![
        AntiPatternRule {
            name: "long_function".into(),
            description: "Function exceeds 100 lines".into(),
            severity: AntiPatternSeverity::Warning,
            checker: AntiPatternCheck::MaxFunctionLength(100),
        },
        AntiPatternRule {
            name: "long_file".into(),
            description: "File exceeds 1000 lines".into(),
            severity: AntiPatternSeverity::Warning,
            checker: AntiPatternCheck::MaxFileLength(1000),
        },
        AntiPatternRule {
            name: "todo_fixme".into(),
            description: "Contains TODO/FIXME without ticket".into(),
            severity: AntiPatternSeverity::Info,
            checker: AntiPatternCheck::ForbiddenPattern("TODO".into()),
        },
        AntiPatternRule {
            name: "unwrap".into(),
            description: "Contains .unwrap() call".into(),
            severity: AntiPatternSeverity::Info,
            checker: AntiPatternCheck::ForbiddenPattern(".unwrap()".into()),
        },
    ]
}
```
