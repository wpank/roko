# RP_05: Add auto-cherry-pick conveyor belt

## Task
Create a `CherryPickConveyor` that automatically cherry-picks completed, gate-passing commits from agent worktree branches onto the batch/main branch.

## Runner Context
Runner RP_A (Runner Patterns), batch 5 of 5. Depends on RP_01, RP_02.

## Problem
After an agent completes work in its isolated worktree and passes gates, the changes need to be integrated into the main branch. Currently this requires manual merge. An auto-cherry-pick conveyor automates this for independent changes.

## Exact Changes

### Step 1: Create `crates/roko-orchestrator/src/cherry_pick.rs`

```rust
//! Auto-cherry-pick conveyor belt (RP_05).
//!
//! Monitors completed worktree branches and automatically cherry-picks
//! their commits onto the target branch when gates pass.

use std::path::{Path, PathBuf};

use tokio::process::Command;
use tracing::{info, warn, error};

/// A cherry-pick candidate.
#[derive(Debug, Clone)]
pub struct CherryPickCandidate {
    /// Plan or task ID.
    pub id: String,
    /// Source branch with the changes.
    pub source_branch: String,
    /// Commit hash(es) to cherry-pick.
    pub commits: Vec<String>,
    /// Whether gates passed for this candidate.
    pub gates_passed: bool,
    /// Files changed (for conflict detection).
    pub files_changed: Vec<String>,
}

/// Result of a cherry-pick attempt.
#[derive(Debug)]
pub enum CherryPickResult {
    /// Successfully cherry-picked.
    Success { commits_applied: usize },
    /// Conflict detected -- needs manual resolution.
    Conflict { conflicting_files: Vec<String> },
    /// Cherry-pick failed for other reasons.
    Failed { error: String },
    /// Skipped (gates did not pass).
    Skipped { reason: String },
}

/// Conveyor that processes cherry-pick candidates.
pub struct CherryPickConveyor {
    repo_root: PathBuf,
    target_branch: String,
}

impl CherryPickConveyor {
    pub fn new(repo_root: PathBuf, target_branch: String) -> Self {
        Self {
            repo_root,
            target_branch,
        }
    }

    /// Attempt to cherry-pick a candidate onto the target branch.
    pub async fn cherry_pick(&self, candidate: &CherryPickCandidate) -> CherryPickResult {
        if !candidate.gates_passed {
            return CherryPickResult::Skipped {
                reason: "gates did not pass".to_string(),
            };
        }

        if candidate.commits.is_empty() {
            return CherryPickResult::Skipped {
                reason: "no commits to cherry-pick".to_string(),
            };
        }

        // Ensure we're on the target branch
        let checkout = Command::new("git")
            .args(["checkout", &self.target_branch])
            .current_dir(&self.repo_root)
            .output()
            .await;

        if let Err(e) = checkout {
            return CherryPickResult::Failed {
                error: format!("checkout {}: {e}", self.target_branch),
            };
        }

        let mut applied = 0;
        for commit in &candidate.commits {
            let result = Command::new("git")
                .args(["cherry-pick", "--no-commit", commit])
                .current_dir(&self.repo_root)
                .output()
                .await;

            match result {
                Ok(output) if output.status.success() => {
                    applied += 1;
                }
                Ok(output) => {
                    let stderr = String::from_utf8_lossy(&output.stderr);

                    // Abort the cherry-pick
                    let _ = Command::new("git")
                        .args(["cherry-pick", "--abort"])
                        .current_dir(&self.repo_root)
                        .output()
                        .await;

                    if stderr.contains("CONFLICT") {
                        return CherryPickResult::Conflict {
                            conflicting_files: parse_conflict_files(&stderr),
                        };
                    }

                    return CherryPickResult::Failed {
                        error: stderr.to_string(),
                    };
                }
                Err(e) => {
                    return CherryPickResult::Failed {
                        error: format!("cherry-pick {commit}: {e}"),
                    };
                }
            }
        }

        // Commit the cherry-picked changes
        let msg = format!(
            "cherry-pick({}): {} commit(s) from {}",
            candidate.id,
            applied,
            candidate.source_branch
        );

        let commit_result = Command::new("git")
            .args(["commit", "-m", &msg])
            .current_dir(&self.repo_root)
            .output()
            .await;

        match commit_result {
            Ok(output) if output.status.success() => {
                info!(
                    id = %candidate.id,
                    applied,
                    "cherry-pick conveyor: committed"
                );
                CherryPickResult::Success { commits_applied: applied }
            }
            Ok(output) => {
                let stderr = String::from_utf8_lossy(&output.stderr);
                CherryPickResult::Failed { error: stderr.to_string() }
            }
            Err(e) => {
                CherryPickResult::Failed { error: format!("commit: {e}") }
            }
        }
    }

    /// Get the list of commits on a branch that are not on the target.
    pub async fn commits_on_branch(&self, branch: &str) -> Result<Vec<String>, String> {
        let output = Command::new("git")
            .args(["log", "--format=%H", &format!("{}..{}", self.target_branch, branch)])
            .current_dir(&self.repo_root)
            .output()
            .await
            .map_err(|e| format!("git log: {e}"))?;

        Ok(String::from_utf8_lossy(&output.stdout)
            .lines()
            .filter(|l| !l.is_empty())
            .map(String::from)
            .collect())
    }
}

fn parse_conflict_files(stderr: &str) -> Vec<String> {
    stderr
        .lines()
        .filter(|l| l.contains("CONFLICT"))
        .filter_map(|l| l.split_whitespace().last())
        .map(String::from)
        .collect()
}
```
