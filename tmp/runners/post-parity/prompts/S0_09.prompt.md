# S0_09: Wire CascadeRouter to live callers

## Task
Load `CascadeRouter` from disk in `crates/roko-cli/src/model_selection.rs` and pass it to all callers of `resolve_effective_model()`.

## Runner Context
Post-parity Phase 0 (Stability), batch 9 of 25. No dependencies. Group S0C (Learning Wiring).

## Problem
The `CascadeRouter` is persisted at `.roko/learn/cascade-router.json` and the
`resolve_effective_model()` function accepts `Option<&CascadeRouter>` as a parameter.
However, most callers pass `None` because loading the router is not wired.

The `load_cascade_router()` helper from S0_07 (or equivalent) provides disk loading.
This batch wires it into the actual call sites.

## Exact Changes

### Step 1: Find all callers of resolve_effective_model

Search for `resolve_effective_model(` across the codebase. Each call site that passes
`None` for the `cascade_router` parameter needs to be updated.

Likely call sites:
- `crates/roko-cli/src/commands/util.rs` (run command entry)
- `crates/roko-cli/src/commands/plan.rs` (plan run entry)
- `crates/roko-cli/src/chat_session.rs` (chat session creation)
- `crates/roko-cli/src/run.rs` (workflow dispatch)

### Step 2: Load router at each call site

At each call site, before calling `resolve_effective_model()`, load the router:

```rust
let cascade_router = crate::model_selection::load_cascade_router(&workdir);
let selection = resolve_effective_model(
    cli_model,
    task_hint,
    role,
    cascade_router.as_ref(),
    &config,
)?;
```

If `load_cascade_router` does not exist yet (S0_07 not yet applied), inline the loading:

```rust
fn load_cascade_router(workdir: &Path) -> Option<roko_learn::cascade_router::CascadeRouter> {
    let path = workdir.join(".roko/learn/cascade-router.json");
    std::fs::read_to_string(&path)
        .ok()
        .and_then(|json| serde_json::from_str(&json).ok())
}
```

### Step 3: Add the import

At each file that calls `load_cascade_router`, ensure the import is present:
```rust
use crate::model_selection::load_cascade_router;
```

Or if defined locally, no import needed.

- Router loaded from `.roko/learn/cascade-router.json`
