# S0_13: Replace ANTHROPIC_API_KEY env read in episode_completion

## Task
Replace the direct `std::env::var("ANTHROPIC_API_KEY")` read in `crates/roko-neuro/src/episode_completion.rs` with a `ModelCaller` parameter fallback.

## Runner Context
Post-parity Phase 0 (Stability), batch 13 of 25. No dependencies. Group S0D (Env Var Elimination).

## Problem
In `crates/roko-neuro/src/episode_completion.rs:46-49`, when no `ModelCaller` is provided,
the code falls back to reading `ANTHROPIC_API_KEY` directly from the environment:

```rust
let Some(api_key) = std::env::var("ANTHROPIC_API_KEY")
    .ok()
    .map(|key| key.trim().to_owned())
    .filter(|key| !key.is_empty())
```

This bypasses the config system and creates a hard dependency on the env var being set.
The `model_caller` parameter (line 26) already provides the correct abstraction -- the
env fallback should be removed or replaced with a config-sourced key.

## Exact Changes

### Step 1: Remove the env var fallback

In `crates/roko-neuro/src/episode_completion.rs`, find the `distill_episode()` function
at line 35. Replace the `model_caller` branching:

Current code (lines 40-54 approximately):
```rust
let distiller = if let Some(model_caller) = model_caller {
    Distiller::with_backend(Arc::new(GatewayDistillationBackend::new(
        model_caller,
        GATEWAY_DISTILLATION_MODEL,
    )))
} else {
    let Some(api_key) = std::env::var("ANTHROPIC_API_KEY")
        .ok()
        .map(|key| key.trim().to_owned())
        .filter(|key| !key.is_empty())
    else {
        tracing::debug!("no model_caller and no ANTHROPIC_API_KEY; skipping distillation");
        return Ok(());
    };
    Distiller::with_backend(Arc::new(HttpDistillationBackend::new(api_key)))
};
```

Replace with:
```rust
let Some(model_caller) = model_caller else {
    tracing::debug!(
        "no model_caller provided for episode distillation; skipping. \
         Pass a ModelCaller to spawn_episode_distillation() to enable."
    );
    return Ok(());
};
let distiller = Distiller::with_backend(Arc::new(GatewayDistillationBackend::new(
    model_caller,
    GATEWAY_DISTILLATION_MODEL,
)));
```
