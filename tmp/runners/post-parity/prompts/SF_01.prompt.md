# SF_01: Wire AgentContract into WorkflowEngine

## Task
Load the `AgentContract` for each agent role in the `EffectDriver::spawn_agent` path and apply its `allowed_tools` list to the dispatch request.

## Runner Context
Runner SF_A (Safety: Contract Enforcement), batch 1 of 5. No dependencies.

## Problem
`AgentContract` is defined in `crates/roko-agent/src/safety/contract.rs` with full load, cache, and validation logic, but the `WorkflowEngine` / `EffectDriver` dispatch path never loads a contract for the agent role being spawned. The contract's `allowed_tools` list and `governance` rules are ignored at dispatch time.

## Current Code

**AgentContract** -- `crates/roko-agent/src/safety/contract.rs:60-82`:
```rust

    pub invariants: Vec<Invariant>,
    pub governance: Vec<GovernanceRule>,
```

**Load method** -- `contract.rs:120`:

**EffectDriver::spawn_agent** -- `crates/roko-runtime/src/effect_driver.rs:87-157`:
Assembles a `ModelCallRequest` with role, prompt, system_prompt, modulation -- but never loads or applies a contract.

## Exact Changes

### Step 1: Add contract loading to `spawn_agent`

In `effect_driver.rs`, after the prompt assembly block (~line 139), add contract loading:

```rust
// Load behavioral contract for role (fail-closed with RestrictedFallback).
let contract = roko_agent::safety::contract::AgentContract::load_for_role_with_mode(
    role,
    roko_agent::safety::contract::ContractLoadMode::RestrictedFallback,
);
let contract = match contract {
    Ok(c) => c,
    Err(err) => {
        tracing::warn!(%err, role, "contract load failed, using restricted fallback");
        roko_agent::safety::contract::AgentContract::restricted(role)
    }
};
```

### Step 2: Apply `allowed_tools` to the request

If the contract has an `allowed_tools` list, set it on the request metadata so the downstream provider can enforce it:

```rust
if let Some(ref tools) = contract.allowed_tools {
    // Attach allowed-tools metadata to the model call request
    // so the provider/agent can enforce the tool allowlist.
    request.metadata.insert(
        "allowed_tools".to_string(),
        serde_json::to_value(tools).unwrap_or_default(),
    );
}
```

If `request.metadata` does not exist as a `HashMap`, add it to the `ModelCallRequest` struct or use whatever existing metadata field is available. Check the struct definition first.

### Step 3: Log the contract application

```rust
tracing::info!(
    run_id = %self.run_id,
    role,
    contract_role = %contract.role,
    invariant_count = contract.invariants.len(),
    governance_count = contract.governance.len(),
    allowed_tools = ?contract.allowed_tools.as_ref().map(|t| t.len()),
    "applied agent contract to dispatch"
);
```
