# SF_02: Generate default contracts during roko init

## Task
When `roko init` creates the `.roko/` directory, also create `.roko/contracts/` with a set of bundled default YAML contracts for standard roles.

## Runner Context
Runner SF_A (Safety: Contract Enforcement), batch 2 of 5. No dependencies.

## Problem
`AgentContract::load_for_role` looks for YAML files under `src/safety/contracts/` in the roko-agent crate directory (build-time path). In a user workspace, there are no contract files, so every role falls back to either a permissive or restricted default. Users need workspace-local contract overrides that `roko init` creates.

## Exact Changes

### Step 1: Add contract directory creation

In the init flow (find where `.roko/` is created -- likely in the main.rs init handler), after creating the `.roko/` directory structure, add:

```rust
fn create_default_contracts(roko_dir: &Path) -> Result<()> {
    let contracts_dir = roko_dir.join("contracts");
    std::fs::create_dir_all(&contracts_dir)?;

    let defaults = [
        ("implementer", include_str!("contracts/implementer.yaml")),
        ("reviewer", include_str!("contracts/reviewer.yaml")),
        ("strategist", include_str!("contracts/strategist.yaml")),
        ("researcher", include_str!("contracts/researcher.yaml")),
    ];

    for (role, content) in &defaults {
        let path = contracts_dir.join(format!("{role}.yaml"));
        if !path.exists() {
            std::fs::write(&path, content)?;
        }
    }

    Ok(())
}
```

### Step 2: Create the bundled YAML files

Create minimal YAML contract files that will be included via `include_str!`. Each should specify:
- `role`: the role name
- `allowed_tools`: appropriate tool list for the role
- `invariants`: basic safety invariants
- `governance`: governance rules (MaxCostPerTurn, ForbiddenTools where appropriate)

Example for `implementer.yaml`:
```yaml
role: implementer
allowed_tools:
  - read_file
  - write_file
  - edit_file
  - multi_edit
  - glob
  - grep
  - bash
  - ls
invariants:
  - NoNetworkAccess
governance:
  - !MaxCostPerTurn 10.0
```

### Step 3: Wire into init

Call `create_default_contracts` from the init handler after creating `.roko/`.
