# SF_05: Tool output sanitization pipeline -- wire ResultFilter

## Task
Wire `ResultFilter` into the `ToolDispatcher` post-dispatch path so every tool result passes through secret scrubbing, truncation, and external-source annotation before reaching the LLM.

## Runner Context
Runner SF_A (Safety: Contract Enforcement), batch 5 of 5. No dependencies.

## Problem
`ResultFilter` exists in `crates/roko-agent/src/safety/result_filter.rs` with `sanitize()`, secret scrubbing via `ScrubPolicy`, truncation, and external-source annotation. It implements `SafetyHook`. However, it is not wired into the standard `ToolDispatcher` dispatch path -- tool results go directly to the LLM without sanitization.

## Current Code

**ResultFilter** -- `crates/roko-agent/src/safety/result_filter.rs:30-37`:
```rust
pub struct ResultFilter {
    pub max_response_bytes: usize,
    pub scrub_policy: ScrubPolicy,
    pub annotate_external: bool,
}
```

**ResultFilter::sanitize** -- `result_filter.rs:61`:
```rust
pub fn sanitize(&self, content: &str, tool_name: &str) -> String
```

**ToolDispatcher** -- `crates/roko-agent/src/dispatcher/mod.rs:85-102`:
```rust
pub struct ToolDispatcher {
    registry: Arc<dyn ToolRegistry>,
    resolver: Arc<dyn HandlerResolver>,
    max_result_bytes: usize,
    safety: Option<SafetyLayer>,
    tool_cache: Option<...>,
    hook_chain: Option<hook_chain::SafetyHookChain>,
    tool_selector: Option<tool_selector::ToolSelector>,
}
```

The dispatcher has `truncate_result` logic but does NOT apply `ResultFilter.sanitize()` to tool outputs.

## Exact Changes

### Step 1: Add ResultFilter to ToolDispatcher

```rust
// In ToolDispatcher struct, add:
result_filter: Option<ResultFilter>,
```

### Step 2: Add builder method

```rust
impl ToolDispatcher {
    /// Attach a result filter for post-dispatch output sanitization.
    #[must_use]
    pub fn with_result_filter(mut self, filter: ResultFilter) -> Self {
        self.result_filter = Some(filter);
        self
    }
}
```

### Step 3: Apply filter in dispatch path

In the `dispatch_one` (or equivalent single-call dispatch method), after the handler returns a successful result and before returning it to the caller, apply the filter:

```rust
// After handler.execute() returns Ok(content):
let content = if let Some(ref filter) = self.result_filter {
    filter.sanitize(&content, &call.name)
} else {
    content
};
```

### Step 4: Wire default filter when SafetyLayer is present

In `with_safety` or wherever `SafetyLayer` is attached:
```rust
// When a SafetyLayer is attached, also create a default ResultFilter
// using its scrub_policy.
let filter = ResultFilter {
    max_response_bytes: self.max_result_bytes,
    scrub_policy: safety.scrub_policy.clone(),
    annotate_external: true,
};
self.result_filter = Some(filter);
```
