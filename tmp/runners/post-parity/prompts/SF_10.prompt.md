# SF_10: MCP tool call sanitization -- SSRF/injection/size

## Task
Create a new `mcp_sanitize.rs` module that validates MCP tool call arguments for SSRF, command injection, and payload size attacks before forwarding to MCP servers.

## Runner Context
Runner SF_C (Safety: MCP & Audit), batch 2 of 4. No dependencies.

## Problem
MCP tool calls pass arbitrary JSON arguments to external servers. Without sanitization, an agent could craft arguments that cause SSRF (e.g., `http://169.254.169.254/` in a URL field), command injection (e.g., `; rm -rf /` in a path argument), or denial-of-service (e.g., a 100MB payload).

## Current Code

**NetworkPolicy** -- `crates/roko-agent/src/safety/network.rs`:
Has URL allowlisting but only for `web_fetch`/`web_search` tools, not MCP tools.

**SafetyLayer** -- `crates/roko-agent/src/safety/mod.rs:183-199`:
Does not have MCP-specific sanitization.

## Exact Changes

### Step 1: Create `crates/roko-agent/src/safety/mcp_sanitize.rs`

```rust
//! MCP tool call sanitization (SF_10).
//!
//! Validates MCP tool call arguments before forwarding to external
//! MCP servers. Checks for:
//! - SSRF: internal/metadata IP addresses in URL arguments
//! - Command injection: shell metacharacters in string arguments
//! - Payload size: reject oversized arguments

use std::net::IpAddr;

use roko_core::tool::ToolError;
use url::Url;

/// Maximum total size of MCP tool arguments in bytes.
const MAX_MCP_ARGS_BYTES: usize = 256 * 1024; // 256 KB

/// Forbidden IP ranges for SSRF prevention.
const SSRF_BLOCKED_PREFIXES: &[&str] = &[
    "169.254.",    // AWS metadata
    "127.",        // localhost
    "10.",         // RFC 1918
    "172.16.",     // RFC 1918
    "192.168.",    // RFC 1918
    "0.",          // current network
    "::1",         // IPv6 localhost
    "fd",          // IPv6 ULA
    "fe80:",       // IPv6 link-local
];

/// Shell metacharacters that suggest injection attempts.
const SHELL_META: &[char] = &[';', '|', '&', '`', '$', '(', ')', '{', '}', '<', '>'];

/// Result of MCP sanitization.
#[derive(Debug, Clone, PartialEq, Eq)]
pub enum McpSanitizeResult {
    /// Arguments are safe.
    Safe,
    /// SSRF attempt detected.
    SsrfBlocked { url: String, reason: String },
    /// Command injection detected.
    InjectionBlocked { field: String, reason: String },
    /// Payload too large.
    PayloadTooLarge { size: usize, max: usize },
}

/// Sanitize MCP tool call arguments.
///
/// Returns `Ok(())` if safe, `Err(ToolError)` if blocked.
pub fn sanitize_mcp_args(
    tool_name: &str,
    args_json: &serde_json::Value,
) -> Result<(), ToolError> {
    // 1. Size check
    let serialized = serde_json::to_string(args_json)
        .map_err(|e| ToolError::Other(format!("failed to serialize MCP args: {e}")))?;
    if serialized.len() > MAX_MCP_ARGS_BYTES {
        return Err(ToolError::Other(format!(
            "MCP tool '{tool_name}' args exceed {MAX_MCP_ARGS_BYTES} byte limit ({} bytes)",
            serialized.len()
        )));
    }

    // 2. Walk all string values for SSRF and injection
    check_value_recursive(tool_name, args_json, &[])?;

    Ok(())
}

fn check_value_recursive(
    tool_name: &str,
    value: &serde_json::Value,
    path: &[String],
) -> Result<(), ToolError> {
    match value {
        serde_json::Value::String(s) => {
            check_ssrf(tool_name, s, path)?;
            check_injection(tool_name, s, path)?;
        }
        serde_json::Value::Object(map) => {
            for (k, v) in map {
                let mut p = path.to_vec();
                p.push(k.clone());
                check_value_recursive(tool_name, v, &p)?;
            }
        }
        serde_json::Value::Array(arr) => {
            for (i, v) in arr.iter().enumerate() {
                let mut p = path.to_vec();
                p.push(i.to_string());
                check_value_recursive(tool_name, v, &p)?;
            }
        }
        _ => {}
    }
    Ok(())
}

fn check_ssrf(tool_name: &str, value: &str, path: &[String]) -> Result<(), ToolError> {
    // Only check values that look like URLs
    if let Ok(url) = Url::parse(value) {
        if let Some(host) = url.host_str() {
            for prefix in SSRF_BLOCKED_PREFIXES {
                if host.starts_with(prefix) {
                    let field = path.join(".");
                    return Err(ToolError::Other(format!(
                        "SSRF blocked: MCP tool '{tool_name}' field '{field}' targets internal address '{host}'"
                    )));
                }
            }
        }
    }
    Ok(())
}

fn check_injection(tool_name: &str, value: &str, path: &[String]) -> Result<(), ToolError> {
    // Only flag strings with shell metacharacters that also look like commands
    if value.len() > 2 && value.chars().any(|c| SHELL_META.contains(&c)) {
        // Heuristic: if value contains a shell metachar adjacent to a common command
        let suspicious = ["rm ", "cat ", "curl ", "wget ", "python ", "node ", "sh "];
        for cmd in suspicious {
            if value.contains(cmd) {
                let field = path.join(".");
                return Err(ToolError::Other(format!(
                    "Injection blocked: MCP tool '{tool_name}' field '{field}' contains suspicious command pattern"
                )));
            }
        }
    }
    Ok(())
}
```
