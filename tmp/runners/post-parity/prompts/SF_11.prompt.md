# SF_11: Unified security audit trail -- append-only JSONL

## Task
Create a new `audit.rs` module in `roko-runtime` that provides an append-only JSONL security audit log for all safety-relevant events (permission checks, contract violations, tool blocks, secret detections).

## Runner Context
Runner SF_C (Safety: MCP & Audit), batch 3 of 4. No dependencies.

## Problem
Safety events are currently logged via `tracing` at various levels but there is no structured, append-only audit trail that can be reviewed after the fact. Compliance and debugging require a durable log of all security decisions.

## Current Code

**JsonlLogger** -- `crates/roko-runtime/src/jsonl_logger.rs:15-19`:
```rust
pub struct JsonlLogger {
    path: PathBuf,
    seq: AtomicU64,
    writer: Mutex<Option<std::io::BufWriter<std::fs::File>>>,
}
```

This logs `RuntimeEvent`s. Security audit events need their own dedicated log with different schema.

**EventBus** -- `crates/roko-runtime/src/event_bus.rs`:
Broadcast bus for runtime events. Could be a source of audit events.

### Step 1: Create `crates/roko-runtime/src/audit.rs`

```rust
//! Append-only security audit trail (SF_11).
//!
//! Records all safety-relevant events to `.roko/audit.jsonl`.
//! The log is append-only and never truncated or rotated during
//! a session. Each entry is self-describing JSON with a monotonic
//! sequence number for tamper detection.

use std::io::Write;
use std::path::{Path, PathBuf};
use std::sync::Mutex;
use std::sync::atomic::{AtomicU64, Ordering};
use std::time::{SystemTime, UNIX_EPOCH};

use serde::{Deserialize, Serialize};

/// Classification of audit events.
#[derive(Debug, Clone, PartialEq, Eq, Serialize, Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum AuditEventKind {
    /// A tool call was permitted.
    ToolPermitted,
    /// A tool call was denied.
    ToolDenied,
    /// A safety contract was loaded.
    ContractLoaded,
    /// A contract violation was detected.
    ContractViolation,
    /// A secret was detected and scrubbed.
    SecretDetected,
    /// A path escape attempt was blocked.
    PathEscapeBlocked,
    /// A bash command was blocked.
    BashCommandBlocked,
    /// An MCP tool call was sanitized.
    McpSanitized,
    /// A permission was escalated.
    PermissionEscalated,
    /// Budget was exhausted.
    BudgetExhausted,
    /// A rate limit was triggered.
    RateLimitHit,
}

/// A single audit log entry.
#[derive(Debug, Clone, Serialize, Deserialize)]
pub struct AuditEntry {
    /// Monotonic sequence number.
    pub seq: u64,
    /// ISO 8601 timestamp.
    pub timestamp: String,
    /// Event classification.
    pub kind: AuditEventKind,
    /// Agent role that triggered the event.
    pub role: String,
    /// Run ID or session ID.
    pub run_id: String,
    /// Tool name involved, if any.
    #[serde(default, skip_serializing_if = "Option::is_none")]
    pub tool: Option<String>,
    /// Human-readable detail.
    pub detail: String,
    /// Structured metadata.
    #[serde(default, skip_serializing_if = "serde_json::Value::is_null")]
    pub metadata: serde_json::Value,
}

/// Append-only audit logger.
pub struct AuditLogger {
    path: PathBuf,
    seq: AtomicU64,
    writer: Mutex<Option<std::io::BufWriter<std::fs::File>>>,
}

impl AuditLogger {
    /// Create a new audit logger at the given path.
    pub fn new(path: PathBuf) -> Self {
        Self {
            path,
            seq: AtomicU64::new(0),
            writer: Mutex::new(None),
        }
    }

    /// Create from the standard .roko directory.
    pub fn from_roko_dir(roko_dir: &Path) -> Self {
        Self::new(roko_dir.join("audit.jsonl"))
    }

    /// Record an audit event.
    pub fn record(&self, kind: AuditEventKind, role: &str, run_id: &str, detail: &str) {
        self.record_with_metadata(kind, role, run_id, None, detail, serde_json::Value::Null);
    }

    /// Record an audit event with tool and metadata.
    pub fn record_with_metadata(
        &self,
        kind: AuditEventKind,
        role: &str,
        run_id: &str,
        tool: Option<&str>,
        detail: &str,
        metadata: serde_json::Value,
    ) {
        let entry = AuditEntry {
            seq: self.seq.fetch_add(1, Ordering::Relaxed),
            timestamp: now_iso8601(),
            kind,
            role: role.to_string(),
            run_id: run_id.to_string(),
            tool: tool.map(String::from),
            detail: detail.to_string(),
            metadata,
        };

        if let Err(err) = self.write_entry(&entry) {
            tracing::error!(%err, "failed to write audit entry");
        }
    }

    fn write_entry(&self, entry: &AuditEntry) -> std::io::Result<()> {
        let mut writer = self.writer.lock()
            .unwrap_or_else(std::sync::PoisonError::into_inner);

        if writer.is_none() {
            if let Some(parent) = self.path.parent() {
                std::fs::create_dir_all(parent)?;
            }
            let file = std::fs::OpenOptions::new()
                .create(true)
                .append(true)
                .open(&self.path)?;
            *writer = Some(std::io::BufWriter::new(file));
        }

        if let Some(ref mut w) = *writer {
            let json = serde_json::to_string(entry)
                .map_err(|e| std::io::Error::new(std::io::ErrorKind::InvalidData, e))?;
            writeln!(w, "{json}")?;
            w.flush()?;
        }
        Ok(())
    }
}

fn now_iso8601() -> String {
    let ts = SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap_or_default()
        .as_millis();
    // Simple ISO format: 2026-04-29T12:00:00.000Z
    chrono::Utc::now().to_rfc3339()
}
```

## Acceptance Criteria
- `AuditLogger` writes to `.roko/audit.jsonl`
- `AuditEventKind` covers all safety event types
- Each entry has monotonic seq, timestamp, kind, role, run_id, detail
- Append-only (never truncates, uses `OpenOptions::append`)
- Thread-safe via `Mutex<Option<BufWriter>>`

## Do NOT
- Modify `JsonlLogger` (separate concerns)
- Add audit writing to any existing code paths (wiring is a later task)
- Add new crate dependencies (chrono is already available)
