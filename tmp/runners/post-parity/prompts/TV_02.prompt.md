# TV_02: Create CLI binary runner for integration tests

## Task
Create a `CliRunner` test helper that invokes the `roko` binary as a subprocess and captures stdout, stderr, and exit code for assertion.

## Runner Context
Runner TV_A (Testing: Infrastructure), batch 2 of 3. No dependencies.

## Problem
Integration tests that test the CLI binary need to invoke it as a subprocess. Each test reinvents this pattern with slight variations. A shared runner ensures consistent invocation, timeout handling, and output capture.

## Exact Changes

### Step 1: Create `crates/roko-cli/tests/cli_runner.rs`

```rust
//! CLI binary runner for integration tests (TV_02).
//!
//! Invokes the `roko` binary built by `cargo test` as a subprocess
//! and captures its output for assertion.

use std::path::Path;
use std::process::{Command, Output};
use std::time::Duration;

/// Result of a CLI invocation.
#[derive(Debug)]
pub struct CliOutput {
    /// Exit code.
    pub code: i32,
    /// Standard output.
    pub stdout: String,
    /// Standard error.
    pub stderr: String,
}

impl CliOutput {
    pub fn from_output(output: Output) -> Self {
        Self {
            code: output.status.code().unwrap_or(-1),
            stdout: String::from_utf8_lossy(&output.stdout).to_string(),
            stderr: String::from_utf8_lossy(&output.stderr).to_string(),
        }
    }

    /// Assert the command succeeded (exit code 0).
    pub fn assert_success(&self) {
        assert_eq!(
            self.code, 0,
            "expected success but got exit code {}\nstdout:\n{}\nstderr:\n{}",
            self.code, self.stdout, self.stderr
        );
    }

    /// Assert the command failed (non-zero exit code).
    pub fn assert_failure(&self) {
        assert_ne!(
            self.code, 0,
            "expected failure but got exit code 0\nstdout:\n{}\nstderr:\n{}",
            self.stdout, self.stderr
        );
    }

    /// Assert stdout contains a specific string.
    pub fn assert_stdout_contains(&self, expected: &str) {
        assert!(
            self.stdout.contains(expected),
            "stdout does not contain: {expected}\nactual stdout:\n{}",
            self.stdout
        );
    }

    /// Assert stderr contains a specific string.
    pub fn assert_stderr_contains(&self, expected: &str) {
        assert!(
            self.stderr.contains(expected),
            "stderr does not contain: {expected}\nactual stderr:\n{}",
            self.stderr
        );
    }

    /// Assert stdout does NOT contain a string.
    pub fn assert_stdout_not_contains(&self, unexpected: &str) {
        assert!(
            !self.stdout.contains(unexpected),
            "stdout unexpectedly contains: {unexpected}\nactual stdout:\n{}",
            self.stdout
        );
    }
}

/// Runner for invoking the roko CLI binary.
pub struct CliRunner {
    binary: String,
    workdir: Option<String>,
    env: Vec<(String, String)>,
    timeout: Duration,
}

impl CliRunner {
    /// Create a new CLI runner.
    pub fn new() -> Self {
        // Find the roko binary built by cargo test
        let binary = env!("CARGO_BIN_EXE_roko");
        Self {
            binary: binary.to_string(),
            workdir: None,
            env: Vec::new(),
            timeout: Duration::from_secs(30),
        }
    }

    /// Set the working directory.
    pub fn workdir(mut self, dir: impl AsRef<Path>) -> Self {
        self.workdir = Some(dir.as_ref().to_string_lossy().to_string());
        self
    }

    /// Add an environment variable.
    pub fn env(mut self, key: impl Into<String>, value: impl Into<String>) -> Self {
        self.env.push((key.into(), value.into()));
        self
    }

    /// Set the timeout.
    pub fn timeout(mut self, timeout: Duration) -> Self {
        self.timeout = timeout;
        self
    }

    /// Run the CLI with the given arguments.
    pub fn run(&self, args: &[&str]) -> CliOutput {
        let mut cmd = Command::new(&self.binary);
        cmd.args(args);

        if let Some(ref dir) = self.workdir {
            cmd.current_dir(dir);
        }

        for (key, value) in &self.env {
            cmd.env(key, value);
        }

        // Disable colors and interactive mode
        cmd.env("NO_COLOR", "1");
        cmd.env("TERM", "dumb");

        let output = cmd.output().expect("failed to execute roko binary");
        CliOutput::from_output(output)
    }

    /// Run and assert success.
    pub fn run_success(&self, args: &[&str]) -> CliOutput {
        let output = self.run(args);
        output.assert_success();
        output
    }
}

impl Default for CliRunner {
    fn default() -> Self {
        Self::new()
    }
}
```
