# TV_05: Config smoke test -- roko config show, validate

## Task
Add smoke tests for `roko config show` and `roko config validate` that verify configuration is correctly loaded and validated.

## Runner Context
Runner TV_B (Testing: Smoke Tests), batch 2 of 3. Depends on TV_01, TV_02.

## Problem
Configuration loading and validation are critical paths that lack automated smoke tests. A broken config parser or validator can prevent all commands from running.

## Exact Changes

### Step 1: Add config smoke tests

```rust
#[test]
fn smoke_roko_config_show() {
    let ws = TestWorkspace::new();
    let runner = CliRunner::new().workdir(ws.path());

    let output = runner.run(&["config", "show"]);

    output.assert_success();

    // Should display the loaded config
    // Check for expected sections
    output.assert_stdout_contains("agent");
}

#[test]
fn smoke_roko_config_validate() {
    let ws = TestWorkspace::new();
    let runner = CliRunner::new().workdir(ws.path());

    let output = runner.run(&["config", "validate"]);

    output.assert_success();
}

#[test]
fn smoke_roko_config_validate_invalid() {
    let ws = TestWorkspace::new();

    // Write an invalid config
    ws.write_file("roko.toml", "invalid = [[[[ toml");

    let runner = CliRunner::new().workdir(ws.path());
    let output = runner.run(&["config", "validate"]);

    // Should fail with a parse error
    output.assert_failure();
}

#[test]
fn smoke_roko_config_path() {
    let ws = TestWorkspace::new();
    let runner = CliRunner::new().workdir(ws.path());

    let output = runner.run(&["config", "path"]);

    output.assert_success();
    // Should print the path to roko.toml
    output.assert_stdout_contains("roko.toml");
}
```

## Acceptance Criteria
- `roko config show` displays loaded config
- `roko config validate` passes on valid config
- `roko config validate` fails on invalid config
- `roko config path` prints the config file path

## Do NOT
- Modify config commands
- Add tests for config edit or set (those modify state)
- Add new dependencies
