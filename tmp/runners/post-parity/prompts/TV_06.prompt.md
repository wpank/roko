# TV_06: Plan smoke test -- roko plan validate

## Task
Add a smoke test for `roko plan validate` that verifies the plan validator correctly accepts valid and rejects invalid task definitions.

## Runner Context
Runner TV_B (Testing: Smoke Tests), batch 3 of 3. Depends on TV_01, TV_02.

## Problem
Plan validation is the entry point for executing any plan. A broken validator can silently accept malformed plans or reject valid ones. Smoke tests ensure the validator works correctly.

## Exact Changes

### Step 1: Add plan smoke tests

```rust
#[test]
fn smoke_roko_plan_validate_valid() {
    let ws = TestWorkspace::new();

    // Create a valid plan directory with tasks.toml
    ws.write_file("plans/test-plan/tasks.toml", r#"
[[task]]
id = "T01"
title = "Test task"
role = "implementer"
prompt = "Do something"
deps = []

[[task]]
id = "T02"
title = "Second task"
role = "implementer"
prompt = "Do something else"
deps = ["T01"]
"#);

    let runner = CliRunner::new().workdir(ws.path());
    let output = runner.run(&["plan", "validate", "plans/test-plan"]);

    output.assert_success();
}

#[test]
fn smoke_roko_plan_validate_invalid_deps() {
    let ws = TestWorkspace::new();

    // Create a plan with a dependency cycle
    ws.write_file("plans/bad-plan/tasks.toml", r#"
[[task]]
id = "T01"
title = "Task A"
role = "implementer"
prompt = "Do A"
deps = ["T02"]

[[task]]
id = "T02"
title = "Task B"
role = "implementer"
prompt = "Do B"
deps = ["T01"]
"#);

    let runner = CliRunner::new().workdir(ws.path());
    let output = runner.run(&["plan", "validate", "plans/bad-plan"]);

    // Should fail (cycle detected)
    output.assert_failure();
}

#[test]
fn smoke_roko_plan_validate_missing_dir() {
    let ws = TestWorkspace::new();
    let runner = CliRunner::new().workdir(ws.path());

    let output = runner.run(&["plan", "validate", "plans/nonexistent"]);

    // Should fail with a clear error
    output.assert_failure();
}

#[test]
fn smoke_roko_plan_list() {
    let ws = TestWorkspace::new();

    // Create a plan directory
    ws.write_file("plans/my-plan/tasks.toml", r#"
[[task]]
id = "T01"
title = "Task"
role = "implementer"
prompt = "Do it"
deps = []
"#);

    let runner = CliRunner::new().workdir(ws.path());
    let output = runner.run(&["plan", "list"]);

    output.assert_success();
}
```
