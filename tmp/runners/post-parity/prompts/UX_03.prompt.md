# UX_03: Implement --dry-run mode for roko run

## Task
Add a `--dry-run` flag that previews what dispatch would do without actually executing.

## Runner Context
Runner UX, batch 3 of 5. No dependencies.

## Problem
Users want to preview what `roko run "prompt"` will do before committing to execution. A dry-run mode should show: the resolved model, system prompt, tools, estimated cost, and gate configuration without actually calling the LLM.

## Current Code

**main.rs** -- `crates/roko-cli/src/main.rs`:
Defines CLI args via clap. Already has `--json`, `--quiet`, etc.

**run.rs** -- `crates/roko-cli/src/run.rs`:
Entry point for `roko run`. Resolves config, dispatches, runs gates.

## Exact Changes

### Step 1: Add --dry-run flag to CLI

In `crates/roko-cli/src/main.rs`, in the struct that holds CLI args, add:

```rust
    /// Preview dispatch without executing. Shows model, prompt, tools, cost estimate.
    #[arg(long)]
    dry_run: bool,
```

### Step 2: Add dry_run handling in run.rs

In `crates/roko-cli/src/run.rs`, after config resolution but before dispatch, add:

```rust
pub fn print_dry_run(
    prompt: &str,
    model: &str,
    system_prompt: &str,
    tools: &[String],
    gates: &[String],
    estimated_cost: f64,
) {
    println!("=== Dry Run Preview ===");
    println!();
    println!("  Model:    {model}");
    println!("  Prompt:   {} chars", prompt.len());
    println!();

    // System prompt preview (first 5 lines)
    println!("  System Prompt ({} chars):", system_prompt.len());
    for line in system_prompt.lines().take(5) {
        println!("    | {line}");
    }
    if system_prompt.lines().count() > 5 {
        println!("    | ... ({} more lines)", system_prompt.lines().count() - 5);
    }
    println!();

    // Tools
    if !tools.is_empty() {
        println!("  Tools ({}):", tools.len());
        for tool in tools.iter().take(10) {
            println!("    - {tool}");
        }
        if tools.len() > 10 {
            println!("    ... and {} more", tools.len() - 10);
        }
        println!();
    }

    // Gates
    if !gates.is_empty() {
        println!("  Gates ({}):", gates.len());
        for gate in gates {
            println!("    - {gate}");
        }
        println!();
    }

    // Cost estimate
    if estimated_cost > 0.0 {
        println!("  Estimated cost: ${:.4}", estimated_cost);
    }
    println!();
    println!("  Run without --dry-run to execute.");
}
```

### Step 3: Short-circuit execution on dry_run

In the main run path:
