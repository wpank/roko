# XC_02: Add correlation IDs to logging

## Task
Add a correlation ID (run ID + task ID) to the tracing subscriber so all log lines within a workflow run can be correlated for debugging.

## Runner Context
Runner XC_A (Cross-Cutting: Error Handling), batch 2 of 3. No dependencies.

## Problem
Log lines from concurrent runs interleave in the output. Without a correlation ID, it is impossible to trace a sequence of events across agent dispatch, gate execution, and commit operations back to the originating run.

## Current Code

**CLI main** -- `crates/roko-cli/src/main.rs`:
Initializes a `tracing_subscriber` with formatting. Check the exact initialization code.

## Exact Changes

### Step 1: Add a correlation ID span layer

In the CLI's tracing subscriber initialization:

```rust
use tracing_subscriber::layer::SubscriberExt;
use tracing_subscriber::util::SubscriberInitExt;

// Generate a session-level correlation ID
let session_id = uuid_short();

// Create a span that wraps the entire session
let session_span = tracing::info_span!(
    "session",
    session_id = %session_id,
);
let _guard = session_span.enter();
```

### Step 2: Add run ID propagation

When entering a workflow run, create a child span:

```rust
// In the run/plan dispatch path:
let _run_span = tracing::info_span!(
    "run",
    run_id = %run_id,
    task_id = tracing::field::Empty, // filled in per-task
).entered();
```

### Step 3: Add correlation ID formatting

Update the tracing subscriber format to include span fields:

```rust
let format = tracing_subscriber::fmt::format()
    .with_target(false)
    .with_thread_ids(false)
    .with_span_list(true); // Include span fields in output

tracing_subscriber::registry()
    .with(
        tracing_subscriber::fmt::layer()
            .event_format(format)
            .with_span_events(tracing_subscriber::fmt::format::FmtSpan::NONE)
    )
    .with(tracing_subscriber::EnvFilter::from_default_env())
    .init();
```

### Step 4: Add helper for generating short UUIDs

```rust
fn uuid_short() -> String {
    use std::time::{SystemTime, UNIX_EPOCH};
    let ts = SystemTime::now()
        .duration_since(UNIX_EPOCH)
        .unwrap_or_default()
        .as_millis();
    format!("{:x}", ts & 0xFFFFFFFF)
}
```
