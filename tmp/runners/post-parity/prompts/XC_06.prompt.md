# XC_06: Worktree cleanup policy

## Task
Add a configurable cleanup policy to `WorktreeManager` that automatically removes worktrees based on age, idle time, and completion status.

## Runner Context
Runner XC_B (Cross-Cutting: Lifecycle), batch 3 of 3. Depends on RP_01.

## Problem
Worktrees accumulate over time as plans run and complete. Without automatic cleanup, disk space is consumed by stale worktrees that are no longer needed.

## Current Code

**WorktreeManager** -- `crates/roko-orchestrator/src/worktree.rs`:
Has `reclaim_idle()` that removes worktrees past their `idle_ttl`. But there is no configurable policy for cleanup frequency, maximum age, or completion-based removal.

### Step 1: Add cleanup policy configuration

```rust
/// Policy for automatic worktree cleanup.
#[derive(Debug, Clone)]
pub struct WorktreeCleanupPolicy {
    /// Maximum age for any worktree. Worktrees older than this are
    /// removed regardless of activity.
    pub max_age: Duration,
    /// Remove worktrees whose associated plan has completed.
    pub remove_on_completion: bool,
    /// Minimum disk space (bytes) to maintain. When disk usage exceeds
    /// this threshold, oldest worktrees are removed first.
    pub min_free_bytes: Option<u64>,
    /// How often to run the cleanup sweep.
    pub sweep_interval: Duration,
}

impl Default for WorktreeCleanupPolicy {
    fn default() -> Self {
        Self {
            max_age: Duration::from_secs(24 * 3600), // 24 hours
            remove_on_completion: true,
            min_free_bytes: None,
            sweep_interval: Duration::from_secs(300), // 5 minutes
        }
    }
}
```

### Step 3: Add cleanup sweep method

```rust
impl WorktreeManager {
    /// Run a cleanup sweep according to the policy.
    ///
    /// Returns the number of worktrees removed.
    pub async fn cleanup_sweep(&self) -> usize {
        let mut removed = 0;
        let now_ms = now_epoch_ms();

        let handles: Vec<_> = self.list().into_iter().collect();

        for handle in handles {
            let age_ms = now_ms - handle.created_at_ms;
            let age = Duration::from_millis(age_ms as u64);

            // Remove if older than max_age
            if age > self.config.cleanup.max_age {
                tracing::info!(
                    id = %handle.id,
                    age_secs = age.as_secs(),
                    "cleanup: removing aged worktree"
                );
                if self.remove(&handle.id).await.is_ok() {
                    removed += 1;
                }
                continue;
            }

            // Remove idle worktrees (existing logic)
            let idle_ms = now_ms - handle.last_active_ms;
            let idle = Duration::from_millis(idle_ms as u64);
            if idle > self.config.idle_ttl {
                tracing::info!(
                    id = %handle.id,
                    idle_secs = idle.as_secs(),
                    "cleanup: removing idle worktree"
                );
                if self.remove(&handle.id).await.is_ok() {
                    removed += 1;
                }
            }
        }

        if removed > 0 {
            // Run git worktree prune to clean up metadata
            let _ = self.prune().await;
        }

        removed
    }
}

fn now_epoch_ms() -> i64 {
    std::time::SystemTime::now()
        .duration_since(std::time::UNIX_EPOCH)
        .unwrap_or_default()
        .as_millis() as i64
}
```
