# D06: `CostsDb::predict_task_cost()` (percentile-based) + pre-dispatch logging

## Goal

Predict the cost of a task before dispatching, using percentile statistics over the historical cost records for similar `(model, complexity_band, role)` tuples. Fall back to `CostTable::estimate_cost` when there's no history.

Logged at every dispatch. Used by D01's budget guard for tighter pre-flight checks.

Depends on D01 (the per-plan/task/session budget machinery).

## Step 1: define `CostEstimate`

In `costs_db.rs`:

```rust
#[derive(Debug, Clone, Copy, serde::Serialize)]
pub struct CostEstimate {
    /// p10 of historical observations, or 0.0 when no data.
    pub min_usd: f64,
    /// p50 (median) of historical observations, or table-based fallback.
    pub expected_usd: f64,
    /// p90 of historical observations, or expected * 2.0 when no data.
    pub max_usd: f64,
    /// `[0.0, 1.0]`. Roughly: 0 with no data, 1 at >=20 observations.
    pub confidence: f64,
    /// Number of historical records used.
    pub support: u64,
}

#[derive(Debug, Clone, Copy, PartialEq, Eq, serde::Serialize, serde::Deserialize)]
#[serde(rename_all = "snake_case")]
pub enum ComplexityBand {
    Trivial,
    Easy,
    Medium,
    Hard,
    Frontier,
}

impl ComplexityBand {
    /// Map a continuous difficulty score in `[0.0, 1.0]` to a band.
    pub fn from_difficulty(d: f64) -> Self {
        match d {
            d if d < 0.20 => Self::Trivial,
            d if d < 0.40 => Self::Easy,
            d if d < 0.60 => Self::Medium,
            d if d < 0.80 => Self::Hard,
            _ => Self::Frontier,
        }
    }
}
```

## Step 2: implement `predict_task_cost`

```rust
impl CostsDb {
    pub fn predict_task_cost(
        &self,
        model: &str,
        complexity: ComplexityBand,
        role: &str,
    ) -> CostEstimate {
        let mut samples: Vec<f64> = self
            .records()
            .iter()
            .filter(|r| r.model.as_deref() == Some(model))
            .filter(|r| r.role.as_deref() == Some(role))
            .filter(|r| {
                // Match band from difficulty when present; if the record has no
                // difficulty, accept it (degraded match).
                r.difficulty
                    .map(|d| ComplexityBand::from_difficulty(d) == complexity)
                    .unwrap_or(true)
            })
            .map(|r| r.cost_usd)
            .collect();

        if samples.is_empty() {
            return CostEstimate {
                min_usd: 0.0,
                expected_usd: 0.0,
                max_usd: 0.0,
                confidence: 0.0,
                support: 0,
            };
        }

        samples.sort_by(|a, b| a.partial_cmp(b).unwrap_or(std::cmp::Ordering::Equal));
        let n = samples.len();
        let pct = |p: f64| -> f64 {
            let idx = ((n - 1) as f64 * p).round() as usize;
            samples[idx.min(n - 1)]
        };
        let confidence = ((n as f64) / 20.0).min(1.0);

        CostEstimate {
            min_usd: pct(0.10),
            expected_usd: pct(0.50),
            max_usd: pct(0.90),
            confidence,
            support: n as u64,
        }
    }
}
```

The exact `CostRecord` field names (`model`, `role`, `difficulty`, `cost_usd`) must match what's actually there. Read the file before assuming.

## Step 4: pre-dispatch logging in orchestrate

Insert near the same place D01 lives (orchestrate.rs:15169–15188 area, after model selection and provider availability):

```rust
let complexity = ComplexityBand::from_difficulty(task_def.difficulty.unwrap_or(0.5));
let role_label = role.label();
let estimate = self.learning.costs_db().predict_task_cost(&selected_model, complexity, role_label);

let fallback_estimate = if estimate.support == 0 {
    self.config.cost_table()
        .estimate_cost(prompt_tokens_est, expected_completion_tokens, &selected_model)
        .unwrap_or(0.0)
} else {
    estimate.expected_usd
};

tracing::info!(
    target: "roko::cost::predict",
    model = %selected_model,
    role = %role_label,
    complexity = ?complexity,
    expected_usd = estimate.expected_usd,
    min_usd = estimate.min_usd,
    max_usd = estimate.max_usd,
    confidence = estimate.confidence,
    support = estimate.support,
    fallback_estimate_usd = fallback_estimate,
    "pre-dispatch cost estimate"
);
```
