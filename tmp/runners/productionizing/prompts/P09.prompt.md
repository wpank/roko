# P09: Pre-dispatch context-overflow check

## Goal

Before dispatching to a model, estimate the prompt's token count and compare it against the resolved model's `context_window`. If the prompt would exceed 85% of the window, switch to the next-larger model among `available_model_keys_for_cascade()`. Log a warning either way. Never silently truncate the user's prompt.

## Code to insert

```rust
// P09: pre-dispatch context overflow check.
//
// Tokens are approximated as `chars / 4` — a deliberate underestimate to avoid
// false positives on agents with strict tokenizers. Threshold is 85% of the
// model's context window so the response has room (15% reserved).
let prompt_chars: usize = prompt_for_estimate.chars().count();
let prompt_tokens_est: u64 = (prompt_chars as u64) / 4;

if let Some(window) = self
    .config
    .effective_context_window_for(&selected_model)
{
    let threshold = (window as f64 * 0.85) as u64;
    if prompt_tokens_est > threshold {
        let bigger = self
            .config
            .available_model_keys_for_cascade()
            .into_iter()
            .filter_map(|k| {
                let w = self.config.effective_context_window_for(&k)?;
                Some((k, w))
            })
            .filter(|(_, w)| *w > window)
            .min_by_key(|(_, w)| *w)
            .map(|(k, _)| k);

        if let Some(bigger_model) = bigger {
            tracing::warn!(
                from_model = %selected_model,
                to_model = %bigger_model,
                prompt_tokens_est,
                from_window = window,
                "P09: prompt exceeds 85% of context window; falling back to larger-context model"
            );
            selected_model = bigger_model;
        } else {
            tracing::warn!(
                model = %selected_model,
                prompt_tokens_est,
                window,
                "P09: prompt exceeds 85% of context window and no larger model is available; \
                 dispatch may fail with ContextOverflow"
            );
        }
    }
}
```

```rust
/// Effective context window in tokens for a given model key.
///
/// Returns `None` when the model is not registered or has no context window
/// recorded (in which case callers should treat the model as unconstrained).
#[must_use]
pub fn effective_context_window_for(&self, model_key: &str) -> Option<u64> {
    let resolved = crate::agent::resolve_model(self, model_key);
    resolved
        .profile
        .as_ref()
        .and_then(|p| p.context_window_tokens)
}
```
