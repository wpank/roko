# P13: Add `Dockerfile.optimized` (cargo-chef + sccache pattern)

## Goal

Create a Tier-2 Dockerfile (`Dockerfile.optimized`) that uses `cargo-chef` for dependency caching and `sccache` for per-unit compilation caching. Source-only changes rebuild in 15–30 seconds (vs 10–15 minutes today).

This is a NEW file. The existing `Dockerfile` keeps working — `Dockerfile.optimized` is opt-in via `railway.toml` or `--file` builds.

## Files to Edit

- `Dockerfile.optimized` (new file at repo root)
- `.dockerignore` (only if it currently excludes things `Dockerfile.optimized` needs — read first)

## Also Read (do NOT modify)

- `Dockerfile` (root) — for current toolchain versions, env vars, and runtime-image layers
- `tmp/productionizing/08-FAST-BUILD-DEPLOY.md` — full reference

## Step 1: write Dockerfile.optimized

```dockerfile
# syntax=docker/dockerfile:1.7
#
# Dockerfile.optimized — Tier-2 build (cargo-chef + sccache).
#
# Source-only changes rebuild in 15–30 seconds with a warm cache.
# Cold builds take ~3–5 minutes.
#
# Use:
#   docker build -f Dockerfile.optimized -t roko:chef .
#
# Or via railway.toml: dockerfilePath = "Dockerfile.optimized"

# ─── Frontend (Vite) ──────────────────────────────────────────────────────────
FROM node:22-bookworm-slim AS frontend
WORKDIR /app/demo/demo-app
COPY demo/demo-app/package.json demo/demo-app/package-lock.json* ./
RUN --mount=type=cache,target=/root/.npm \
    npm ci --prefer-offline
COPY demo/demo-app/ ./
RUN npm run build

# ─── Planner (cargo-chef recipe) ─────────────────────────────────────────────
FROM lukemathwalker/cargo-chef:latest-rust-1.91-bookworm AS planner
WORKDIR /app
COPY Cargo.toml Cargo.lock ./
COPY crates/ ./crates/
COPY contracts/ ./contracts/
RUN cargo chef prepare --recipe-path recipe.json

# ─── Builder (cook deps from recipe, then build the binary) ──────────────────
FROM lukemathwalker/cargo-chef:latest-rust-1.91-bookworm AS builder
WORKDIR /app

# Pull in build-time system deps once (libssl-dev for openssl-sys, pkg-config, etc.)
RUN apt-get update \
    && apt-get install -y --no-install-recommends pkg-config libssl-dev git \
    && rm -rf /var/lib/apt/lists/*

# sccache for per-unit caching across runs (mounted as a build cache below).
ENV RUSTC_WRAPPER=sccache
ENV SCCACHE_DIR=/sccache
ENV SCCACHE_CACHE_SIZE=10G

COPY --from=planner /app/recipe.json recipe.json

# Cook deps. Cached unless Cargo.lock or root Cargo.toml change.
RUN --mount=type=cache,target=/usr/local/cargo/registry \
    --mount=type=cache,target=/sccache \
    cargo install sccache --locked --version ^0.8 \
    && cargo chef cook --release --recipe-path recipe.json --bin roko

# Copy source.
COPY Cargo.toml Cargo.lock ./
COPY crates/ ./crates/
COPY contracts/ ./contracts/
COPY roko.toml ./
COPY --from=frontend /app/demo/demo-app/dist ./demo/demo-app/dist

# Build the actual binary (deps already cooked + warm in cache).
RUN --mount=type=cache,target=/usr/local/cargo/registry \
    --mount=type=cache,target=/app/target \
    --mount=type=cache,target=/sccache \
    cargo build --release -p roko-cli \
    && cp target/release/roko /tmp/roko \
    && strip /tmp/roko \
    && sccache --show-stats || true

# ─── Runtime ─────────────────────────────────────────────────────────────────
FROM debian:bookworm-slim AS runtime
RUN apt-get update \
    && apt-get install -y --no-install-recommends \
       ca-certificates libssl3 git curl coreutils bash \
    && rm -rf /var/lib/apt/lists/*

COPY --from=builder /tmp/roko /usr/local/bin/roko
COPY --from=builder /app/roko.toml /workspace/roko.toml

# Docker containers always bind 0.0.0.0 — preserve the existing patches the
# hand-rolled Dockerfile applies so behavior is identical.
RUN sed -i 's/acknowledge_public_risk = false/acknowledge_public_risk = true/' /workspace/roko.toml \
    && sed -i '/^\[providers\.anthropic\]$/,/^\[/ { s/kind = "claude_cli"/kind = "anthropic_api"/; /^command = /d; }' /workspace/roko.toml \
    && sed -i '/^\[providers\.claude_cli\]$/,/^\[/ { s/kind = "claude_cli"/kind = "anthropic_api"/; }' /workspace/roko.toml \
    && sed -i '/^\[agent\]$/,/^\[/ { s/^command = "claude"/# command = "claude"/; }' /workspace/roko.toml

RUN echo 'export PS1="> "' >> /root/.bashrc

VOLUME ["/workspace/.roko"]
WORKDIR /workspace

ENV RUST_LOG=info
ENV SHELL=/bin/bash
EXPOSE 6677

ENTRYPOINT ["/usr/local/bin/roko"]
CMD ["serve", "--bind", "0.0.0.0", "--port", "6677"]
```
