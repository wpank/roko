# AG06: PheromoneField module + JSONL persistence + decay tick

## Goal

Add `crates/roko-serve/src/pheromone.rs` implementing an in-process
pheromone state store with exponential decay. Persist to
`.roko/pheromones.jsonl` for restart resilience. Spawn a 60 s decay
loop from `serve(...)`.

## Context

Mirage's `apps/mirage-rs/src/chain/pheromone.rs` is being deleted (M04).
The functional behaviour (intensity, decay, kind) must move to
roko-serve. The on-chain pheromone contract does not exist yet, so
this is a pure roko-serve module — when the contract lands later, AG06
can swap behind a `PheromoneSource` trait.

Half-life table (matches the legacy `knowledge_kinds` defaults):

| Kind | Half-life |
|------|-----------|
| Threat | 3 600 s (1 h) |
| Opportunity | 21 600 s (6 h) |
| Wisdom | 86 400 s (24 h) |

Decay: `i' = i * exp(-ln(2) * dt / half_life)`.

## Changes Required

1. Create `crates/roko-serve/src/pheromone.rs`:

```rust
//! In-process pheromone state with exponential decay.
//!
//! Source-of-truth until the on-chain pheromone contract ships
//! (Wave PH). At that point this module's storage becomes a cache
//! behind a `PheromoneSource` trait.

use std::path::{Path, PathBuf};
use std::sync::Arc;
use std::time::{Duration, SystemTime};

use chrono::{DateTime, Utc};
use parking_lot::RwLock;
use serde::{Deserialize, Serialize};
use tokio::task::JoinHandle;

const DECAY_TICK: Duration = Duration::from_secs(60);
const PERSIST_FILE: &str = "pheromones.jsonl";
const COMPACT_AT_BYTES: u64 = 50 * 1024 * 1024;
const EPSILON: f64 = 0.001;

#[derive(Copy, Clone, Debug, PartialEq, Eq, Hash, Serialize, Deserialize)]
#[serde(rename_all = "lowercase")]
pub enum PheromoneKind {
    Threat,
    Opportunity,
    Wisdom,
}

impl PheromoneKind {
    pub fn half_life(self) -> Duration {
        match self {
            Self::Threat => Duration::from_secs(3_600),
            Self::Opportunity => Duration::from_secs(21_600),
            Self::Wisdom => Duration::from_secs(86_400),
        }
    }

    pub fn parse(s: &str) -> Option<Self> {
        match s {
            "threat" => Some(Self::Threat),
            "opportunity" => Some(Self::Opportunity),
            "wisdom" => Some(Self::Wisdom),
            _ => None,
        }
    }

    pub fn as_str(self) -> &'static str {
        match self {
            Self::Threat => "threat",
            Self::Opportunity => "opportunity",
            Self::Wisdom => "wisdom",
        }
    }
}

#[derive(Clone, Debug, Serialize, Deserialize)]
pub struct PheromoneSample {
    pub agent_id: String,
    pub kind: PheromoneKind,
    pub intensity: f64,
    pub last_update: DateTime<Utc>,
}

type Key = (String, PheromoneKind);

pub struct PheromoneField {
    inner: RwLock<std::collections::BTreeMap<Key, PheromoneSample>>,
    persist_path: PathBuf,
}

impl PheromoneField {
    /// Load existing state from `<dir>/pheromones.jsonl` if present;
    /// otherwise start empty.
    pub fn load_or_new(dir: impl AsRef<Path>) -> Self {
        let persist_path = dir.as_ref().join(PERSIST_FILE);
        let inner = RwLock::new(load_from_disk(&persist_path).unwrap_or_default());
        Self { inner, persist_path }
    }

    pub fn list_all(&self) -> Vec<PheromoneSample> {
        self.inner.read().values().cloned().collect()
    }

    pub fn list_by_kind(&self, kind: PheromoneKind) -> Vec<PheromoneSample> {
        self.inner
            .read()
            .values()
            .filter(|s| s.kind == kind)
            .cloned()
            .collect()
    }

    pub fn list_active(&self) -> Vec<PheromoneSample> {
        self.inner
            .read()
            .values()
            .filter(|s| s.intensity > EPSILON)
            .cloned()
            .collect()
    }

    pub fn topology(&self) -> serde_json::Value {
        let samples = self.list_all();
        // Group: { agent_id -> { kind -> intensity } }
        let mut grouped: std::collections::BTreeMap<&str, std::collections::BTreeMap<&str, f64>> =
            Default::default();
        for s in &samples {
            grouped.entry(&s.agent_id).or_default().insert(s.kind.as_str(), s.intensity);
        }
        serde_json::json!({
            "nodes": grouped.iter().map(|(id, kinds)| serde_json::json!({
                "agent_id": id,
                "intensities": kinds,
            })).collect::<Vec<_>>(),
        })
    }

    pub fn deposit(&self, sample: PheromoneSample) {
        let key = (sample.agent_id.clone(), sample.kind);
        self.inner.write().insert(key, sample.clone());
        if let Err(err) = append_jsonl(&self.persist_path, &sample) {
            tracing::warn!(?err, "failed to persist pheromone deposit");
        }
        compact_if_needed(&self.persist_path);
    }

    pub fn decay_tick(&self) {
        let now = Utc::now();
        let mut g = self.inner.write();
        let mut to_remove = Vec::new();
        for (key, s) in g.iter_mut() {
            let dt = (now - s.last_update).num_seconds().max(0) as f64;
            let half = s.kind.half_life().as_secs_f64();
            let factor = (-std::f64::consts::LN_2 * dt / half).exp();
            s.intensity *= factor;
            s.last_update = now;
            if s.intensity < EPSILON {
                to_remove.push(key.clone());
            }
        }
        for k in to_remove {
            g.remove(&k);
        }
    }
}

fn load_from_disk(
    path: &Path,
) -> std::io::Result<std::collections::BTreeMap<Key, PheromoneSample>> {
    let raw = std::fs::read_to_string(path)?;
    let mut map = std::collections::BTreeMap::new();
    for line in raw.lines().filter(|l| !l.is_empty()) {
        if let Ok(s) = serde_json::from_str::<PheromoneSample>(line) {
            map.insert((s.agent_id.clone(), s.kind), s);
        }
    }
    Ok(map)
}

fn append_jsonl(path: &Path, sample: &PheromoneSample) -> std::io::Result<()> {
    use std::io::Write;
    if let Some(parent) = path.parent() {
        std::fs::create_dir_all(parent)?;
    }
    let mut f = std::fs::OpenOptions::new()
        .create(true)
        .append(true)
        .open(path)?;
    let line = serde_json::to_string(sample).expect("PheromoneSample serializes");
    writeln!(f, "{line}")?;
    Ok(())
}

fn compact_if_needed(path: &Path) {
    let Ok(meta) = std::fs::metadata(path) else { return };
    if meta.len() < COMPACT_AT_BYTES { return; }
    // Read all, dedupe by key (last wins), rewrite.
    let raw = match std::fs::read_to_string(path) {
        Ok(r) => r, Err(_) => return,
    };
    let mut latest: std::collections::BTreeMap<Key, PheromoneSample> = Default::default();
    for line in raw.lines().filter(|l| !l.is_empty()) {
        if let Ok(s) = serde_json::from_str::<PheromoneSample>(line) {
            latest.insert((s.agent_id.clone(), s.kind), s);
        }
    }
    let tmp = path.with_extension("compacting");
    let mut buf = String::new();
    for s in latest.values() {
        buf.push_str(&serde_json::to_string(s).expect("serialize"));
        buf.push('\n');
    }
    if std::fs::write(&tmp, buf).is_ok() {
        let _ = std::fs::rename(tmp, path);
    }
}

pub fn spawn_decay_loop(field: Arc<PheromoneField>) -> JoinHandle<()> {
    tokio::spawn(async move {
        let mut tick = tokio::time::interval(DECAY_TICK);
        tick.set_missed_tick_behavior(tokio::time::MissedTickBehavior::Skip);
        loop {
            tick.tick().await;
            field.decay_tick();
        }
    })
}

#[cfg(test)]
mod tests {
    use super::*;
    use tempfile::tempdir;

    #[test]
    fn round_trips_through_disk() {
        let dir = tempdir().unwrap();
        let f = PheromoneField::load_or_new(dir.path());
        f.deposit(PheromoneSample {
            agent_id: "agent-1".into(),
            kind: PheromoneKind::Threat,
            intensity: 0.8,
            last_update: Utc::now(),
        });
        let f2 = PheromoneField::load_or_new(dir.path());
        let entries = f2.list_all();
        assert_eq!(entries.len(), 1);
        assert_eq!(entries[0].kind, PheromoneKind::Threat);
    }

    #[test]
    fn decay_tick_reduces_intensity() {
        let dir = tempdir().unwrap();
        let f = PheromoneField::load_or_new(dir.path());
        f.deposit(PheromoneSample {
            agent_id: "agent-1".into(),
            kind: PheromoneKind::Threat,
            intensity: 1.0,
            last_update: Utc::now() - chrono::Duration::seconds(3_600),
        });
        f.decay_tick();
        let s = f.list_all().pop().unwrap();
        // Half-life 1 h → after 1 h intensity ≈ 0.5
        assert!((s.intensity - 0.5).abs() < 0.05);
    }

    #[test]
    fn list_active_filters_below_epsilon() {
        let dir = tempdir().unwrap();
        let f = PheromoneField::load_or_new(dir.path());
        f.deposit(PheromoneSample {
            agent_id: "agent-1".into(),
            kind: PheromoneKind::Threat,
            intensity: 0.0001,
            last_update: Utc::now(),
        });
        assert!(f.list_active().is_empty());
        assert_eq!(f.list_all().len(), 1);
    }
}
```

## Do NOT

- Do not run the decay tick at 1 s intervals. 60 s is the contract;
  faster is wasted CPU.
- Do not store pheromones inside the database / chain reader. This
  module is *the* in-process source of truth until the contract
  arrives.
- Do not block the deposit path on disk I/O. The `append_jsonl`
  function is allowed to block briefly; if disk is slow this becomes
  a hot path, in which case batch via `tokio::sync::mpsc`. Not in
  this batch.
