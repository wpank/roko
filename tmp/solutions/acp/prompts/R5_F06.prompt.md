## Problem
When roko applies a pattern or makes a decision, the user has no visibility into WHY. Other agents treat context as a flat bag of tokens. Roko has the infrastructure to trace every decision back through:
- Which playbook suggested the pattern (mined from N episodes)
- Which episodes taught it (with success/failure rates)
- Which knowledge entries informed it (from neuro store)
- Which dream insight reinforced it (if any)

This provenance chain is the single most differentiating UX roko can offer — no other agent can do this because no other agent has persistent episodic memory + playbook mining + knowledge tiers.

2. **Build the chain** by querying each store:
```rust
async fn build_provenance(prompt: &str, workdir: &Path) -> Option<ProvenanceChain> {
    let mut sources = Vec::new();

    // 1. Check playbook store for matching patterns
    if let Ok(playbooks) = PlaybookStore::open(workdir).query(prompt, 3) {
        for pb in playbooks {
            sources.push(ProvenanceSource::Playbook {
                id: pb.id.clone(),
                summary: pb.summary.clone(),
                source_episode_count: pb.source_episodes.len() as u32,
                success_rate: pb.success_rate,
            });
        }
    }

    // 2. Check episode log for similar past tasks
    if let Ok(episodes) = EpisodeStore::open(workdir).similar(prompt, 2) {
        for ep in episodes {
            sources.push(ProvenanceSource::Episode {
                id: ep.id.clone(),
                summary: ep.summary.clone(),
                gate_results: ep.gate_summary(),
            });
        }
    }

    // 3. Knowledge entries (already queried in R5_F05, reuse)
    // 4. Dream insights if available
    if let Ok(report) = DreamReport::latest(workdir) {
        for insight in report.routing_advice {
            sources.push(ProvenanceSource::DreamInsight {
                date: report.timestamp.format("%Y-%m-%d").to_string(),
                insight: insight.summary.clone(),
            });
        }
    }

    if sources.is_empty() {
        return None;
    }

    let confidence = sources.iter().filter_map(|s| match s {
        ProvenanceSource::Playbook { success_rate, .. } => Some(*success_rate),
        ProvenanceSource::Knowledge { relevance, .. } => Some(*relevance),
        _ => None,
    }).sum::<f64>() / sources.len() as f64;

    Some(ProvenanceChain {
        pattern_name: extract_pattern_name(prompt),
        sources,
        confidence,
    })
}
```

3. **Emit as ToolCall card**:
```rust
if let Some(chain) = build_provenance(prompt, workdir).await {
    let content_lines: Vec<String> = chain.sources.iter().map(|s| match s {
        ProvenanceSource::Playbook { summary, source_episode_count, success_rate, .. } =>
            format!("├─ Playbook (from {} episodes, {:.0}% success)\n│   {}", source_episode_count, success_rate * 100.0, summary),
        ProvenanceSource::Episode { id, summary, gate_results } =>
            format!("├─ Episode #{}\n│   {} ({})", id, summary, gate_results),
        ProvenanceSource::Knowledge { tier, summary, relevance } =>
            format!("├─ Knowledge [{}] ({:.2})\n│   {}", tier, relevance, summary),
        ProvenanceSource::DreamInsight { date, insight } =>
            format!("├─ Dream ({date})\n│   {insight}"),
    }).collect();

    let card_text = format!(
        "Pattern: {}\n{}\n\nConfidence: {:.2}",
        chain.pattern_name,
        content_lines.join("\n"),
        chain.confidence
    );

    event_sender.send(CognitiveEvent::ToolCallStart {
        tool_call_id: "provenance".to_string(),
        title: "Decision provenance".to_string(),
        kind: ToolCallKind::Other,
    }).await?;

    event_sender.send(CognitiveEvent::ToolCallComplete {
        tool_call_id: "provenance".to_string(),
        status: ToolCallStatus::Completed,
        content: vec![ContentBlock::Text { text: card_text }],
    }).await?;
}
```

4. **Only emit when there's meaningful provenance** (at least 1 playbook or 2+ sources). Don't show empty chains.
