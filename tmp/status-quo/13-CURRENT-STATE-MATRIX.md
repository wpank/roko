## Systemic Pattern

Most migration debt has the same shape: a new v2 abstraction exists, an older working path still carries real behavior, and a compatibility adapter makes them appear closer than they are. The roadmap should prioritize the seams where a compatibility adapter can silently change behavior: default plan engine, resume, event projection, gate pass/stub semantics, provider tool execution, and state persistence.
